　　将残破的地图拿在手中，慕清霄来到柜台前，微笑道：“老板，能否给我一份精准的沙漠地图。”
<br/>　　老者依旧坐在板凳上，目光却紧紧盯着他，眼眸中闪过一丝精茫，道：“东西都在柜台上，你自己选吧。”
<br/>　　微微摇头，慕清霄道：“我说的是，需要店内最精准的地图。”
<br/>　　老者微微皱了皱眉，目光在慕清霄手中的残破地图上扫过，沧桑的眼眸中，掠过一丝莫名的意味：“你以前见过这种残图？”
<br/>　　“见过如何，没见过又如何？不过，堂堂冰皇竟在这种鸟不拉屎的地方卖地图，真是令人吃惊啊。”
<br/>　　话语刚落，老者瞳孔剧烈收缩，豁然站起身来，全身上下散发出一股强横的气息，紧紧盯着慕清霄，道：“我看你一点不像吃惊的样子，你到底是什么人，来此处有何目的？”
<br/>　　冰皇！
<br/>　　躲在慕清霄身后，小医仙脸色有些惨白，简直不敢相信自己的耳朵，面前的老者，居然是加玛帝国曾经的十大强者之一，冰皇.海波东！
<br/>　　慕清霄负手而立，饶有兴致的打量了他一眼，随后一道恐怖的气息，笼罩着整个店铺：“省省吧，就算你全盛时期都不是我的对手，杀斗灵级别的你，就跟碾死蚂蚁一样。”
<br/>　　恐怖的气息，令冰皇惊骇莫名，难怪看不透，面前的青年，居然是一尊斗皇，识趣的将气息收敛了起来，道：“地图可以给你，但我也有条件。”
<br/>　　能成为斗皇，哪一个不是人老成精，冰皇也知道，现在小命被攥在面前的青年手中，若是他不给地图，只怕不会手下留情。
<br/>　　而且，后者来历神秘，小小年纪实力却恐怖如斯，哪怕全盛时期的他都惹不起，倒不如做个顺水人情，看看能不能提一点条件。
<br/>　　“你倒是说说看。”慕清霄微笑着说道。
<br/>　　冰皇叹息一声，道：“你也知道，我曾经是斗皇，但现在却落魄成斗灵，替我解开封印，我就将最精确的地图与你手中的残图都送给你，你的目的我多少能猜到一点。”
<br/>　　慕清霄微微点头，道：“把手伸出来。”
<br/>　　闻言，冰皇眼中闪过一丝困惑，但还是将手伸了出去，因为后者要杀他，根本不必废话。
<br/>　　握住他的手腕，慕清霄眼中浮现暗黑色原子图案，后者心脏上，确实有一副蛇形图案的封印。
<br/>　　不过，封印终归是封印，力量在强，终究有承受的极限。
<br/>　　永恒万花筒一放即收，在九霄琉璃焱的恐怖威能下，封印瞬间泯灭，下一秒，冰皇体内就涌出一股恐怖的气息，二星斗皇，隐隐有突破三星的趋势。
<br/>　　慕清霄松开他的手腕，道：“把地图给我吧。”
<br/>　　此刻，冰皇处于呆泄状态，眼中尽是不可思议，忙活这么多年，却让他无可奈何的封印，居然就如此轻松的破开了！
<br/>　　“这......你这！”
<br/>　　“别废话了，把地图给我。”
<br/>　　封印已经解除，慕清霄也不想与他啰嗦，直接讨要地图，不就破开个封印，至于这么大惊小怪吗？
<br/>　　“好，好的。”
<br/>　　懵逼中的冰皇回过神来，手掌在怀中掏了掏，最后摸出一张薄薄的羊皮纸递了过来，道：“在我的探测中，塔戈尔沙漠中，有三处地方最有可能存在异火。”
<br/>　　接过羊皮纸，慕清霄将之摊开，这是一张绘制极其详细的塔戈尔大沙漠地图，信息量完全不是柜台上的地图能比的。
<br/>　　上面不仅精确标明了沙漠中水源存在点，而且还将散布在沙漠中的蛇人部落也仔细标明了出来。
<br/>　　“地图上的三个火焰标志，其中一处隐藏着异火，当然，这些只是经过我的探测而推理出来的，精确率不可能达到百分百，不过比起你在沙漠中乱窜，无疑要好上许多。”
<br/>　　慕清霄微微点头，确实要比无头苍蝇来的好，毕竟塔戈尔大沙漠一望无际，真要自己去找，无疑会浪费很多时间。
<br/>　　“多谢。”
<br/>　　“不必客气，你帮了我很大的忙，是我应该谢你，对了，你不是加玛帝国的人吧？”
<br/>　　微微点头，慕清霄拉着小医仙，道：“仙儿，我要进沙漠一段时间，你在这里等我，一周内肯定回来。”
<br/>　　“我......嗯，我在这里等你。”
<br/>　　原本，小医仙是想说一起去，但是想想自己现在的实力，去了恐怕会拖后腿，她不想给慕清霄增加麻烦，最后还是选择在此等候。
<br/>　　慕清霄自储存空间中取出两瓶紫色液体，交给小医仙，在她玉额上轻轻一吻，道：“液体中的狂暴力量，我已经抽离，以你现在的实力，慢慢吸收不会有问题，在这里等我。”
<br/>　　接过玉瓶，小医仙知道，其中的东西肯定无比珍贵，乖巧的点了点头，温柔道：“你自己小心。”
<br/>　　“这是，伴生紫晶源，真是好东西啊！”冰皇毕竟是斗皇强者，一眼就看出，玉瓶中的液体是何物，但是想想后者的恐怖实力，能弄到并不奇怪。
<br/>　　慕清霄看了一眼海波东，语气平淡，道：“请你帮我照看妻子一段日子，算我欠你一个人情。”
<br/>　　“放心，你不再的日子，若是仙儿姑娘出一丝差错，我把命抵给你。”
<br/>　　海波东自然满口答应，慕清霄的天赋在他看来简直是妖孽，绝不是一个加玛帝国能框住的，能让这样的人物欠一个人情，算是他的机遇。
<br/>　　如此天上掉馅饼的事情，他完全没理由拒绝。
<br/>　　将‘净莲妖火’的残图收进储存空间中，离开商店后，慕清霄便在附近的客栈开了一间客房，有冰皇在，小医仙的安全也有保障。
<br/>　　特地叮嘱了一声，慕清霄就转身离开客栈，向着一望无际的沙漠深处而去。
<br/>　　小医仙站在客房中的窗户边，望着遥远的金色沙漠，美眸中尽是担忧，握着手中的伴生紫晶源，美眸变得坚定起来。
<br/>　　“我一定会快速修炼，虽不可能赶上你的脚步，却也不能拖你的后腿。”
<br/>　　说罢，自怀中取出的软骨散，挥洒在房间各个角落，随后盘坐在床榻上，开始吸收起伴生紫晶源中的力量。
