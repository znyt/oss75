　　第546章
<br/>　　“欧公子，你觉不觉得，咱们今天有点砸场子的意思？”
<br/>　　刘发财如此问，在人前他也担心暴露程大雷的身份。
<br/>　　“有嘛？”程大雷摸了摸鼻子：“咱们今天不是来求人办事的么，怎么会搞成这个样子？”
<br/>　　“嗳。”程大雷随之挥挥手：“说来说去，还是他们心眼太小。读书人么，怎么不能心胸开阔些，一肚子圣贤文章，全部塞住了心窍。”
<br/>　　“虽然是他们不对，但要是事情闹僵了，怕上了山不太好说话。”刘发财。
<br/>　　“你要这样想。以我的风格，没有拎着大斧从山下砍到山上，已经算是很温柔的方式了。难道我没有给他们面子么？”
<br/>　　“呃……欧公子大人大量，自然不会和他们斤斤计较。”刘发财抬头望了一眼：“可接下来的路，怕是不太好走。”
<br/>　　如程大雷所言，这件事本身没多大。读书人的圣地，才子的窝子，每天都要许多稀奇古怪的事情发生。今日孟夫子破格要见三人也算不上什么稀罕的事情。两道难关，也并非读书人都心眼小，只要三人拿出值得尊重的实力，对方自然会放行。
<br/>　　然而，程大雷过关的方式实在有些……钟牧山胆战心惊，李舜时口吐鲜血。程大雷是一点面子也没给书院留。书院毕竟是书院，也是一个要脸面的地方。
<br/>　　如今书院的师长还未有动静，但弟子们已经团结在一切，必须郑重起来，将三人当做对手，替书院扳回一些颜面。
<br/>　　刘发财抬头看，看到的就是这样的情景。眼前密密麻麻都是人影，三五成群立在山道两侧的石台上。人人身穿白袍，被海风卷起袍袖，倒是有些飘飘御风的味道。
<br/>　　他们脸上的表情倒也不是咬牙切齿，面目狰狞，读书人总有读书人的体面，但无数道冷冰冰的目光汇聚在一起，总能给人带来莫名的压力。若胆子小些的，面对这一幕怕呼吸都难顺畅。
<br/>　　可程大雷委实不是一个被别人气场所吓住的人；刘发财又是靠招摇撞骗吃饭的人，最擅长的就是伪装气场。至于楚云生……他求学之路颇为坎坷，白眼这种东西实在司空见惯。
<br/>　　三人拾阶而上，对肃杀的目光视若无睹，程大雷和刘发财坦然的很，楚云生依旧如一尊石佛一般，面无表情。
<br/>　　“公子请留步。”
<br/>　　一白袍书生跃出来，抱拳行礼，从怀里掏出一叠纸。
<br/>　　“在下周佛，有道不解的数算之题，还请三位解惑？”
<br/>　　……
<br/>　　一路上山，一路坎坷，几乎每走几丈，就有人跃出来刁难。
<br/>　　这个说还请公子解惑，那个说请赐教，言语都很客气，态度都很冰冷。
<br/>　　接下来的路程上，程大雷和刘发财便没有出什么力了，都是楚云生一路过关斩将。
<br/>　　在这条路上，他被考了术算、经义、田政、军论。
<br/>　　不说书院上下大惊失色，程大雷也对这貌不惊人的楚云生刮目相看。这不仅不是个躺赢的货，竟是个扮猪吃老虎的天才。
<br/>　　程大雷算是看出来了。楚云生在诗词、绘画、乐理上没什么天赋。但对于数字却极为敏感。说白了，他是个理科生，对艺术无感，但理科真的很强。
<br/>　　一路上山，一路用学问杀人，书院的颜面被落得一干二净。慢慢，楚云生自己的气场也出来了，脚步向前，虽仍旧是一副死鱼脸，但眼神已经看着让人有些怕。
<br/>　　后来，山道上想向他挑战的人越来越少，大家心里不由得有些打鼓。心里暗自琢磨着：这货是哪里冒出来的，琅琊城竟还有如此人物，怎么以前从未听说过。
<br/>　　而此时此刻，山上已乱作一团。
<br/>　　“什么，李小蜓也没难住他。那周佛呢……”
<br/>　　“啊，周佛的数算都被他比下去了。陈莲生当是没有问题吧。”
<br/>　　“呃，原来陈莲生已经败了。那么快让钱观鱼去，多少替书院争回些面子。”
<br/>　　“喔，观鱼已经去了。好好，那我就放心了。”
<br/>　　一路上山，两侧白袍人影立着不少，却无人再上前拦路。三人此刻已差不多算是登上山顶，眼前亭台楼阁，算是书院的腹心。迎面是一座白玉牌坊，和山下那座相比是小了些，但却是同样的气派。牌坊上只书一个字：仁。
<br/>　　跨过这座牌坊，当就算正式进入书院，应该就没人再好意思当拦路虎。楚云生也是松了口气，像今天这样的事，是他所做的少有的狂事。终于，他也如那摆擂三年不败的李舜时，红楼坐而论道的才子一样，有了自己的战绩，可以在琅琊城留下自己浓墨重彩的一笔了。
<br/>　　“呔，住了吧。”
<br/>　　仁字牌坊下立着一道人影，少年模样，持剑横眉。
<br/>　　“想要上山入书院，先过了我这柄剑。如果胜不过我手中剑，赶紧滚下山去。”
<br/>　　剑道也是书院必修的课目，帝国有不少扬名的大将，都是出自书院。以剑道难为三人，书院也算说得过去，不会太伤书院的脸面。
<br/>　　如果书院还剩下些脸面的话。
<br/>　　但对于楚云生来说，他的确对此道一窍不通。目光看向程大雷。
<br/>　　“啧。”程大雷瞬间来了精神，毕竟，一路给人当配角对程大雷这种人来说，的确有些憋屈。
<br/>　　“来来来，什么人，报个名号吧？”程大雷摁剑在仁字牌坊下，雄姿英发，不可一世。
<br/>　　“书院钱观鱼。”少年态度比程大雷更加骄横：“你们想进书院，能过得了我手中剑么？”
<br/>　　“什么路数？”程大雷低声问。
<br/>　　“钱观鱼，七贤十二秀中的老幺，年方弱冠，却有一手出神入化的快剑。号称书院第一剑客，也就是仗着这手快剑，才在七贤十二秀中排上名号。”刘发财。
<br/>　　“你这人还算有些见识。”
<br/>　　钱观鱼从怀里抓出一把铜钱，挥手撒开，只见剑光一闪，十三枚铜钱没有一枚落地，都稳稳搁在剑身上，而所有铜钱都被斩成了两半。
<br/>　　“你的剑有我的快么？”钱观鱼问：“只要你也能斩开铜钱，我便放你入书院？”
<br/>　　咦，这一幕为何隐隐有些熟悉。
<br/>(本章完)
