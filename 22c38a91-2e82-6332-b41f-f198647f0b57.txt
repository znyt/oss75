
<br/>　　<br/>　　    小林昙一的求胜欲望很强，强到了脑子转的快反应也快，他指了指两扇石门后黑漆漆的宫殿，王惊蛰若有所思的回过头，陈三岁暂时放弃了出手的念头。<br/>　　<br/>　　    石门后很黑，黑到根本就看不清里面是什么有什么会发生什么，之前的三井家人不清楚，后来的王惊蛰也不知道，那这时候他自然希望有人能探一探了，他自己绝对不会先过去，孔良和陈三岁他同样舍不得，无疑小林昙一这个鬼子最合适不过了。<br/>　　<br/>　　    这个家伙很机灵，切入点很准啊！<br/>　　<br/>　　    小林昙一见他们没啥反应，就边朝着台阶走去边颤悠悠的说道：“我觉得那里可能有魔鬼，也可能是地狱，三井良河他们进去就一点动静都没有了，等再出来的时候就全都疯了，眼睛是红的全是血丝，见人就要杀，我觉得他们的身上充满了罪恶的味道”<br/>　　<br/>　　    陈三岁和王惊蛰听到他的话都皱起了眉头，莫名的愣了，这种状况明显超出了他俩的意料。<br/>　　<br/>　　    陈三岁就嘀咕道：“鬼上身么？不太可能啊，这可是神庙，你看那些佛陀的佛像，明显是被人供奉过的，收了信徒的念力，这种地方会有鬼上身？那我还不如相信，母猪能上树了”<br/>　　<br/>　　    神庙前，那些柱子后面代表着苯教神明的一尊尊石像，雕刻的看起来不单单是栩栩如生的，上面也焕发着一种神圣的光彩，就像之前初来神庙前时，小林昙一和三井美奈说的，这座宫殿从里到外都一副破旧的样子，为什么那些石像看起来却干干净净的？<br/>　　<br/>　　    不说各名山大川里的寺庙或者道观中的佛像，看起来很干净也很有光，就像乐山这种大佛也同样如此，这可不是因为常年雷雨都有专人打扫而导致佛像不落灰的原因，那是因为常年中都有信徒和人祭拜，这才会让石像看起来焕然一新的。<br/>　　<br/>　　    小林昙一走上了台阶，走着走着脸上就忽然露出了一抹狡黠的表情，他眼睛余光瞥着陈三岁，王惊蛰还有孔良，又瞄着他们来时过来的那片林子，忽然之间小林昙一就快速的从台阶上跳了下去，想要一头就扎进林中，然后顺着原路返回，他想离开天坑，甚至马上离开雪域高原。<br/>　　<br/>　　    他的意图就是声东击西，因为陈三岁和王惊蛰估计也不会防着他这一点，但计划永远没有变化快，而且还是惊变。<br/>　　<br/>　　    当小林昙一从台阶上跳下来，身体还没落地的时候，他忽然间就察觉到自己后背传来一股极大的吸力，就跟有人在后面拉着他似的，让他完全挣扎不得，接着小林昙一人在半空就倒退着朝着两扇石门后急射而去。<br/>　　<br/>　　    “不要……”小林昙一惊恐的叫了一声，随即身子就隐没在了神庙里。<br/>　　<br/>　　    变故很快，三井美奈没有反应过来，陈三岁和王惊蛰也愣了。<br/>　　<br/>　　    三井美奈稍稍的往后退了两边，轻声说道：“他要疯了。”<br/>　　<br/>　　    王惊蛰皱眉问道：“刚才究竟发生了什么？”<br/>　　<br/>　　    “我不清楚，进去的人开始都没有什么动静，等再出来的时候就失去了神志，见人就想杀，全然不认得之前的人了，他们很疯狂，眼睛很红……”<br/>　　<br/>　　    陈三岁“哦”了一声，点头道：“和那些二战的纳税士兵一样，自相残杀了”<br/>　　<br/>　　    三三井美奈顿时惊讶的说道：“什么，是马丁鲍曼的士兵？”<br/>　　<br/>　　    “天坑下面，我们下来的时候看见了很多的白骨，死的人应该都是几十年前的纳粹士兵”王惊蛰回过头问道：“那批珍宝什么的是怎么回事？”<br/>　　<br/>　　    三井美奈没犹豫，直接说道：“当初，二战要结束的时候阿道夫为了东山再起，让他手下的亲信带了三批黄金，珠宝和艺术品离开了，前往各地埋藏，其中马丁鲍曼带着人远涉千里进入到了雪域高原，后来我们家族得到了马丁鲍曼的笔记本，大概知道了他埋藏这些珍宝的范围，于是近十年间我们派遣了不少人进入藏区，直到一个月之前，找到了天坑附近的一个藏族村，这个日记本里记录的东西只有一小部分，后半段不知被什么人给撕掉了，撕去的无疑就是这处天坑里的秘密”<br/>　　<br/>　　    王惊蛰忽然又转了过去，这时石门前果然出现了小林昙一的身影，眼睛一如既往的通红，布满了血丝。<br/>　　<br/>　　    三井美奈紧张的说道：“他，他果真又疯了”<br/>　　<br/>　　    王惊蛰朝着孔良摆了摆手示意他躲开，陈三岁走过来和他并肩站在一起，皱眉说道：“不是鬼上身，但他已经明显不在是他了，这是怎么回事，有点太离奇了一些，我还是第一次碰到这种状况”<br/>　　<br/>　　    王惊蛰也很迷惑说了一句看看再说，小林昙一就已经一头扎下了台阶，然后径直的朝着两人这边冲了过来，之前离的太远没有什么异样的感觉，离的近了一些，王惊蛰猛然发现对方的身上萦绕着一层淡淡的光辉。<br/>　　<br/>　　    他的脸上，颇有种宝相庄严的气势，尽管人还看着疯疯癫癫的。<br/>　　<br/>　　    小林昙一的嘴中唱着一句句晦涩难懂的经文，那种古老的音阶听的人云里雾里一阵迷糊，陈三岁和王惊蛰骤然分开。<br/>　　<br/>　　    “要死的，活的？”<br/>　　<br/>　　    王惊蛰说道：“生擒了他，我看看……”<br/>　　<br/>　　    小林昙一跃过他俩躲开后，人也没停，略微一转奔向了三井美奈。<br/>　　<br/>　　    陈三岁脚尖一点地面，急速往前蹿去，伸手就搭在了对方肩膀上，但没想到这人力气出奇的大，陈三岁手搭在上面就用力抓了下去，但却被小林昙一一下子就给挣脱开了。<br/>　　<br/>　　    陈三岁随即两腿一弯，屏住呼吸斜着身子就跃了起来，然后两手搂住对方的半身，用力的向地上倒下，带着他滚了一圈后，陈三岁右脚伸到小林昙一的脖子下方，突然回钩，就把人的脑袋给锁上了。
