　　超越万倍的价格，这已经不是狮子大开口了。
<br/>　　敢出这样的价格，简直就是没有将他欧阳烈，放在眼中。
<br/>　　“怎么，你听不懂人话？一枚龙血圣王丹，一百块皇品灵石，一个穷鬼，买不起就滚！”
<br/>　　楚风眠瞥了一眼欧阳烈，眼神中尽是不屑。
<br/>　　这一道眼神，仿佛真的就像是欧阳烈，在楚风眠眼中，就是一个穷鬼一样。
<br/>　　此言一出，周围的众人，无数武者，脸色都是僵住了。
<br/>　　一个神力境九重的武者，居然敢跟欧阳烈如此说话。
<br/>　　欧阳烈是谁？在这虚空海中，几乎是最为强大的神力境武者之一。
<br/>　　眼前的楚风眠，虽然也是神力境九重的境界，跟欧阳烈一样。
<br/>　　但是就算是同为神力境九重的境界，其中的实力，也是无比巨大的，像是欧阳烈的实力，灭杀寻常生死境武者，都并非难事。
<br/>　　如今对付一个神力境武者，简直是手到擒来，轻松无比。
<br/>　　“这小子是疯了不成？”
<br/>　　“看他的容貌，极为陌生，应该是刚来虚空海的新人，一个新人，不知天高地厚，居然敢挑衅欧阳烈？”
<br/>　　“找死，这小子是在自寻死路，欧阳烈出手，一招之内就足以轰杀这个小子。”
<br/>　　无数围观武者，议论纷纷，楚风眠是谁，他们都没见过。
<br/>　　但是欧阳烈的大名，他们可都听说过的，在这虚空海中，神力境武者中，能够抗衡欧阳烈的人，都寥寥无几。
<br/>　　一个不知名的小子，怎么可能跟欧阳烈抗衡。
<br/>　　“这小子的性格，太狂了，如此狂妄，不是好事，不过毕竟是在我们天商会中起的冲突，出现死伤倒是不好，等着小子受到一个教训，我在出手制止吧。”
<br/>　　那任务殿的李长老，看着楚风眠，也是摇了摇头道。
<br/>　　楚风眠跟白萱二人，能够斩杀虚空五霸，的确是有些实力，但是这实力，跟欧阳烈抗衡，还远远不够。
<br/>　　“哈哈哈哈。”
<br/>　　突然间，欧阳烈大笑起来，在这笑声之中，有着一种令人听到就感觉到浑身战栗的恐怖。
<br/>　　在这笑声之中，欧阳烈身上的力量，都瞬间爆发了出来，滚滚气浪，令在场无数众人都是退后连连，急忙躲开。
<br/>　　“一个小小的蝼蚁，也敢来狮子大开口？看来我欧阳烈许久未出手，许多人都已经忘记我的威严了！”
<br/>　　欧阳烈看向楚风眠，眼神中已经充满了杀意。
<br/>　　欧阳烈的境界，虽然还是神力境的境界，但是他的实力，已经是超脱于神力境的范畴了。
<br/>　　他的一身实力，足以抗衡生死境四重的强者，就算是许多生死境高阶的强者，见到他，都极为客气，平起平坐。
<br/>　　因为一旦是欧阳烈选择度过生死大劫，踏入生死境高阶，便是板上钉钉的一件事。
<br/>　　在这混元大陆中，无数商会势力都要给他几分面子。
<br/>　　今日他本来以为，他开口，楚风眠便是就会乖乖献上龙血圣王丹，然后便会灰溜溜的走人了。
<br/>　　却没有想到，楚风眠居然丝毫不给他面子，甚至是根本就是在讥讽他。
<br/>　　这让欧阳烈如何忍受。
<br/>　　“好，很好，看你的样子，你是北邙学院的弟子？不过就算是北邙学院的弟子，敢来对我这样说话，也是死罪！落长恨都不敢在我面前放肆！”
<br/>　　欧阳烈一双眼睛，死死的盯着楚风眠，一字一字的开口道。
<br/>　　“跪下，自废修为，今日我饶你不死！”
<br/>　　“跪下？自废修为？一个神力境的小子罢了，还真以为自己已经是生死境高阶的大人物了？让我跪下，在这个北域，还没有谁有这个资格！”
<br/>　　楚风眠看着欧阳烈，冷哼一声道。
<br/>　　“欧阳烈？天盟的人？在我的面前，不过就是一条狂吠的狗。”
<br/>　　楚风眠的话一出，全场无声。
<br/>　　这狂吠的狗四个字出来，简直是让所有人都愣住了。
<br/>　　一个个看向楚风眠，简直就看向一个疯子。
<br/>　　“这家伙是疯了不成？”
<br/>　　“完了，这天商会要见血了。”
<br/>　　“上一个敢跟欧阳烈这样说话的人，就被欧阳烈当场斩杀了。”
<br/>　　“这里是天商会，天商会的人会阻止吧。”
<br/>　　“那又如何？等出了天商会，这小子一样得死，我就不信他一辈子就待在天商会里！”
<br/>　　“咔嚓！”
<br/>　　一声地砖破碎的声音，在欧阳烈的脚下地砖，都直接被他踩碎。
<br/>　　欧阳烈的脸，已经是血气上涌，气的通红，心中已经是愤怒至极。
<br/>　　欧阳烈在这虚空海纵横，还从有人，敢说他是一条狗，这种侮辱，比起杀了他还要令他难受。
<br/>　　“好伶牙俐齿的小子，我看你的嘴，能不能救得了你的命，给我去死！”
<br/>　　欧阳烈终于忍不住了。
<br/>　　他的身上，一股股灵力展现出来，周围的空间，仿佛都是开始破碎，变化。
<br/>　　一道道空间，就在欧阳烈的手中凝聚，化为了锋利的刀锋，无声无息之间，向着楚风眠猛然斩了过来。
<br/>　　这刀锋，跟周围的空间简直便是融为一体，令人防不胜防，毫无轨迹，躲都无法躲开，四面八方，无时无刻都有可能出现，斩杀过去。
<br/>　　“凝空为刀！这是将空间法则，领悟到极致的能力！相传这欧阳磊，有着远古神兽鲲鹏的血脉，足以操控空间。”
<br/>　　“这一刀出，除非是生死境高阶的武者，不然几乎无法抵挡，而且就算是生死境高阶的武者，也只能一力破之，对于这空间法则，也没有什么好办法。”
<br/>　　一尊围观武者，惊叹一声道。
<br/>　　鲲鹏，便是远古神兽之一，跟远古战龙一样，皆是远古时代位于巅峰的存在。
<br/>　　远古战龙的强大，是极致强横的力量，而鲲鹏，则是对空间法则的领悟。
<br/>　　相传鲲鹏足以从虚空中任意穿行，足以轻易破碎一处虚空，将其中的一切都泯灭。
<br/>　　在远古时代，也只有几尊九域巅峰的存在，才可以跟鲲鹏抗衡，寻常武者，瞬间都会泯灭在虚空之中。
<br/>　　眼前这欧阳烈，居然是有着几分鲲鹏血脉，对空间法则，领悟极高，拥有如此血脉，在同境界中，近乎无敌。
<br/>　　这虚空长刀，无声无息之间，众人便是感觉的出，已经是向着楚风眠，斩了过去。
<br/>　　好像下一刻，楚风眠便是就要在这虚空长刀之下，当场陨落。
