　　对于女儿的教育，老实说，李承乾一直不怎么上心。
<br/>　　而且人家不是都说了么，穷养儿富养女，这闺女就不能管的太严，活泼一些多好。
<br/>　　重要的是，女儿嘛，早晚还不是要嫁人，到时候自然有夫家管束，他这个当爹的现在就别跟着起哄了，一共也没几年自由自在玩的时间，放任自流好了。
<br/>　　正是因为李承乾这样的一种心态，所以养成了琅琊天不怕地不怕的性格，以前在宫里李二夫妻在的时候，还能收收性子，而现如今……呵呵……。
<br/>　　……
<br/>　　话分两头，却说前任帝国主义头子，太上皇李二陛下带走了东方青云父子二人，顺带也将大唐在南海一带的势力带走了一半，这多少让南海一带的各个属国长长的松了一口气。
<br/>　　要知道，东方青云这家伙在南海一直是推行高压政策，他的儿子东方玉林同样也是如此，南海水师就是南海上最大的恶霸。
<br/>　　海盗？不好意思，南海没有海盗，随着钢铁战舰的投入使用，再也没有人敢下水为盗。
<br/>　　商人？这个可以，不过要交税，根本属国不同，交的费用也不一样，其中大唐人最少，几乎是免费的。
<br/>　　林邑人次之，毕竟林邑每年还要向大唐运送不少的粮食。
<br/>　　真腊、骠国再次之，这两个国家与大唐接壤。
<br/>　　然后就是天竺人，虽然天竺人也不知道为什么自己要交那么多的税，不过没办法，这就是王八的屁股——规定，觉得钱多可以回去，没人逼着你入境。
<br/>　　最后，就是大食人了，自从有过一次在天竺道（天竺已经被李承划为单独的一道）边境的对峙之后，大食商人就再也没有了进入大唐的机会，甚至随着时间的推移，就连天竺境内都开始限制他们的船只。
<br/>　　正是因为这样，所以南海一带的形势空前紧张，大食人为此出动了军队，数百艘战船在马六甲与大唐南海水师对峙，每天就是谈判谈判再谈判。
<br/>　　大唐无所谓，来来回回只有一句话，上面不同意。
<br/>　　可是大食人不行啊，他们放着大唐这么大的市场，他们进不去，只能看着眼馋谁也受不了不是。
<br/>　　所以在不知道谈了多少次之后，大食人终于急了，拍着桌子叫嚣不开放边境就打开边境，要让大唐知道知道他们的厉害。
<br/>　　嗯，对，就是这样，大食人打算一雪前耻，一报当年打输了海战的血海深仇。
<br/>　　可大唐会怕他们？船还是当年的小破船，人还是当年的那些人，反观大唐，战船全是铁的不说，个头儿一艘能顶他五艘还多，这还不算战舰上的各种火炮。真打起来别的不说，单单用撞的就能让大食人这几百条船跑不出百里。
<br/>　　不得不说的是，没有一个得力干将就是差劲，南海的事情只要长安任何一个纨绔在这里，根本就不用谈这么长时间，早就一声令下碾压过去了。
<br/>　　可是眼下却不行，舰队里连个主事儿的人都没有，独孤玉林跟着太上皇寻仙问道去了，拥有五艘铁甲战船的南海水师只有一个副将在，他也不敢擅自开战。
<br/>　　所以在大食人叫嚣着要开战的时候，舰队的副将立刻派人将消息送了回去，然后便严阵以待，等着大食人攻过来。
<br/>　　一天、两天过去了。
<br/>　　十天八天又过去了。
<br/>　　三个月之后，尉迟宝琪万里迢迢杀到南海，结果大食人还是没动静。
<br/>　　“你这里什么情况？不是说打起来了么？怎么回事儿？”尉迟宝琪上船之后，连休息都没顾得上，就把那副将扯了过来，急吼吼的问道。
<br/>　　他可是不眠不休的整整赶了两个月的路，人都瘦了不止一圈，本以为能赶上一场大战，结果到了南海却发现，根本没打起来。
<br/>　　“末，末将也不知道啊，那些大食人天天都在布置，也不攻过来，末将没有将令也不敢攻上去。”副将一脸的纠结，心里已经把那些大食人骂了个狗血淋头。
<br/>　　你说你要开战，那就开战好了，这磨磨唧唧的算怎么一回事儿，这特么日子还能过不了？
<br/>　　尉迟宝琪拧着眉毛听那副将说完，指着他的鼻子就骂了起来：“你个驴日的，他不攻过来，你就不会杀过去，都特么宣战了，你还想啥呢！”
<br/>　　“我……”
<br/>　　“我什么我，给老子把命令传达下去，备战，明日一早，干他丫挺的！”
<br/>　　“是！”
<br/>　　副将虽然被骂了，但好在尉迟宝琪并没有多说什么，而有了主心骨之后，副将的腰板也硬了起来，叫过一旁的传令兵，将备战的命令传达了下去。
<br/>　　……
<br/>　　翌日清晨，大海上还飘着一层薄雾，海面上除了浪花拍打船舷的声音，一片寂静。
<br/>　　“呜……”陡然间，一阵惊天的气笛声撕开了清晨宁静的海面，五舰钢铁战舰携排山倒海之势撞开重重薄雾，以闪电般的速度出现在大食舰队的面前。
<br/>　　原本还在沉睡中的大食人被响亮人气笛声从睡梦中惊醒，还没反应过来发生了什么，在外面甲板上放哨的哨兵已经大喊大叫着冲进了船舱：“不，不好，大唐人疯了，他们杀过来了！”
<br/>　　“什么？怎么回事儿，他们不是一直都……”还没等船上的船长把话说完，他的战船已经被一团巨大的黑影所笼罩，接着数十万斤重的钢铁战舰便从他的小舢板上碾了过去。
<br/>　　一船两百余水手，活下来的不足半数。
<br/>　　疯了，真特么疯了，大唐人怎么可能一言不发就开始进攻呢？没道理啊！
<br/>　　宣战的使者呢？在哪里？这特么是不宣而战好吧？
<br/>　　大食舰队的将军在看到那五艘钢铁战舰的时候，整个人都是懵的，完全忘记了在数月之前，他拍着桌子喊出的那句：“不开放边境就开战。”的狠话。
<br/>　　不过现在已经不是追究这个问题的时候了，面对大唐的战舰，他能做的就是应战，如果继续让大唐的军舰肆虐下去的话，他不知道自己的舰队还能坚持多久。
<br/>　　他的舰队排出的队形实在是太密集了，必须尽快的散开才有作战的能力。
