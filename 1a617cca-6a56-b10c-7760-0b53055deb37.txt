　　听着易老的话，萧逸下意识地问道，“难道不是？”
<br/>　　下一秒，他反应过来，暗暗惊讶，难道....
<br/>　　“不知易老是内门哪堂长老？”萧逸惊讶地问道。
<br/>　　易老淡淡一笑，道，“又是谁跟你说我是长老的？”
<br/>　　“这...”萧逸无语了。
<br/>　　萧逸想起来，前些天考核时，似乎确实都只听别人称他为易老，却并没有称呼执事或者长老。
<br/>　　“我非执事，也非长老。”易老沉声道，“我只是这外门的守库人，也只是个闲人。”
<br/>　　“就因为我是闲人，不用处理剑派的事务，不用教导弟子，所以我才有空指点你一二。”
<br/>　　“当然，还是那句话，遇到不懂的，你可以来问我，这是我代表剑派给予你的一点补偿，随你来不来。”
<br/>　　说罢，易老没再说什么，一如既往地回到桌案前看书。
<br/>　　萧逸也没再多问，微微行了一礼，道，“小子告辞。”
<br/>　　.....
<br/>　　十日后，仍旧是外门后山。
<br/>　　萧逸从修炼中醒来，双眸闪过一道精光，“修为终于完全稳固了。”
<br/>　　他站了起来，拍了拍身上的灰尘，自语道，“也该出去走一趟了。”
<br/>　　既然修为已稳固，他便要准备突破洞玄境了。
<br/>　　可是，突破洞玄境到底该如何，他虽知道一点儿，但终究没有十足把握。
<br/>　　在玄界碑上悟道的知识，更多的是境界上的感悟，并非详细的做法。
<br/>　　突破洞玄境，名为破丹、融脉、凝气泉，乃是一个破而后立的过程，虽无比困难，却能给武者带来翻天覆地的变化。
<br/>　　当然，也是因为这困难，导致他没有把握。
<br/>　　这个时候，若有修为高深的武者亲自指导，是最好的办法。
<br/>　　若没有，便只能出去历练了，通过无数次战斗，让自己在这个层次的修为中达到如臂指使、了然于胸的地步。
<br/>　　这样，在真正开突破时，把握便更大了。
<br/>　　所以萧逸打算出去历练一趟。
<br/>　　加入裂天剑派，并不代表要一直留在这里。
<br/>　　萧逸来这里，要的只是一段时间，换个说法，他要的只是一个身份，裂天剑派弟子的身份。
<br/>　　在剑派中，若能修炼得更快，实力得提升得更快，自然是最好的。
<br/>　　可现在既然不能，便只能靠自己了。
<br/>　　至于去请教易老，这个念头早就在他脑海中消失了。
<br/>　　和易老接触的次数不多，但易老的性格太古怪了。
<br/>　　沉默寡言，不喜热闹，并非好相处的人。
<br/>　　再者，自己和易老无师徒名分，非亲非故的，平白去请教，只会落下人情。
<br/>　　虽说易老可能并不在乎自己的小小人情。
<br/>　　但人情始终是人情，自己最讨厌的，就是欠下人情。
<br/>　　“呼。”萧逸深呼吸一口气，从后山上离开，往山门而去。
<br/>　　外门弟子，相对不太严格。
<br/>　　弟子若想出外，只需到守山门的外门执事处做个汇报，大概讲明出外的原因即可。
<br/>　　不多时，萧逸来到山门。
<br/>　　刚想去和执事做汇报，不想，一道人影竟凭空而现。
<br/>　　而后，人影满脸戏谑之意地盯着萧逸。
<br/>　　“易老。”守山门的执事见人影出现，连忙行礼。
<br/>　　“额。”萧逸一愣，也恭敬地行了一礼，“易老。”
<br/>　　“小子，你想出去历练？”易老没有管身旁的执事，而是看向萧逸，嘴上虽然问着，但语气显然很确定。
<br/>　　“小子，你很狂，也很傲。”
<br/>　　易老说着，径直走到萧逸面前。
<br/>　　“易老言重了。”萧逸拱拱手，道。
<br/>　　“为何不来找我？”易老沉声问道。
<br/>　　萧逸回答道，“易老不是说过，随我去不去吗？”
<br/>　　“哼。”易老冷哼一声，“小子，倒是将我的话记得很清。”
<br/>　　“那你是否记得我说过，我是个闲人？”
<br/>　　“这...”萧逸一愣，明显感觉到易老的怒气，这种脾气古怪的老头，果然难以捉摸。
<br/>　　易老冷声打断道，“我是个闲人，所以我有的是时间陪你耗下去。你不来，我偏要你来。”
<br/>　　话音刚落，易老大手一挥，一股滔天灵气直接将萧逸束缚住。
<br/>　　下一秒，易老身影一闪，消失在原地。
<br/>　　顺带着，萧逸也消失无踪。
<br/>　　两个守山门的执事看得一愣一愣的，半晌，只得无奈地摇摇头。
<br/>　　.....
<br/>　　后山，萧逸平日里修炼的地方。
<br/>　　易老和萧逸同时出现。
<br/>　　萧逸站稳身子后，皱眉问道，“不知易老是何意？”
<br/>　　“哼。”易老冷声道，“小子，我说过，我代表剑派给予你一点儿补偿。你要也得要，不要也得接着。”
<br/>　　“我...”萧逸想说些什么。
<br/>　　“闭嘴。”易老打断道，“突破洞玄境之事，我会指点你一天。一天之后，便算两清了。你要去历练便历练，我懒得拦你。”
<br/>　　“一天？”萧逸点点头，本是对易老这种古怪和霸道很是不喜，但想想，既然只是一天时间，便无所谓了。
<br/>　　一天之后，再出去历练也不迟。
<br/>　　就当敬老，顺他一天的意。
<br/>　　再者，一天时间，也指点不出什么来，还不至于落下人情。
<br/>　　“谢易老指点。”萧逸心中腹诽着，嘴上还是礼貌地说道。
<br/>　　“嗯。”易老收敛了脸上的怒气，恢复了以往那股古井无波的表情。
<br/>　　“看清楚了。”
<br/>　　说着，易老大手一挥，无数天地灵气涌来，自动形成了一道光幕。
<br/>　　光幕中，灵气自动排列，模拟出武者的四肢百骸、经脉丹田投影图。
<br/>　　图中，真气游走，活灵活现，极为奇妙。
<br/>　　“看好。”易老伸出手，在光幕上凌空指点着，“那是武者的气丹、以及九条真脉，看清楚他们的变化。”
<br/>　　“先是破丹，再是融脉，最后凝气泉；步骤不可错。”
<br/>　　“嗯。”萧逸点点头，目光紧紧地盯着投影画面。
<br/>　　他本来对易老的指点并不为意，但现在，这投影画面，却彻底改变了他的想法。
<br/>　　对比典籍的详细讲解，乃至高深武者的言传身教；这投影画面的直接展示，无疑效果更大。
<br/>　　甚至不用多去理解，照着这投影画面的步骤和变化去做就好了。
<br/>　　照葫芦画瓢，自然最直接，最容易。
<br/>　　易老继续说道，“破丹，融脉，凝气泉，皆需要精纯的力量。所需力量，视武者本身情况而定。”
<br/>　　“气丹越强，便越难破；真脉越强，便越难融；所需的力量也就更多。”
<br/>　　随着易老话音落下，光幕上的图画再次变了。
<br/>　　气丹开始破开，九条真脉也开始融化。
<br/>　　“气丹，本是武者丹田，受真气蕴养而成；凡境突破到后天境时所得。”
<br/>　　“真脉，由主脉而来。”
<br/>　　“主脉，本是人体生而堵塞的九条经脉；乃武者为了真气更流畅通过而炼化。它存在的意义，只是存储真气，并让真气流淌。”
<br/>　　“所以，即便气丹和九条真脉破了、融化了，也不会对人体有任何影响，你可以放心去做。”
<br/>　　易老不知不觉间，像极了一个学识渊博的老师，耐心地指导着。
<br/>　　“破丹、融脉，需要一个过程和时间，我便不慢慢展示了，你知道如何去做即可。下一个步骤，凝气泉。”
<br/>　　易老说着，大手一挥。
<br/>　　投影图中的画面再次一变，光幕，也陡然间变的极其巨大。
<br/>　　气丹、九条真脉，彻底融化，并以一种玄妙无比的方式凝聚成一种莫名的东西。
<br/>　　看真切些，那像极了一个荒芜的空间，空间正中央，有一干涸的水泉。
<br/>　　水泉不偏不倚，正好长十丈、宽十丈、高十丈。
<br/>　　而整个空间，则正好长二十长、宽二十丈、高二十丈，是水泉的两倍。
<br/>　　“那空间，你可以理解为一个小世界；那干涸水泉，便是气泉。”
<br/>　　“这两者，便是气丹和九条真脉所化。”
<br/>　　“你或者会疑惑，体内一颗小小的气丹和真脉，怎么可能幻化出这样庞大的空间。”
<br/>　　“你可以想象成乾坤袋，那小小的袋子，里面却内含庞大的空间。你要知道，乾坤袋本身就是人类武者创造的，修为参天的武者，甚至可以操控空间之道。”
<br/>　　“你....”
<br/>　　易老详细地解释着，却发现萧逸皱起了眉头，以为他不理解，顿时脸上闪过一丝不悦。
<br/>　　“若你这都不明白的话....”易老不悦地说着。
<br/>　　“芥子纳须弥，一花一世界。气丹为基，真脉为材，方成这气泉与空间。”萧逸沉声说着。
<br/>　　事实上，不是他不明白。而是，他完全被光幕上的变化所震撼与吸引。
<br/>　　他忽然明白，为何洞玄境是武道修炼上最重要的一道分水岭。
<br/>　　更加明白，为何突破洞玄境后会有翻天覆地的变化。
<br/>　　甚至于，他稍稍明白，为何那些传说中的武道至强者，会有那等翻手间排山倒海、破碎虚空、遨游天地的力量。
<br/>　　今天，易老的教导，仿佛为他敞开了一个新的世界，一道新的大门。
<br/>　　他开始了解，这个世界的武道，远没有自己想象的那么简单。
<br/>　　这时，易老听罢萧逸的话，愣了愣，而后露出了极为满意的笑容。
<br/>　　“芥子纳须弥，一花一世界。气丹为基，真脉为材。”
<br/>　　“不错，你这样的理解更为透彻。”
<br/>　　“一粒渺小的沙砾，一朵小花，都能内含世界；更别说武者体内的丹田和真脉了。”
<br/>　　易老说罢，继续道，“气泉，将成为日后储存真气的地方。“
<br/>(本章完)
