        不想一个人在家吃晚饭，但晚上还要去见傅夜沉，童筱颖特意精心打扮了一番。
<br/>
<br/>        平常几乎从不化妆的她，今天她脸上化了淡淡的彩妆，还特意用电卷棒，将自己的长发烫了个一次性的大波浪卷。
<br/>
<br/>        女为悦己者容。
<br/>
<br/>        童筱颖都没有意识到，自己这小小的举动，已经出卖了她自己的心灵。
<br/>
<br/>        她梳妆打扮完后，将小爱装入了宠物袋里，带着小爱一起出了门。
<br/>
<br/>        滨江城里能允许带宠物入内的高档餐厅并不多，所以，能在没有提前预订的情况下，还有空位给她坐，就已经是她运气好了。
<br/>
<br/>        当餐厅的服务员带着童筱颖进入餐厅时，隔着一扇镂空屏风，童筱颖一眼就看到了傅夜沉以及坐在对面的一个她不认识的女人。
<br/>
<br/>        那个女人一袭冷艳的绛紫色抹胸连衣裙，胸前高耸半露的雪白，性感又不失风雅，身材也玲珑有致。
<br/>
<br/>        而傅夜沉只穿了件纯黑色衬衫，未打领带，衣领口最上面两颗纽扣散开，露出男性销魂的锁骨，脖颈间凸出的喉结偶尔滑动，禁欲感十足。
<br/>
<br/>        他俩还真是天造地设的一对。
<br/>
<br/>        童筱颖也没料到，自己的心情，在此刻一落千丈。
<br/>
<br/>        服务员领着童筱颖在一个比较偏的角落里落座，而这个角度，刚好可以与傅夜沉面对面地在一根斜线上。
<br/>
<br/>        童筱颖不打算刻意回避，而是带着小爱，默默地吃顿饭。
<br/>
<br/>        服务员还给小爱推来了宠物椅，虽然可以带宠物入店，但也要求宠物带上口罩，不能使用人类餐具给宠物喂食，并且需将宠物拴在宠物椅上，不影响他人用餐。
<br/>
<br/>        童筱颖一一遵守，菜上齐后，她埋头用餐，克制自己不往傅夜沉那边看，小爱也很乖巧地坐在宠物椅上安安静静地陪着她。
<br/>
<br/>        “我这次来，除了当你的‘冒牌女友’以外，还有就是要替叔叔寻回他失散多年的那个亲生女儿。叔叔在每个城市，都安排了人去帮他找。而滨江城由我负责，夜沉哥，你是这里的东道主，不如帮帮我？”夏玄炘一边切着牛排，一边闲聊。
<br/>
<br/>        傅夜沉皱了下眉头，反问：“这么多年不去寻，现在突然要去寻了，你叔叔心里到底是怎么想的？”
<br/>
<br/>        “我听说，是爷爷想要找回这个小幺孙。爷爷年纪大了，想一家团聚，也很正常。”夏玄炘接着说道。
<br/>
<br/>        “可以，你有什么地方需要我帮忙的，尽管开口就是。”傅夜沉点了点头。
<br/>
<br/>        夏玄炘微微一笑：“那我待会把我所需要的资料，发你电子邮箱里。”
<br/>
<br/>        “好。”傅夜沉爽朗地应了声。
<br/>
<br/>        那边的童筱颖，食物一口一口地送入嘴中，却如同嚼蜡。
<br/>
<br/>        本来一直强忍着自己，不让自己将目光落向傅夜沉那边，但终究她还是没忍住，目光定定地往那边投了过去。
<br/>
<br/>        此时，他们刚好用完餐。
<br/>
<br/>        傅夜沉先行起身，但是当那个女人起身的时候，刚好撞上从她身后过来的端着两杯鲜榨果汁托盘的服务员。
<br/>
<br/>        女人的后背，撞到了服务员的托盘上，是傅夜沉眼疾手快地将女人拉向自己怀中，才没被打翻的果汁杯给泼脏。
<br/>
<br/>        服务员伸手扶住杯子，立即点头哈腰，不停地道歉。
<br/>
<br/>        傅夜沉从裤兜里掏出皮夹，随手抽了两百元放到服务员的餐盘上，十分有绅士风度地替那女人赔了那两杯果汁的钱。
<br/>
<br/>        女人也很矜持地释然一笑，并未数落服务员的不是。
<br/>
<br/>        虽然只是一个小插曲，但是看得出这一男一女的修养素质还是在线的。
<br/>
<br/>        看到这里，童筱颖掏出手机，忍不住给傅夜沉打了通电话。
<br/>
<br/>        电话没响两声就被他接起来。
<br/>
<br/>        “有事？”声音依旧冷冽，看来是还在生她的气。
<br/>
<br/>        尽管他此刻仍旧未发现她其实跟他就在同一个餐馆的大厅里，但童筱颖还是下意识地垂下眼睑，不敢再看向他那边。
<br/>
<br/>        童筱颖犹豫了片刻，才弱弱地问道：“大哥，你今天会回来吃晚饭吗？”
<br/>
<br/>        她也不知道自己为什么要明知故问。
<br/>
<br/>        或许是因为，她一时间，找不到其它的话题可以说。
<br/>
<br/>        “不回。”他那边回答得很快，这让童筱颖很是心塞。
<br/>
<br/>        “那……你今天晚上会回来吗？”她又问，声音有些哽咽。
<br/>
<br/>        “不回。”依旧是这淡漠的两个字。
<br/>
<br/>        他惜字如金，似乎不愿意跟她多少一个字，好像在他看来，跟她多说一个字都显得十分浪费。
<br/>
<br/>        就在这个时候，电话里那头还突然传来一个娇媚的女声：“夜沉哥，待会陪我去楼下的服装店里逛逛吧！”
