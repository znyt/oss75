　　场上的时间慢慢的走着，不知不觉间便是已经到了16分钟，而在这个时间段上，场上的人头比依旧是0：0。
<br/>　　“越是高端的对决，就越不容易发生人头，双方打的都特别的谨慎，除非是他们有意打架。”看着这个人头比，米粒有些感叹的开口说道。
<br/>　　“大魔王这边第一件装备不出所料的还是做出了冥火之拥这个装备，这件装备无论是减冷却还是法强的提升对于妖姬来说都是非常重要的，不过这样一来的话他倒是有些缺蓝了，第二件装备不出所料的话大魔王还是要补鬼书的。”看着叶帝的装备，林雅楠跟着说道。
<br/>　　“这一局大魔王一直在找机会击杀对手，只不过暗夜打的太谨慎了，尤其是在他升级过自己的e技能之后，清兵速度变的更快了，现在一个e技能已经是可以轻松的清掉后排兵，丝毫不给大魔王游走或者是越塔过来杀他的机会。”米粒跟着说道。
<br/>　　看着这个局势，叶帝的眉头也是皱了起来，虽然他在补兵上小小的压制了一下蒋念，不过这种情况下这点优势可是一点没有用的。
<br/>　　真要是到了后期，对方一个大招绝对是毁天灭地般的存在，除非到时候自己能先手秒掉一个人，要不然自己的作用绝对是没有对方高的。
<br/>　　不过这个时候蒋念铁了心的是想要将自己牵制在中路了，如果自己真要利用wr去清兵的话，那么到时候就算是自己去其他路gank，没有了大招的伤害，想要击杀对方也是有些难度的。
<br/>　　“18分钟准备小龙团。”看了看小龙处坐落着的那条土龙，叶帝只能将希望寄托在这条龙的身上了。
<br/>　　在这个时候打团，自己的妖姬还是要比对方强的，只要一套输出打残或者秒杀对方一个英雄，那么接下来的团战就很容易了，而且能够拿下这条土龙，对于接下来无论是推塔还是拿龙来说都要方便了很多，毕竟土龙的属性可是对防御塔和野怪造成百分之十的真实伤害的。
<br/>　　听着叶帝的指挥，香蕉几人纷纷应好，他们也都知道自家的阵容，所以几人也都是想要提早打开局势，很显然这个时候打团是最好的选择。
<br/>　　当时间到达17分钟20秒的时候，香蕉这边利用自己手中提亚马特的效果，将一波兵线推到对方塔下后，便是操控着手中的巨魔回到草丛中选择回城。
<br/>　　“这一波白队是对小龙有所想法了，香蕉已经提前向着下路走来，而这个时候south的大树才刚刚开始回城。”看到回到家就向着小龙这边走来的香蕉，米粒眼睛一亮，大声开口喊道。
<br/>　　与此同时夜雪狂歌的蜘蛛也是在向着小龙这个区域靠近着，而反观蓝队这边，除了一个上单之外，其他几个位置这个时候也都是围绕在小龙附近徘徊着。
<br/>　　“蓝队这一波并不打算放掉这条小龙，他们在疯狂的标记着对方蓝buff那个饰品眼，这是打算要让大树传送过去反包白队一波么。”看着蓝队频频在地图上标记着，林雅楠连忙跟着喊道。
<br/>　　而场上率先在小龙这里摆好阵型的叶帝几人，在做好周围的视野后，此刻已经是将小龙拉出了小龙圈子，在那里对着这条土龙攻击了起来。
<br/>　　“传送了！大树传送了！！”突然间蓝buff那里亮起的一道红光，顿时吸引了场上所有人的注意力。
<br/>　　此刻小龙的血量已经被打倒了三分之一，明显蓝队是想要在这个时候过来接盘，或者是打一波团战。
<br/>　　“不打了，先杀大树！”看到大树传送的一刹那，叶帝当机立断，直接带领着几人向着对方传送的那个位置走去。
<br/>　　看到叶帝几人的动作，楚南倒是也很果断，直接取消传送，随后操控着手中的大树向着这边走来。
<br/>　　此刻小龙已经回满血了，楚南知道对方在自己到来之前肯定是击杀不了这条土龙的，如果他们真的敢继续打的话，那么加上小龙对他们造成的伤害，这一波团战自家这边是必赢的。
<br/>　　“还打么？”看着楚南取消传送，香蕉开口问道。
<br/>　　“现在咱们状态还好，可以打，一会茄子开群体加速，我找机会将林萧或者蒋念给秒了。”叶帝果断的说道。
<br/>　　下一刻他们五人再次汇聚到了小龙入口这里，一步一步向着对方那里逼近着。
<br/>　　“撤，等楚南！”秦霄此刻也是果断，指挥着几人后撤，向着楚南的方向靠近着。
<br/>　　看到这一幕，叶帝的眉头也是皱了起来，：“夜雪找机会定住一个人。”叶帝对着夜雪狂歌说道。
<br/>　　不过蓝队那面明显不想接这一波团，他们就是想趁着叶帝几人打龙的时候，被龙消耗掉一部分血量在上，现在这种情况他们完全拖的起，所以秦霄几人倒是也不怕。
<br/>　　眼看对方五人已经汇聚在了一起，依旧是不接团，叶帝这边也是有些无奈，耗费了这么大的功夫，他也不想就这样无功而返：“准备打龙吧，一会找机会开团，一定要小心对方的动作。”
<br/>　　说话间，叶帝五人便是重新对着小龙攻击了起来，而在他们攻击小龙的时候，另一边秦霄几人也是缓步的向着这边逼近着。
<br/>　　“白队这个时候还要打龙，感觉这一波很冒险啊，如果被蓝队找到机会的话，那白队可就直接要炸了。”看着这一幕，米粒开口分析着。
<br/>　　“这也是没办法的事，白队只能靠前期打出优势，他们这边的阵容除了后期巨魔这个点占据一些优势之外，其他几个位置都是要被对方压制的。”林雅楠跟着说道。
<br/>　　就当小龙被打到半血的时候，没等到白队这边发动攻势，蓝队这一边已经是率先展开了攻击。
<br/>　　楚南的大树一个闪现已经是一马当先的冲了过来，随后他手中的w技能便是向着扇子妈这个位置捆绑了过来。
<br/>　　刹那间蓝队其余四名队员也都是群起而动，快速的向着这边冲了过来。
