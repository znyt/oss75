　　分开了之后，楚风眠离开了这海沟，突然变化成了海十四的模样，前往了一座同样属于海剑宗的城池，镇南剑城。
<br/>　　这里，已经是离开了寂海，前往了沧海星域最大的海域之一，剑海。
<br/>　　这剑海，就是海剑宗所在的海域，前往剑海，楚风眠乘坐太古蜈蚣，都是花费了足足七日的时间。
<br/>　　这沧海星域之大，比起寻常的星域，还要大上数十倍，上百倍。
<br/>　　而且全然都是一处巨大无比的海洋，海族的天堂，一路之上，不止是海族，许许多多古老的海兽，异兽都生存在这沧海星域之中。
<br/>　　这一次要混入海剑宗去，必须是要借助海十四的身份，所以楚风眠才化为了海十四的模样。
<br/>　　以现在楚风眠的实力，隐去气息化为海十四，除非是真正的天人境强者，不然便是根本无法看破。
<br/>　　海剑宗之中，虽然是有着真正的天人境强者，不过这等强者，平日里绝不会插手海剑宗内的事。
<br/>　　按照海十四的说法，那临剑海帝，数百年间都很难露面一次，绝大多数时间都是在闭关潜修。
<br/>　　只要不是碰到临剑海帝，楚风眠这样混入到海剑宗内，倒是不成问题。
<br/>　　至于海公主跟奇阳师那一边。
<br/>　　他们二人，本就是九劫古帝的境界，奇阳师恢复了伤势，更是一位九劫古帝巅峰的强者。
<br/>　　这样的人物，在沧海星域之中，不能算是一方霸主，也算是强者了，只要是不招惹到海剑宗，海祭宗这等庞然大物，也没有什么危险。
<br/>　　毕竟他们二人的气息都被楚风眠遮掩，看起来只是寻常的龙族妖神，这海剑宗，海祭宗的人去查，也不可能查到龙族的身上。
<br/>　　蓝家的事，楚风眠也懒得管，他现在的目的，便是夺取剑道图腾，然后在找个机会，杀了蓝冥。
<br/>　　至于其他的，便是只有看看有没有机会，去寻找仙石，为踏入半步天人境做准备。
<br/>　　“主人，前方就是镇南海城了。”
<br/>　　一路顺着海十四的指引，就在楚风眠的面前，一座巨大无比的城池便是映入了双瞳之中。
<br/>　　这镇南海城，远远看起来，比起之前楚风眠去过的寂海城要大得多。
<br/>　　一道巨大无比的剑阵，笼罩这整个镇南海城，哪怕是半步天人境强者，去尝试抗衡这剑阵，都会瞬间被杀。
<br/>　　从海十四哪里楚风眠知道的，这镇南海城，乃是海剑宗的门户之一，有着通往海剑宗的大门。
<br/>　　就像是剑道城跟剑源界的关系。
<br/>　　不过海剑宗无比巨大，这样的门户，不止一处，这镇南海城，便是位于剑海的南端。
<br/>　　在往南，便是另一处海域，永恒祭海。
<br/>　　这永恒祭海，就是海祭宗的所在，也是沧海星域的中心。
<br/>　　楚风眠收敛起气息身形，化为海十四的模样，看起来海十四独身一人一样，进入到了镇南海城之中。
<br/>　　“十四公子。”
<br/>　　镇南海城守城的弟子，见到楚风眠到来，纷纷恭敬的行礼道。
<br/>　　这些镇南海城的海剑宗弟子，在外看来，一个个都是地位超然的存在，不可招惹的人物。
<br/>　　可如今在楚风眠的面前，却都是毕恭毕敬的。
<br/>　　海十四，乃是海剑宗的核心弟子，更是被赐名海字。
<br/>　　按照海剑宗古老的规矩，被赐名海字的核心弟子，未来都是有着争夺海剑宗宗主之位的资格的。
<br/>　　当然正常情况下，海十四是没有这个能耐的，不过海剑宗历史之上，也未必是没有排名不高的核心弟子，得到惊天奇遇争夺到宗主之位的。
<br/>　　对其他的海剑宗弟子来说，海十四几乎是少宗主一样的人物，自然是没有人敢不敬，甚至许多弟子都是走了过来，恭维道。
<br/>　　“十四公子，听说您这一次出马，是发现了蓝家余孽？不知道是否得到了玄鲸血脉？”
<br/>　　一名海剑宗弟子，更是凑到了楚风眠的身边来，恭维的说道。
<br/>　　“十四公子一旦是得到了玄鲸血脉，那其他的几位公子，哪里还是十四公子的对手？倒时候十四公子横扫了诸位公子，夺得宗主之位，指日可待。”
<br/>　　“滚！”
<br/>　　楚风眠冷哼一声，故作愤怒的模样，呵斥道。
<br/>　　楚风眠表现而出的语气模样，都给真正的海十四一模一样。
<br/>　　楚风眠以诸天生死印控制了海十四，海十四的记忆，都落入到楚风眠的脑海之中。
<br/>　　而且真正的海十四，也还是被楚风眠关在了境王布袋之中，以防万一，楚风眠随时都可以唤出他来。
<br/>　　现在的楚风眠，几乎就是跟真正的海十四一样，只有跟海十四关系无比亲近的人，才可以区分出一点点区别。
<br/>　　不过海十四生性狂妄，在海剑宗内，几乎没有跟他亲近的弟子，这倒是免去了许多的麻烦，也让楚风眠少了很多的顾忌。
<br/>　　“是，是。”
<br/>　　听到楚风眠呵斥，其他围上来的海剑宗弟子，急急忙忙都是四散退去了。
<br/>　　以他们的身份，可不敢去招惹这位“海十四”。
<br/>　　一直是看着楚风眠离开的背影，那些海剑宗弟子才纷纷怒骂了起来。
<br/>　　“该死，这海十四太狂妄了，他在海姓的核心弟子中，只是中下等，居然是这么狂妄。”
<br/>　　“算了，小声点，不要让他听见了，这海十四哪怕是当不上宗门，未来也是一位长老，我们招惹不起。”
<br/>　　“看样子这一次十四公子，都是吃了一个大亏啊，不然不会如此愤怒。”
<br/>　　“不是说蓝家已经没落了吗？怎么还能让海十四吃个大亏？”
<br/>　　“也许是蓝家留下的手段，毕竟蓝家可是玄鲸血脉，曾经也是强横一时，也许有着什么留下的手段，也不足为奇。”
<br/>　　“既然十四公子没有成功，那蓝家的人，只怕还在沧海星域之中，我们若是可以寻找到，那可是天大的机缘，玄鲸血脉，先天神兽血脉啊，我若是得到，未来都有着冲击宗主的机会。”
<br/>　　“别做梦了，诸位公子都纷纷从圣地中动身，打算去找这蓝家余孽了，我们还是好好拉拢其他人吧，你看，十三公子也回来了。”
<br/>　　又是一道身影降临到了镇南海城之中，这些海剑宗弟子，才停下了议论，急急忙忙的过去迎接了。
