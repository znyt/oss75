<br/>　　“666666666666666，这个真的有些牛逼了，如果狐狸有这么高的伤害，这个英雄完全可以上场比赛了。”
<br/>　　“你们注意到没，大魔王出的是那件刚刚重做之后的冥火之拥，刚刚击杀机械先驱的时候，那个装而比就已经冷却了。”
<br/>　　“我去，你们看看冥火之拥的属性，这个真的有些牛逼了，直接能够打掉对方最大生命值百分之十五，而且还会为后续输出提高百分之二十，这对于一个法师来说简直就是bug级别的武器啊。”
<br/>　　“无敌的大魔王，有了这件装备这个狐狸真的可以上场了啊，要伤害有伤害，而且清线和游走能力都是首屈一指的，真的有些无敌了。”
<br/>　　“也不能那么说吧，狐狸这个英雄还是有些太耗蓝了，而且在没有做出冥火之拥前还是很难过的，狐狸这个英雄能不能上场我不知道，但是我知道冥火之拥这件装备肯定是要火了，接下来就都是ap中单的时代了。”
<br/>　　看到叶帝将对方的中单瞬间秒杀，直播间里的无数观众纷纷开口议论着，每个人的眼中都是流露出不可思议的神色。
<br/>　　而叶帝这边能够这么轻易的秒杀机械先驱，不仅仅是有着冥火之拥这件装备的缘故，更重要的是狐狸如今版本的e技能依旧是能够对被魅惑的敌人所造成的魔法伤害会提高20%。
<br/>　　并不是后来被削弱的只可以魅惑对手的技能，而有着这百分之二十的伤害提高，在加上冥火的伤害提高，这使得狐狸这个英雄完全能够在一瞬间打出超级高的爆炸伤害来。
<br/>　　看到自己这么轻易的秒杀对手，叶帝心里也是有些开心，在他内心里他还是非常喜欢这种瞬间击杀敌人的快感的。
<br/>　　接下来的局势几乎完全沦为叶帝一个人的秀场了，只要他手中的冥火之拥转好，那么就代表着对方肯定有人要交出自己的性命了。
<br/>　　最终这一局游戏在时间到达26分钟的时候便是早早结束了，而叶帝手中的战绩则是豪华的9-0-5，在这种局中能够打出超神战绩来，这可是非常不容易的。
<br/>　　看到游戏结束，直播间里的弹幕顿时又是满屏刷6的评论，紧接着大家自然是让叶帝赶紧唱歌了。
<br/>　　“一局游戏已经打完了，大魔王赶紧唱歌啊。”
<br/>　　“草，刚刚说好的要在游戏里唱一段啊，我怎么一点都没听到。”
<br/>　　“大魔王说话不算数，表示鄙视一秒。”
<br/>　　“66666666666666，快点唱歌，我的大刀早就已经饥渴难耐了。”
<br/>　　“今晚我想吃大魔王下面。”
<br/>　　直播间里的众人纷纷开口在那里说着。
<br/>　　“咳咳，刚才打游戏打的太认真了，将这个给忘了，不好意思，真的太不好意思了。”看着弹幕上的言论，叶帝尴尬的咳嗽了两声，刚刚他是真的将这件事给忘了。
<br/>　　“不过大家也不用急，马上我就给大家演绎这首歌曲的完整版，希望大家能够喜欢，不过在这之前我还是有几句话想要说一下，也算是提前给大家打一声招呼，要不然到时候可能会显得太过突兀了。”看着直播间里的人数已经突破了8个亿，感觉差不多了，叶帝便是开口说道。
<br/>　　看着叶帝突然一副认真的面孔，众人还真的有些不习惯。
<br/>　　“你说事你就说事啊，干嘛弄出这么一副认真的样子，看的我心里有种不好的预感。”
<br/>　　“就是啊，大魔王你笑着说话，千万别这样严肃。”
<br/>　　“我总感觉好像要有什么大事发生是的。”
<br/>　　直播间里的众人纷纷开口说着，而在看到他们的留言后，叶帝的脸上果然浮现出了一副笑容，不过那笑容怎么看起来都有些牵强。
<br/>　　“其实这个决定我翻来覆去已经想了很久了，今天终于是下定了决心，也许你们可能没发现，但是我现在的实力确实是在下降着，就在刚刚那一场排位赛中，我就出现了一次失误，这在以前是绝对不可能出现的，所以今年可能是我最后的职业巅峰期了。”叶帝开口说道，眼神中还有些怀恋。
<br/>　　地球五年征战，来到这里征战两年，对于这个游戏他真的有太多的舍不得，如果可以他真的想在玩十年，只不过他清楚的感觉到他的身体已经不再适合在场上待着了。
<br/>　　来到这个世界虽然自己的手速已经提到了巅峰期，但是这将近两年的征战，其实叶帝有一点一直没有跟别人说，那就是他始终都被伤病困扰着。
<br/>　　在无数个夜里他都是在自己的床上翻来覆去，肩膀疼的他睡不着觉，只不过第二天早上他还要装出一副没事的样子继续锻炼。
<br/>　　他的伤病在半年多以前就出现了，只不过那个时候他们正忙着打外卡赛，然后世界赛，所有的事情都连在了一起，那种情况下根本不可能给他休息的时间的。
<br/>　　所以这才让他的病情越来越严重，拖到领土战结束的时候，虽然叶帝悄悄的去治疗了一番，不过还是留下了病根，毕竟耽误了这么长时间。
<br/>　　之前一段时间他的身体还好好的，不过最近肩膀又疼了起来，这个样子让叶帝知道，他距离自己的职业生涯结束的时间已经是所剩无几了。
<br/>　　与其在赛场上混吃等死，倒不如早早的结束，留下一个最巅峰的身影，在地球上自己没有做到，在这里他要留下一个最好的自己。
<br/>　　“所以我决定在今年的领土战结束之后就退役，感谢大家一直以来的支持，不过虽然我退役了，但是我依旧不会离开这个赛场的，我还会陪着大家继续走下去的。”终于叶帝将自己内心深处的那句话说了出来。
<br/>　　说完之后叶帝心中既是不舍，又是有些轻松，种种复杂的情绪萦绕在他的心头。
<br/>　　而直播间里还有叶帝身后的蒋念等人听到这句话后，一个个却都是心下一震。
