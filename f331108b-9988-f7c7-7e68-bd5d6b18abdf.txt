　　第395章
<br/>　　灯火飘摇，李乐天抬起头，发现明帝的脸色是从未有过的难看。眼中的神采，也如飘摇的烛火一般，随时都可能熄灭。
<br/>　　“父王，敌弱我强，匪贼不过是一群乌合之众，最后必然会被击溃。”
<br/>　　李乐天如此说着，但不知是安慰明帝，还是安慰自己。的确，无论从任何方面说，帝国朝廷在这场战斗中都占绝对优势。怎么看，最后胜利的也是朝廷，而不是正义教的匪贼。
<br/>　　可战争，并不是摆在桌面上的数据对比，做一下加减乘除，就能决定最后的胜败。
<br/>　　即便在所有方面都占优的情况下，李乐天也和所有人一样，有种无力回天的感觉。如今的长安城上空，布满了彤云，所有人都看到一场大雪将铺天盖地席卷而下。
<br/>　　这是大势，既是大势所趋，也是大势已去。
<br/>　　明帝起身，从书架上打开一个暗间，又从其中取出一个木盒。
<br/>　　李乐天的视线追随着明帝的身影，见明帝的动作已有几分趔趄，他果然已太过苍老。如此简单的动作对他而言，都已是很困难的事情。
<br/>　　木盒打开，里面是一柄剑。
<br/>　　剑长三尺六寸五分，八面磨锋，青铜护手，剑背纹着两个字：山，河。
<br/>　　明帝二十二年，曾采天山铁，铸剑十四，天子剑一，诸侯剑十二，匹夫剑一。
<br/>　　匹夫剑辗转落到程大雷手中，诸侯剑赏给十二位王子，至于天子剑则代表着帝国的最高权力，明帝亲自执掌。
<br/>　　这是山河剑，也是天子剑。
<br/>　　明帝双手捧剑，递到李乐天面前：“若长安城破，你便拿着这柄剑逃走，留得青山在，不怕没柴烧。”
<br/>　　李乐天当然明白这柄剑代表什么，谁若握这柄剑，谁就等于握着天下。他曾经当然也期盼过，可如今握着这柄剑，感觉到的只有沉甸甸的压力。
<br/>　　他慌张跪倒，捧剑在头顶，道：“儿臣誓死守卫长安城。”
<br/>　　明帝坐在龙椅上，忽感觉有些心灰意冷，无论从哪个方面讲，李乐天都是最适合接任帝位的人。但说也奇怪，在诸多皇子中，他最宠爱的反而是那个不成器的六皇子。从一开始，他便有意将皇位传给其，可无可奈何的是，六皇子实在有些不争气。
<br/>　　如今风雨飘摇，大武气数将近，明帝不由得又想起他，心里念叨了一声：老六现在不知在做什么。
<br/>　　……
<br/>　　凉州城，城主府。
<br/>　　软榻之上，两具身体缠绕在一起，伴随着一阵一阵的娇喘。
<br/>　　良久，阿珠从床榻上坐起来，肩膀和脖颈处有阵阵青紫。她回眸一笑，看着塌上的男子，道：“殿下，我伺候您吃些什么。”
<br/>　　李行哉靠在枕头上，微微眯着双眼，欣赏着阿珠的美背。无论如何，阿珠都是一个不错的女子。有精致的脸庞，水做的肌肤，一双健美的长腿，既可以在银盘上起舞，也能踢碎男人的下巴，当然，也能够在床上有力的缠住男人的腰。
<br/>　　他的手顺着小臂，攀上了阿珠的肩膀，道：“我肚子不饿，但其他方面还没有吃饱。”
<br/>　　阿珠轻笑着，头靠在李行哉胸口，手指顺着小腹滑下去。
<br/>　　如今，很多诸侯都会在府中豢养一些歌姬。她们大多是孤儿，接受专业的教导，琴棋书画，诗词歌赋，擅长的就是如何讨一个男人欢心，同样也擅长刺探情报和杀人。
<br/>　　李行哉抿起嘴角微微一笑，手顺着阿珠的腰肢攀援上去，然后重重将她压在床上。
<br/>　　“殿下，您轻点，我怕疼。”
<br/>　　“放心，我一向是很温柔的。”
<br/>　　李行哉凑在她耳边，低低的声音说着，然后手爬上了她修长的脖颈，再然后重重掐住。
<br/>　　阿珠猛地睁大了眼睛，四肢如蛇一般挣扎，竭尽全力想要抓住什么。
<br/>　　可她终究什么都抓不住，片刻之后，身体便失去了所有力气。四肢摊开，眼睛外突，眼球布满血丝。
<br/>　　李行哉坐在床上呼哧呼哧喘气，扯了一张薄毯盖在身上。进入冬天，天气已经有些凉了。
<br/>　　此时大门被推开，福德勒跨步进来，他左手提着一个人的脑袋，右手的短刀还在滴血。
<br/>　　“你那边也解决了？”李行哉道。
<br/>　　福德勒点点头，把人头抛过来，骨碌碌旋滚到床边，又弹开。人头的面孔朝上，睁大眼睛看着李行哉，正是城主府的另外一名歌姬阿翠。
<br/>　　福德勒目光在房间内扫了一眼，撇撇嘴道：“先睡了人家，又杀了人家，这样做是不是有些不地道。”
<br/>　　“呵。”李行哉冷笑一声：“你一个戎族，难道还要与我讲仁义礼智信。”
<br/>　　福德勒愣了会儿，似乎是在琢磨李行哉的话。随后点点头，步到床边，一手摁住阿珠的脑袋，一手将刀摁在对方脖子上，猛地发力，血从腔子里喷出，洒得到处都是。
<br/>　　“喂，喂，都喷到我身上了。”李行哉跳起来大叫：“你就不能提前说一声。”
<br/>　　这个时候夜已经比较深了，宋伯康已经入睡。正在此时，城主府突然冒起黑烟，烟里夹杂着火，火光冲天。
<br/>　　“走水啦，走水啦。”
<br/>　　外面传来锣鼓的响声，还有一阵阵大叫。睡梦中的宋伯康被惊醒，揉了揉眼睛走出房门。
<br/>　　“哪里失火了？”
<br/>　　“正是殿下住的院子，如今正组织人救火。”伺候的仆人道。
<br/>　　“什么！”
<br/>　　宋伯康猛地一惊，睡意荡然无存。如今六皇子对他来说，还是十分重要的，就像手头一笔将其珍贵的资产。
<br/>　　这『资产』有任何损失，后果都是难以承受的。
<br/>　　他慌慌张张跑出屋子，只来得及穿了一件袍子，以最快的速度来到失火地点。
<br/>　　黑烟翻天，直冲星斗，靠近就闻到一股刺鼻的气味。幸运的是，院子周围的树木都被伐干净，火势没有向外蔓延的趋势。
<br/>　　但就算把整个城主府烧光，宋伯康也不容六皇子有任何闪失。
<br/>　　“殿下在哪里，殿下在哪里？”宋伯康大叫：“殿下还在火场么，一定要将殿下救出来。”
<br/>　　“嗨，宋大人……”李行哉吸溜着鼻子从一边走过来：“我一点事都没有，哈哈，没想到吧。”
<br/>　　
<br/>(本章完)
