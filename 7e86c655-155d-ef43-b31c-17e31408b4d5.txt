　　苏莫无奈的摇了摇头，随即便静心打坐了起来。
<br/>　　时间缓缓流逝，约莫过了一个时辰之后，阎魔鬼圣终于有了动静。
<br/>　　“小子，若汝所说一切为真，那汝就是这个宇宙最大的变数！”阎魔鬼圣淡淡的说道。
<br/>　　“什么意思？”苏莫不解的问道。
<br/>　　“汝无需知晓，只需努力修炼便可！”阎魔鬼圣叹道，他的心中着实被震惊了一把，半晌才缓过神来。
<br/>　　以他的实力，他的眼界，这个宇宙能让他震惊的事已经不多了，但苏莫的武魂，还是让他震惊了。
<br/>　　他知道，苏莫若是不中途夭折，将来必将逆天崛起，成为惊天动地的强者，横扫寰宇，威震万古。
<br/>　　这种逆天武魂，绝对能排进太古武魂榜前五，乃至前三。
<br/>　　至于为什么不能排进第一，因为阎魔鬼圣也不清楚，那排名前几的武魂，到底有多么强大。
<br/>　　而苏莫的天赋，也让得阎魔鬼圣心中不已，他想要逃脱，看来要从长计议了！
<br/>　　“我自然会努力修炼，还望前辈能悉心指导！”苏莫微笑着说道。
<br/>　　他言语虽然如此，但心中并不会完全信任对方。
<br/>　　“嗯！吾会耐心指导你！”阎魔鬼圣幽幽的说道，至于其心中所想，那就无从得知了。
<br/>　　“那好，请问前辈，我这种武魂进化战魂，参悟何种奥义最为合适？”苏莫凝声问道。
<br/>　　“当然是吞噬奥义！”阎魔鬼圣道。
<br/>　　呃~
<br/>　　苏莫闻言一怔，随即惊讶的问道：“还有吞噬奥义？”
<br/>　　吞噬奥义，这和他的吞噬武魂，实在太契合了，连名字都一样。
<br/>　　“当然，汝参悟此奥义，才能将汝吞噬武魂的能力发挥到最大！”阎魔鬼圣道。
<br/>　　“那好，我就参悟此奥义！”苏莫心中大喜，立刻便做出了决定，不用想他也知道这么奥义绝对适合他。
<br/>　　总算有了选择，接下来他就是奥义的参悟了。
<br/>　　随即，苏莫立刻离开了自己的阁楼，前往神武学府的奥义圣山。
<br/>　　奥义圣山，并不是设立在学府中，而是设立在一个小世界中。
<br/>　　小世界的入口，便在一座叫奥义大殿的空间中。
<br/>　　很快，苏莫便来到了奥义大殿，从入口处进入了奥义圣山所在的小世界中。
<br/>　　这片小世界不大，方圆不过千余里，世界的中央耸立着一座通天巨峰。
<br/>　　此峰占地方圆万丈，高达数万丈不止，犹如通天之柱，直插云霄之中。
<br/>　　此山极为奇怪，有的地方狂风呼啸，有的地方大雪漂飞，有的地方燃烧着熊熊烈焰。
<br/>　　当然无论是大雪还是火焰，又或是其他一切的异象，都显得虚幻无比，并不是真实存在。
<br/>　　而此刻，在奥义圣山的四周，足足聚集了上千名神武学府的弟子，均是盘膝在地，在参悟奥义。
<br/>　　苏莫走到了奥义圣山千丈之内，眼眸扫视整个圣山，灵识仔细的查探。
<br/>　　但是，查探了良久之后，他非常的失望，因为这奥义圣山上虽然蕴含的奥义种类有上百种，却并没有吞噬奥义。
<br/>　　这奥义圣山上的奥义，均是比较普通的奥义，比如五行属性奥义、阴阳奥义、风之奥义、雷电奥义，狂暴奥义等等。
<br/>　　甚至，这圣山上连不朽奥义都没有，奥义的种类远不如苍穹世界的战魂界。
<br/>　　不过，这里的奥义之力极强，对于武者的参悟，远不是战魂界可比。
<br/>　　哎！
<br/>　　苏莫叹了口气，没有吞噬奥义，他该如何参悟？
<br/>　　难道就靠自己琢磨领悟，那也太慢了。
<br/>　　“小子，汝可以从自己的武魂中参悟！”这时，阎魔鬼圣提醒道。
<br/>　　“还可以这样？”苏莫闻言一怔。
<br/>　　“当然！”
<br/>　　毫不犹豫，苏莫立刻盘膝坐在了地上，心神沟通吞噬武魂，并且微微催动了吞噬武魂。
<br/>　　霎时间，微弱的吞噬之力，便弥漫而出。
<br/>　　苏莫心神仔细感受这股吞噬之力，感受其中的玄妙，心中慢慢的有了一丝眉目。
<br/>　　这吞噬之力，虽然不是奥义，但其中却蕴含吞噬的大道规则，比参悟那些奥义之力，更加的事半功倍。
<br/>　　苏莫顿时大喜，这真是家有一老，胜过一宝啊！
<br/>　　这阎魔鬼圣，真是对他帮助太大了。
<br/>　　不由得，苏莫想到了自己拒绝火元圣师的事情，自己真是蠢啊！
<br/>　　他若是不拒绝的话，虽然要学习炼器，但修炼武道的知识，也可以随时向对方请教。
<br/>　　但好在他运气不错，又遇到了阎魔鬼圣。
<br/>　　很快，苏莫便抛却了心中的杂念，努力参悟吞噬奥义。
<br/>　　他完全沉浸在了其中，忘却了时间的流逝。
<br/>　　时间如白驹过隙，很快便过去了一个月的时间。
<br/>　　这时候，奥义圣山之前，又来了两名弟子。
<br/>　　这两名弟子，均是二十余岁左右的青年，一人身穿黑衣，一人身穿华服。
<br/>　　那名身穿黑衣的青年，个子不高，脸颊消瘦，面色冷峻，眸光如刀锋，冷厉无比。
<br/>　　而另外一人，正是上次在魔云海之时，被苏莫反敲诈的华服青年。
<br/>　　“冷大哥，就是这个小子，我已经打听清楚了，他叫苏莫，就是他在魔云海抢劫了我，你一定要帮我报此仇！”
<br/>　　华服青年望着不远处的苏莫，低声对身边的黑衣青年说道，言语中充满了怨恨。
<br/>　　“一个准帝境的蝼蚁，居然也敢抢劫，真是找死！”黑衣青年闻言，眸中杀机一闪，抬脚就要上前。
<br/>　　“冷大哥且慢！”华服青年却立刻出言阻止了对方。
<br/>　　“怎么了？”黑衣青年不解的问道。
<br/>　　“此人实力很强，最主要的是有背景！”华服青年沉声说道。
<br/>　　“什么背景？”黑衣青年皱眉问道，苏莫的实力他倒是不在意，准帝境的修为，再厉害又能厉害到哪里去！
<br/>　　但背景却是不得不考虑，一旦得罪了得罪不起的人，他们两人都要吃不了兜着走。
<br/>　　“据闻，此人和火元圣师有关系，火元圣师曾经要收此人为徒，被他拒绝了！”华服青年凝重的说道。
<br/>　　“火元圣师？”黑衣青年闻言，立刻心中一惊。
<br/>　　随即他心中沉吟了起来，若是对方和火元圣师有关系的话，那就不能直接杀了！
<br/>　　略一沉吟，黑衣青年心中便有了主意，再次向苏莫走了过去。
<br/>　　华服青年见此，也立刻跟了上去。
