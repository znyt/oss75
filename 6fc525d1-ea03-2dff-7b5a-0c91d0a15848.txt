　　实力到了斗尊这种层次，若非灵魂一起泯灭，依旧有着复活的机会。
<br/>　　比如药尘，只要给他炼制一具能够容纳灵魂的身躯，那么他就能再度复活，并且回归巅峰时期。
<br/>　　或者，本身拥有强大的灵魂，像韩枫那种，灵魂强大的也能存活下来，当然，他们的灵魂没有受损。
<br/>　　而眼前天火尊者的完全不同，灵魂消磨的相当严重，在拖延一些时间，恐怕就要魂飞魄散了。
<br/>　　慕清霄依旧是平静的听着，天火尊者瞥了眼幼生期的陨落心炎，目光有些复杂。
<br/>　　“经历了这么多年，没想到当初的异火种子，已经进化成了心火，异火可以给你，但老夫也有要求。”
<br/>　　天火尊者望着慕清霄，见他一副人畜无害的模样，与身上的独特气质，无奈只能选择信任。
<br/>　　否则，等待他的只有死亡。
<br/>　　“前辈请说。”
<br/>　　他也有事情需要天火尊者帮忙，只要后者要求不超出他能力范围，他都可以接受。
<br/>　　况且，天火尊者对他坦白，说明此人心性不坏，而且他会提什么要求，慕清霄心里有数，无非就是助他修复灵魂。
<br/>　　“你能炼化陨落心炎，而且能穿过蜥蜴族群，来到这里，说明你本事不小，帮老夫修复灵魂，要求任你提。”
<br/>　　闻言，慕清霄沉吟起来。
<br/>　　见他思索的模样，天火尊者眼中闪过一丝赞赏，若真是一口答应他，他反而会觉得不安心。
<br/>　　若非慕清霄的皮囊，天火尊者几乎认为，自己面前的不是二十岁左右的青年，而是一名历经沧桑的糟老头。
<br/>　　“前辈的要求，我答应了，不过，晚辈也有要求。”
<br/>　　“老夫先前许诺过，有任何要求，你尽管提，只要老夫有的，或者能办到的，都会力所能及的满足你。”
<br/>　　微微点头，慕清霄道：“斗技丹药，晚辈都不需要，不知前辈，可知道如何封印厄难毒体？”
<br/>　　“......厄难毒体！”
<br/>　　慕清霄的要求，令得天火尊者有些诧异，不禁瞥了他一眼，眼底闪过一抹难以察觉的思索。
<br/>　　“厄难毒体，你小子可真是问对人了，老夫敢打包票，整个斗气大陆，比我更了解厄难毒体的，绝对不超过五指之数。”
<br/>　　见他自信的神色，慕清霄道：“那前辈，可知道如何封印厄难毒体？”
<br/>　　“算你小子走运，老夫正好知晓封印毒体的方法，这种封印之法，如今大陆上，恐怕知者甚少，因为当年，我也有一位拥有厄难毒体的朋友。”
<br/>　　说着，老者眼中闪过一丝怀念。
<br/>　　“封印的方法，正是我为其所创，只可惜，这封印也只能施展一次，并且，顶多封印三五年左右。”
<br/>　　闻言，慕清霄总算是了解一桩心事，如此一来，小医仙暂时算是安全了，封印三五年，足够了。
<br/>　　“还请前辈告知，修复灵魂的事情，晚辈会竭尽全力。”
<br/>　　“甚好，此封印名为‘天都火印’，乃是借助火属性功法的刚正暴烈压制毒气，不过如今你体内拥有异火，便能将封印效果，施展到极限，下面就是封印之法......”
<br/>　　天火尊者话语声刚落，一股信息就涌进慕清霄脑海中，旋即被其迅速接收。
<br/>　　当然，接收过程中，慕清霄一直保持着心神守一的状态，无比谨慎。
<br/>　　哪怕后者在此时，用灵魂攻击他，也绝对讨不到好处。
<br/>　　毕竟，在这种世界中，人心险恶，不得不防，他与天火尊者刚接触半刻钟而已，绝对没有可能，无条件的信任他。
<br/>　　若是后者用灵魂攻击自己，那么，九霄琉璃焱就会第一时间泯灭他的灵魂。
<br/>　　半响过后，慕清霄紧闭的双眼，终于是徐徐双眼，眼中闪过一丝惊喜。
<br/>　　用天火尊者传授的‘天都火印’，配合他的九霄琉璃焱，慕清霄有自信，能够将厄难毒体，封印到七至八年。
<br/>　　察觉到慕清霄露出喜悦之色，天火尊者一愣，道：“看来，患有厄难毒体的女子，对你很重要啊。”
<br/>　　“多谢前辈，封印之法对晚辈来说，确实无比重要。”
<br/>　　说罢，慕清霄又道：“承诺前辈的，晚辈会完成，但需要一些时间，至于前辈的灵魂，晚辈会想办法。”
<br/>　　“可以，希望老夫没看错人。”
<br/>　　天火尊者深深的看了一眼慕清霄，他非常清楚，面前的青年有许多的秘密，却也没有迟疑。
<br/>　　挥手一掷，悬浮在骨骸上的幼生期陨落心炎，便是漂浮到了慕清霄的面前。
<br/>　　因为不宜暴露太多，慕清霄调动灵气，包裹着陨落心炎，随后取出容器，小心翼翼的放入其中。
<br/>　　容器只是普通的玉器，有灵气维持，虽只能保存一段时间，但也是足够了。
<br/>　　望着慕清霄的举动，天火尊者叹息一声，握住一枚雪白色的纳戒，望着身下的骨骸，道：“麻烦将我的骨骸也收起来，斗尊强者的骨骸，也是有一些特殊用处的。”
<br/>　　对于这种小要求，慕清霄自然不会拒绝，手一卷，骨骸就装进了纳戒中，而天火尊者的灵魂，也依附在雪白色纳戒上。

<br/>　　慕清霄微微点头，道：“前辈请放心，它们不懂空间之力，自然不可能发现我们的踪迹。”
<br/>　　说着，慕清霄就将全身气息收敛，周围空间一阵扭曲，身影就消失不见。
<br/>　　“空间奥妙，你小子才九星斗皇吧？”
<br/>　　见慕清霄领悟了空间奥妙，而且非常娴熟，天火尊者惊讶无比，难怪能只身一人来到此处。
<br/>　　朝着岩浆上空掠去的同时，慕清霄不禁朝岩浆底部看了一眼，他当然清楚下面存在着什么东西。
<br/>　　陀舍古帝洞府！
<br/>　　“不知道蜥蜴族是如何出现的，世界果然无奇不有，这里还不是最深处，不过，最深处也不是我现在能前往的。”
<br/>　　听闻慕清霄的话语，天火尊者神色一滞，脸庞上陡然涌现一抹难以掩饰的恐惧。
<br/>　　“小子，我劝你赶紧打消这个念头，我虽然不知道下面有什么东西，当初却模糊的感应到，下面存在这一种极为可怕的存在。”
