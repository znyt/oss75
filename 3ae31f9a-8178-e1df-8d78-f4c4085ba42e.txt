        “我身心干净，从未谈过恋爱，对婚姻忠诚。我懂勤俭持家，会洗衣服做饭。而且我家庭背景简单，父母工作稳定，傅昼景先生愿意娶吗？”
<br/>
<br/>        童筱颖几乎是一鼓作气地吐出这些话，但她这一路说下来，声音却是越说越小，眼神也逐渐扑朔迷离。
<br/>
<br/>        自己撒了谎，一个弥天大谎！
<br/>
<br/>        童筱颖说完后，做贼心虚地低下头去。
<br/>
<br/>        完了完了！她没得救了！她竟然对他撒了谎！
<br/>
<br/>        随即耳畔传来一阵“呵呵”的讥笑声。
<br/>
<br/>        童筱颖怔住，忐忑不安。
<br/>
<br/>        不过……
<br/>
<br/>        值得庆幸的是，这个讥笑声，并不是傅昼景的。
<br/>
<br/>        童筱颖抬头寻声望去，只见邻座有一个身着黑色西装革履，看起来年近三十的男人站起身来。
<br/>
<br/>        “遇见熟人了，我去打个招呼！”
<br/>
<br/>        那男人对自己在约的女士说道。
<br/>
<br/>        然后，那个男人一边“呵呵”地笑着，一边朝童筱颖这里走过来。
<br/>
<br/>        “童筱颖小妹，我们又见面了。”男人走到童筱颖身旁，用力地拍了拍童筱颖的肩头。
<br/>
<br/>        童筱颖被这男人拍得够呛，一脸茫然地仰头看着这个一手搭在自己肩头的男人，问：“你是？”
<br/>
<br/>        童筱颖觉得这个男人有点眼熟，但是怎么也想不起自己在哪跟他见过，莫非以前的相亲对象之一？！
<br/>
<br/>        男人俯视着童筱颖，咧嘴笑道：“啊哈！童筱颖小妹还真是健忘了！就上个星期，我们不也在这个餐厅里相的亲吗？”
<br/>
<br/>        经男人这么一说，童筱颖这才猛然想起。
<br/>
<br/>        果然如此，真的要怪就怪她相亲对象太多，令她忘记也是在所难免的事。
<br/>
<br/>        “你、你好！”童筱颖强颜欢笑，赧然地将这男人的手从自己肩头挪开，心头有种不祥的预感，扰乱了她的心田。
<br/>
<br/>        谁知，这男人像是跟童筱颖很熟一样，挤了挤童筱颖，迫使她向里挪出一个位置。
<br/>
<br/>        然后，这男人一屁股在童筱颖的身旁随意坐下，令童筱颖尴尬不已。
<br/>
<br/>        男人见童筱颖又在跟别的男人相亲，于是乎，开始自娱自乐地调侃：“童筱颖小妹，你就这么急着把自己嫁出去吗？”
<br/>
<br/>        “没……没有……”童筱颖干笑，随即又改口说了，“是”
<br/>
<br/>        她是急着把自己嫁出去……那仅仅只是为了……
<br/>
<br/>        童筱颖刚想解释。
<br/>
<br/>        谁知，这个男人看向傅昼景，将傅昼景上下打量了一番后，又接着说：“这位老弟，我见你一表人才，我才把实情告诉你。你知道她和多少个男人相过亲吗？没有上千也有上百号人了。我听说，这个女人的父亲生意投资失败，导致家企破产，负债累累，还欠了黑社会的高利贷。她仅仅只是为了找有钱人嫁了，顺便要些礼金来替自己还债而已。而且这个女人刚跟你说的都是谎话，她压根就没车没房没工作没存款！”
<br/>
<br/>        把自己的快乐建立在别人的痛苦之上，就这么享受吗？
<br/>
<br/>        童筱颖很想给这男人两巴掌，但是她只能在心底宣泄。
<br/>
<br/>        男人中途喝了口水，接着眉飞色舞地说，毫不顾及童筱颖的颜面，开玩笑似地继续揭着童筱颖的疮疤：“上个星期，她跟我见面的时候，她就这么坦白跟我说的。今天，她估计是见你长得帅又像是很有钱的样子，所以才说了反话想骗你跟她结婚。你可千万别上了这个女人的当啊！”
<br/>
<br/>        童筱颖听着，羞愧地低下头，她面颊发烫烧至耳根，此时此刻，恨不得挖个地洞钻自己进去。
<br/>
<br/>        “是真的吗？”傅昼景面无表情，看着低着头不吭声的童筱颖，只是轻轻地问。
<br/>
<br/>        童筱颖不敢直视傅昼景的目光，于是将头埋得更低。
<br/>
<br/>        她很不想点头，但又不得不诚实地点点头。
<br/>
<br/>        此时，周遭的人都朝童筱颖看过来，大家开始窃窃私语地对她指指点点。
<br/>
<br/>        他们议论的话语虽小，但童筱颖心知肚明，他们在议论她什么。
<br/>
<br/>        说她不要脸，说她见钱眼开，说她不自量力，说她虚伪……那些话肯定没有一句好话。事到如今，她只能临阵脱逃了，不是吗？
<br/>
<br/>        “对……对不起！”
<br/>
<br/>        童筱颖抓起包包起身欲要离开时，却被傅昼景一手抓住了手腕。
<br/>
<br/>        他的手温暖而有力。
<br/>
<br/>        童筱颖停住脚步，包包在手心握紧，眉心微蹙。
<br/>
<br/>        “你还没给我答复。”
<br/>
<br/>        耳畔突然响起傅昼景那透着磁性的声音。
<br/>
<br/>        童筱颖惊怔地抬起头来，一脸茫然：“什么答复？”
<br/>
<br/>        “我已经向你求婚了，可你却没给我答复。”傅昼景语重心长地说。
<br/>
<br/>        “啊？”童筱颖一脸错愕。
<br/>
<br/>        傅昼景会心一笑，慢条斯理地说：“雅座上有一百零一朵粉玫瑰，再加上我刚送到你手里的这七朵，一共是一百零八朵粉玫瑰，代表求婚。”
<br/>
<br/>        “求、求婚……”童筱颖木讷了，完全一副不知所云的样子。
<br/>
<br/>        只见傅昼景语气凝重地接着说道：“我愿意。”
<br/>
<br/>        他愿意？他愿意什么？童筱颖一脸错愕地看向傅昼景。
<br/>
<br/>        转眼间，傅昼景薄唇微启，噙着笑意，接着说：“你不是问我愿不愿意娶你吗？我说我愿意。现在到你了，我都跟你求婚了，你愿不愿意嫁给我？”
<br/>
<br/>        他的话不经令童筱颖目瞪口呆，更是令旁边的那个男人瞠目结舌。
<br/>
<br/>        不等童筱颖回答，男人大言不惭地说：“我说这位老弟，这个女人一看就人品不行……”
<br/>
<br/>        “她敢对你坦白，证明她对你的人并不上心。相反，她有意对我隐瞒，证明她很中意我。而你，当着这么多人的面，说我未婚妻的不是，你应该先怀疑一下你自己的人品！”
<br/>
<br/>        傅昼景将童筱颖往自己身边一拉，自然而然地将她揽入了怀中。
<br/>
<br/>        顿时，男人的脸色半紫半红。他气得对着傅昼景鼓腮帮瞪眼，却比不过傅昼景那强大的气场而只得闷哼了声，然后悻悻地回到自己的雅座里。
<br/>
<br/>        童筱颖受宠若惊般仰头看着傅昼景，45°仰望他的脸，原来现实生活中真有如此帅到360°无死角的美男。而从他怀里传过来的体温，瞬间暖到了她的心窝里。
