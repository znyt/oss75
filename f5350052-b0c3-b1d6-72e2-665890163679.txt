　　萧杰，武魂为火烧藤，橙阶植物系武魂，凡境四重修为。
<br/>　　萧石，武魂为火狼，橙阶兽武魂。虽只有凡境三重修为，但火狼乃是实力不错的妖兽，故他的战力毫不逊色于凡境四重的萧杰。
<br/>　　两人都是武者，但依依只是个普通人，连凡境一重都比不上。
<br/>　　她那瘦弱的身躯，在萧杰萧石两人面前太过微不足道了。
<br/>　　她那胡乱挥舞的棍子更是显得有些可笑。
<br/>　　‘啪’的一声，萧杰轻松握住了棍子。
<br/>　　“丑八怪，识相地就给我滚开。”
<br/>　　“妈的，一个卑微的侍女罢了，每次都组三阻四，小心把你也一并揍了。”
<br/>　　依依还是紧紧地握着棍子，咬牙道，“依依的命是萧逸少爷的，我不会让你们伤害少爷。”
<br/>　　“哼，不知死活。”萧杰一把夺过棍子，并重重地朝依依的手臂砸去。
<br/>　　这要是砸中了，依依的手臂骨折都有可能。
<br/>　　以往，每次萧逸被欺负，依依都是这样挡在他面前。
<br/>　　而且，她每次的阻挡都徒劳无功，最终都是窝囊的萧逸被抢走修炼资源。而后，萧逸会将气撒到她身上，打骂一顿。
<br/>　　“依依，你记住了，你只是个卑微的侍女，本少爷的事不用你管。”
<br/>　　萧逸每次都是气急败坏地打骂依依。
<br/>　　依依已经惯了，但从来没有哪怕一次埋怨过萧逸。
<br/>　　这次也一样，看着即将砸来的棍子，恐惧地闭上了眼睛，身体却没有挪动半分。
<br/>　　‘啪’的一声。
<br/>　　依依以为棍子已经打在了自己肩膀上，可她却疑惑于自己没有感觉到疼痛。
<br/>　　当她奇怪地睁开眼睛时，清澈的双眸瞪得大大的。
<br/>　　因为，此刻萧逸竟然破天荒地站在了她面前，并用手掌接住了棍子。
<br/>　　“少...少爷....”依依感到不可思议。
<br/>　　以往的萧逸是个废物；但此刻的萧逸，乃是地球一代形意拳宗师，更是横行天下的杀手之王，岂会让依依受伤。
<br/>　　萧逸没有回答依依，冷冷地盯着萧杰和萧石两人，“我的侍女，何时轮到你们去打！”
<br/>　　萧杰萧石愣了愣，印象中，萧逸胆敢反抗还是第一次。
<br/>　　“哟呵，小废物果然长脾气了。”
<br/>　　“跟他说废话做什么，不听话就打到他听话。”
<br/>　　萧杰萧石阴笑一声。
<br/>　　“小废物，我看你是忘了被火烧藤教训的滋味了。”萧杰冷笑一声。
<br/>　　萧杰手中猛然出现一条红色的藤蔓，足有两米长，藤上蔓延着丝丝火焰，噼里啪啦。
<br/>　　“火烧藤，去。”
<br/>　　萧杰一甩手臂，火烧藤如有灵性般朝萧逸袭去。
<br/>　　萧逸丝毫不惧，论修为，他只是凡境一重，远远比不上萧杰，但论战斗经验，一百个萧杰都比不上他。
<br/>　　萧逸猛地踏前一步，身子诡异地一扭，躲过了火烧藤，而后双手成爪，直取萧杰脑袋而去。
<br/>　　“躲过了，这怎么可能？”萧杰吃了一惊。
<br/>　　“虎形。”萧逸冷喝一声。
<br/>　　萧杰刚想收回火烧藤防御，却忽然听到一道虎啸之声。
<br/>　　‘吼’，萧逸双爪上竟发出了一声如百兽之王震啸山林般地虎吼。
<br/>　　萧杰打了个冷颤，这一瞬间，他感觉自己正被一只凶猛地野兽盯上了一般，随时会殒命。
<br/>　　下一秒，萧杰猛地被萧逸扑倒。
<br/>　　看起来，萧杰像是被一只猛虎扑倒了一般。
<br/>　　“怎么回事，是我眼花了吗？”一旁的萧石皱了皱眉头，“萧杰乃是凡境四重武者，怎么可能会被小废物扑倒。”
<br/>　　虎形，如猛虎下山，势大力沉，是形意拳中最具攻击力的一式。
<br/>　　此时，萧逸双爪扣着萧杰的脖子，而且恰好是咽喉的部位。
<br/>　　只要他愿意，只需轻轻动动手指，就能毁了萧杰的脖子，要了萧杰的命。
<br/>　　别忘了，萧逸除了是一代形意拳宗师外，更是个强大的杀手，精通各种杀人技艺。
<br/>　　“萧石，愣着干嘛，快救我。”萧杰被吓倒了，没有想自己为什么不是萧逸的对手，立刻向萧石求救。
<br/>　　“来了。”萧石立马反应过来。
<br/>　　“火狼，现。”萧石大喝一声。
<br/>　　他的身上，猛地出现了一头火狼虚影。
<br/>　　火狼，橙阶武魂，能赋予武者极快的速度和火焰的破坏力。
<br/>　　“小废物，去死吧。”
<br/>　　观萧石的动作，竟是真地打算杀了萧逸。
<br/>　　同为萧家族人，竟下手狠辣，想取萧逸性命。
<br/>　　“哼。”萧逸脸色一冷，放开了扣住萧杰脖子的双手。
<br/>　　“豹形。”萧逸的动作很快，如一头觅食地捷豹般，要么不出手，一出手便是将猎物弑于爪下。
<br/>　　“好快。”萧石瞪大了眼睛，他才刚准备发动攻击。
<br/>　　萧逸的双爪已经来到他胸膛前，狠狠一爪，将他的胸膛划出一道深深的血痕。
<br/>　　豹形，如捷豹突袭，速如疾风，快若闪电，乃是形意拳中最适合短距离近身爆发速度的一式。
<br/>　　形意拳，乃是一门内外兼修的武学。
<br/>　　分五行十二式，劈拳、钻拳、崩拳、炮拳、横拳、龙形、虎形、鹰形、蛇形....
<br/>　　彼此间可互相融合或者拆分，交横错纵，复杂无比，可衍生出千百种打法，堪称博大精深。
<br/>　　而萧逸，作为形意拳宗师，不仅完全悟透了所有打法，更将其精炼成五种拳法。
<br/>　　这从千百种打法中衍生出来的五种，被萧逸誉为形意五绝。
<br/>　　分别为虎形、豹形、蛇形、鹤形，以及最强的升龙。
<br/>　　凭此五绝，他才做到了纵横地球，罕逢敌手。
<br/>　　虽只有凡境一重的微末实力，但凭借形意五绝，还是轻松地将萧杰与萧石打败。
<br/>　　“你想死吗？”萧逸双爪一撕，将萧石胸膛上的伤口划得更大。
<br/>　　顿时，汩汩鲜血从萧石身上流落。
<br/>　　“阿...阿...”萧石惊呼着，看着胸膛上不断流出了鲜血，感到身体虚弱外，还有一种对死亡的恐惧。
<br/>　　“我...我会死掉吗？阿...阿....”
<br/>　　“闭嘴，别叫得跟jiao春似地。”萧逸瞪了萧石一眼。
<br/>　　另一边，萧杰看着凄惨地萧石，也浑身哆嗦。
<br/>　　说到底，这两人只不过是温室中的花朵，哪里见过这等血腥地场面。
<br/>　　一旁地依依也是脸色煞白，不过更多地却是在关心着萧逸。
<br/>　　“哼。”萧逸收回了双爪，一拳将萧石轰到地面的萧杰身旁。
<br/>　　“别...别杀我们。”萧杰早就吓得腿软，连站起来的力气都没有。
<br/>　　“小....小废物，不，萧逸，我们是同一个家族的人阿。”萧石捂着伤口，求饶道。
<br/>　　他们明显从萧逸眼中看到了浓郁的杀意。
<br/>　　这一刻，他们能感觉到，面前的萧逸完全不像一个废物，反倒像一个手中沾满了鲜血的刽子手，让他们心里发怵。
<br/>　　“别杀你们？”萧逸冷笑一声，道，“这些年来，你们抢我的修炼资源，现在还想我饶过你们？”
<br/>　　“不..不是阿。”萧杰哆嗦着反驳道，“萧逸，算起来，我们可是你的表哥阿，怎么会抢你的东西呢。以前我们是急着修炼，向你借一些罢了。”
<br/>　　“对..对对，借的，不是抢。”萧石也立刻说道。
<br/>　　“好，借的。”萧逸点点头，道，“那么这些年借的，也该还回来了吧。”
<br/>　　“还，我们一定还，只要你放过我们。”萧杰和萧石如蒙大赦，立刻说道。
<br/>　　“每人一万两，立刻还钱。”萧逸冷笑一声。
<br/>　　“一万两？”两人瞪大了眼睛。
<br/>　　“怎..怎么这么多。”萧杰吓了一跳。
<br/>　　“对阿，每个月的银两只有20两，丹药大概也就值30两左右，一个月顶天了也就50两。”
<br/>　　“一年600两....”
<br/>　　萧逸狠狠地瞪了他们一眼，打断道，“既然你们是借的，那不得收利息？利滚利你们没听过？你们从我8岁开始抢我的东西，到现在足足8年了。”
<br/>　　“才每人一万两，我算是便宜你们了。”萧逸冷笑道。
<br/>　　“额..这...”萧石和萧杰暗暗大骂，明明在诓我们，怎么感觉像我们占了好处一样。
<br/>　　“可是，我们拿不出一万两阿。”萧杰和萧石心中打着如意算盘，心想自己拿不出那么多钱，萧逸也拿他们没办法。
<br/>　　“拿不出钱，那就去死吧。”萧逸双掌呈爪，面露无比杀意，直接朝他们要害袭去。
<br/>　　“不...不要...”
<br/>　　萧杰萧石看着萧逸那嗜血的目光，知道萧逸没有开玩笑，立刻求饶。
<br/>　　“先欠着行吗？”
<br/>　　“可以。”萧逸收回了手掌，笑道，“依依，取纸和笔来，免得这两个家伙赖账。”
<br/>　　“是，萧逸少爷。”依依点了点头，立马准备了纸和笔。
<br/>　　萧杰和萧石不甘地写下欠账单。
<br/>　　萧逸拿起欠账单，看着两人打得手指摸，满意地点了点头。
<br/>　　“滚吧。”萧逸道，“我提醒你们，账单已经写下了，欠债还钱，天公地道，你就是找家族长老也没用。”
<br/>　　“记好了，尽快筹够钱。我不管你们偷也好，抢也罢，筹不够钱，就等着我要你们的命吧。”萧逸冷冷地交待了一句，让两人离开。
<br/>　　其实，从一开始萧逸就没打算杀他们，只不过是为了让他们还钱，自己能得到更多地好处罢了。
<br/>　　从记忆中得知，武者的修炼，可是一项极其耗费资源的事。钱越多，能得多的资源越丰富，修炼起来才更轻松，也更快。
<br/>　　.....
<br/>　　萧杰萧石离开后，在萧逸的院子外绕了个圈，最终在一名少年面前停了下来。
<br/>　　少年长得丰神俊朗，但眉宇间透漏着一股阴鸷地冰冷。
<br/>　　“若寒表哥，我们失败了，那小废物不知怎么回事变得这么厉害了....”
<br/>　　萧杰萧石大概说了些事，却没有将自己写下欠账单的事说出来。
<br/>　　....
<br/>　　没有人知道，在角落中，萧逸正冷冷地盯着这三人。
<br/>　　以他的藏匿本事，除非有人走到他面前，否则，绝对发现不了他。
<br/>　　他早就猜到，今天萧杰和萧石的到来有些不正常，所以在他们两人走后，他偷偷跟了上来。
<br/>　　“果然，又是这家伙搞地鬼。”萧逸心中冷笑。
<br/>　　“最好别来惹我，否则我不介意提前送你下地狱。”萧逸呢喃一声，而后离开了角落。
<br/>　　自始至终，萧若寒三人都没发现他的存在。
