　　比武台上，蓦地陷入了短时间的寂静。
<br/>　　哪怕战斗已结束，胜负已分，两方天骄都已下台。
<br/>　　可剑宫裁判，未有半分话语，更未宣布下一轮天骄。
<br/>　　直至比武台上那句回响彻底落下。
<br/>　　剑宫裁判才咳嗽两声，清了清喉咙，高声宣布下一轮比拼者。
<br/>　　不过，直到新的一轮战斗打响。
<br/>　　可在场几乎所有观赛者，所有参赛天骄的目光，此时都定格在观赛席上那道淡漠的身影之上。
<br/>　　这个年轻妖孽，虽然有着恐怖的战绩，惊人的传闻。
<br/>　　可传闻归传闻。
<br/>　　待真正看到他出手，还是让所有人对他的实力感到一阵可怕。
<br/>　　这真的只是个20来岁不足30岁的年轻人？这真的只是个扬名不过数年的妖孽？
<br/>　　这等实力，怕是足以让天下所有传奇以下的武者感到汗颜，乃至惊骇。
<br/>　　不多时，众人的目光，逐渐收回。
<br/>　　这一战，看似耗时不短，实则，也就十来分钟的事。
<br/>　　但这一战，怕是足以让在场所有人日后津津乐道许久。
<br/>　　萧逸闭目假寐。
<br/>　　一个时辰后。
<br/>　　又是数轮比拼结束，随着裁判的高声宣布，又是两个天骄跃上比武台。
<br/>　　这时，萧逸再次睁开了眼睛。
<br/>　　因为，上台的二人，皆是熟人。
<br/>　　其中一人，一袭白衣，风度翩翩，手握长剑，显然是个剑修强者。
<br/>　　另一人，一身火红衣服，手持一长鞭。
<br/>　　战斗，瞬间打响，又在极短的时间内便结束。
<br/>　　啪…
<br/>　　一声长鞭炸响声，比武台上，一阵劈啪作响。
<br/>　　白衣武者，手中长剑，瞬间被长鞭缠绕，随后脱手而出。
<br/>　　啪…
<br/>　　又是一声长鞭炸响。
<br/>　　白衣武者胸膛一阵疼痛，直接被轰飞十数米。
<br/>　　“岳天，你太弱了。”红衣女子手持长鞭，戏谑一笑。
<br/>　　“火舞姑娘，是你太强了。”岳天苦笑几声。
<br/>　　不错，上台的二人，正是公孙火舞，以及岳天。
<br/>　　观赛席上。
<br/>　　顾怜星掩嘴一笑，“火舞这丫头，这数天来才第一次上台吧，这么快就得意洋洋了。”
<br/>　　叶流亦笑笑，道，“火舞这丫头，原本在十八府天骄排名中还算不错。”
<br/>　　“可前些天浪费了大量修炼时间，上两个月才开始第二次觉醒武魂，现在倒是落了其他人不少实力。”
<br/>　　“现今，只能算是十八府天骄中的末流。”
<br/>　　“可她的实力，又比其它的一流势力天骄要强不少。”
<br/>　　“故她的层次，不上不下，剑宫裁判抽签，也比较麻烦，自然的，她现在才第一战。”
<br/>　　顾怜星点点头，“这岳天，其实实力不错，可惜，他对上了火舞。”
<br/>　　“除却十八府天骄，这岳天，放到一流势力的天骄中，亦可算排在前列了。”
<br/>　　萧逸听着二人交谈，也看了眼比武台上的二人。
<br/>　　岳天的实力，对比上一次在剑帝碑盛事时，长进了不少，但也不至于飙升太多，总的来说，还算不错。
<br/>　　而公孙火舞，此刻的修为，竟只在接近圣皇境巅峰的层次。
<br/>　　对比其余十八府天骄那已然接近武道大能的层次，她显然弱了不少。
<br/>　　比武台上的胜负，不必多说，公孙火舞再怎么荒废了一段修炼时间，也终究是十八府天骄，不是岳天能比的。
<br/>　　不过，比武台上接下来的事情，倒是让萧逸一阵无语。
<br/>　　比武台上。
<br/>　　岳天已然落败，倒在台上。
<br/>　　公孙火舞，居高临下，叉着腰，傲然道，“看在你家域主的份上，我便不揍你了，稍后自己下台。”
<br/>　　“谢公孙姑娘。”岳天苦笑一声。
<br/>　　他虽是四方域有名的剑道天骄，但对比十八府天骄，差距还是不小的，本就没想过能赢。
<br/>　　“别急着谢。”公孙火舞一甩长鞭。
<br/>　　长鞭，在岳天脑袋边一挥而过，直吓了岳天一跳。
<br/>　　“你老实告诉我，你可知你家域主下落？”
<br/>　　“不知。”岳天摇了摇头。
<br/>　　啪…又是一声长鞭呼啸，在岳天耳边闪过。
<br/>　　岳天看了眼距离自己脑袋不足半米的地面，地面上，一条深刻的鞭痕，不禁咽了口口水。
<br/>　　“知不知道？”公孙火舞，再度问了一声。
<br/>　　啪…又是一声长鞭呼啸。
<br/>　　岳天脑袋附近的地面，已然开裂。
<br/>　　岳天看了眼，不禁打了个激灵。
<br/>　　“火舞姑娘，我真的不知道啊。”
<br/>　　“真的？”公孙火舞眯了眯眼。
<br/>　　啪…啪…
<br/>　　一阵鞭影闪烁，岳天周遭鞭影重重，地面上那霹雳作响的声音，直让岳天心惊。
<br/>　　这随便一鞭若是打歪了，他岳天怕是毁容还是轻的，脑袋都能被砸个稀巴烂。
<br/>　　“火舞姑娘，我真不知道，域主他向来行踪漂浮，我…”岳天，哭丧着脸。
<br/>　　“算了。”公孙火舞一甩收鞭，“谅你也不敢骗本姑娘，下去吧。”
<br/>　　“谢公孙姑娘。”岳天连忙起身，道谢一身，随后一个闪身，跃下比武台。
<br/>　　观赛席上。
<br/>　　叶流摇了摇头，“火舞这疯丫头，刚才怕是把岳天吓得出了一声冷汗。”
<br/>　　嗖…
<br/>　　公孙火舞的身影，这时闪烁而来。
<br/>　　“叶流，你说本姑娘坏话？”公孙火舞，大大咧咧地在顾怜星身旁坐下。
<br/>　　叶流撇撇嘴，不语。
<br/>　　聪明如他，自是知道唯女子难养也这个道理。
<br/>　　公孙火舞，亦未理会叶流，而是挽着顾怜星，聊个不停。
<br/>　　“怜星，我这几天可想你了。”
<br/>　　“府里的老家伙，都不让我离开席位。”
<br/>　　顾怜星掩嘴一笑，“还不是怕你胡闹？”
<br/>　　“一个疯丫头。”一旁，冉琦摇了摇头。
<br/>　　“与你何干？”公孙火舞，瞪了冉琦一眼，目光，又看了眼萧逸。
<br/>　　“又是你这个家伙？”
<br/>　　公孙火舞看着萧逸，眼中霎时怒火中烧。
<br/>　　“上次，便是你这个家伙骗我，说那小贼回了风刹殿总殿。”
<br/>　　“我爷爷特意去风刹殿总殿问过，那小贼，根本没回总殿，连总殿的人也寻不着他。”
<br/>　　萧逸心头暗暗无奈，只得语气冷漠道，“你不信易某，易某也无话可说。”
<br/>　　低沉的声音中，仿佛充斥着怒气。
<br/>　　“你…”公孙火舞眼睛一瞪。
<br/>　　“火舞，算了。”叶流连忙道，“易兄不喜言谈，你无谓胡闹。”
<br/>　　“哼。”公孙火舞愤愤坐下。
<br/>　　“咯咯。”一旁，顾怜星再次掩嘴一笑，“火舞，我与你一同长大，还从未见你对别人如此上心。”
<br/>　　“以前我就与你说过，萧逸公子，不曾偷你半分东西，你别总喊他小贼。”
<br/>　　“没想到，你是心被偷了去。”
<br/>　　“哼。”公孙火舞轻哼一声，脸上闪过一阵绯红，扭捏了一下，却不语。
