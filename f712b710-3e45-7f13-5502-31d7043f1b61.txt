        &lt;h3&gt;第三卷第二百一十九章  三个  戏精&lt;/h3&gt;
<br/>
<br/>        陈子昂到西城见了江俞轩，看着江俞轩在车上坐了一晚，她也一直站在窗帘后面看着隐藏在暗夜中的车辆，天下没有不散的宴席，人总是都要各奔东西。
<br/>
<br/>        给江俞轩的那块腕表她买了很久，大概是在认识陆宇霆之前吧，当时还没有那么有钱，那是一对情侣表，她很喜欢，买的时候心里自然而然想到了江俞轩，准备给江俞轩当生日礼物的，只是一直没有送出去，看着自己的这一块，恐怕这一生都不会戴了吧，原本想着都送给江俞轩，他们夫妇俩个人刚好一对，可是她怎么都舍不得将自己的这一块送出去，她戴了一会，轻轻的摩擦着腕表，然后卸下来，锁进了保险柜，就像是将她的心锁起来了一样。
<br/>
<br/>        出门的时候并没有去看江俞轩是不是走了，而是直接打车去了医院看了三姐，然后和潘禺一起回北京。
<br/>
<br/>        潘禺在部队，回来的时间也不是很多，假期没有多长时间，听说陈子昂有孩子了也想去见见孩子，就联系了陈子昂和陈子昂一起到北京。
<br/>
<br/>        陈子昂来去匆匆已经是习惯了，马上要上班，她也没有时间磨磨唧唧，和潘禺一起回到北京，在路上，潘禺有很多话想问陈子昂，但是人多，俩个人就胡天海地的胡谝。
<br/>
<br/>        “小姨，你这学霸的气质一直不改，我是怎么都追不上你的脚步，你让我情何以堪啊。”
<br/>
<br/>        “追你女朋友去，追我干什么？”陈子昂笑着说。
<br/>
<br/>        “女朋友不用追的，都是她追我，我每次都要等她。”
<br/>
<br/>        “吹吧你，就你这样的还有人要？”
<br/>
<br/>        “陈子昂，我怎么了我，我不帅吗？我身高一米八五，体重65公斤，剑眉星目，玉树临风，连续两年的散打冠军，你竟然说没有人要我。”潘禺不高兴了。
<br/>
<br/>        “看看，本性露出来了吧？陈子昂，陈子昂是你叫的？”
<br/>
<br/>        “不要拿你的辈分来压迫我，不就是我妈的妹妹嘛？有什么了不起？”潘禺表示不服气，不就是比他大两三岁吗，从小到大都是被陈子昂吃的死死的。
<br/>
<br/>        “我就了不起了，怎么的？你咬我啊，小时候你拿我没办法，现在你也没办法。”陈子昂最喜欢看潘禺吃瘪的样子。
<br/>
<br/>        俩个人真的和小孩子一样斗起了嘴，斗嘴很欢乐，欢乐的让人忘记了还在旅途，忘记了他们已经是成年人了。
<br/>
<br/>        “陈子昂，你就这智商和情商我都不知道你是怎么活到现在的，哎，亏我这颗小心脏已经为你担惊受怕了这么久。”
<br/>
<br/>        “我就是活到现在了，而且活的很好。”陈子昂笑嘻嘻。
<br/>
<br/>        “很累吧？”潘禺突然很认真的看着陈子昂。
<br/>
<br/>        “还好！”陈子昂转过头看着窗外，一望无际的浩瀚，人是多么的渺小啊。
<br/>
<br/>        “晚上咱俩喝几杯，我听小舅说你酒量很大，这几年练出来了啊。”
<br/>
<br/>        “行啊，你想喝多少我陪你。”
<br/>
<br/>        “唉，想着我要和你家的小屁孩称兄道弟我这心里就难过。”潘禺有一搭没一搭的说着话，他心里有很多问题想要问陈子昂，但是只能等。
<br/>
<br/>        “没办法，辈分在哪里。”只要一提辈分潘禺就抓狂，可是又无可奈何。
<br/>
<br/>        “没意思，以后还是少见为妙。”
<br/>
<br/>        ........
<br/>
<br/>        陈子昂带着潘禺回家，惊坏了陈虞和落妍。
<br/>
<br/>        落妍：“妈妈，你这么快就带个男朋友回来？”
<br/>
<br/>        陈虞：“你带男朋友回来也需要给我们报备一下，打声招呼吧？”
<br/>
<br/>        潘禺哈哈大笑：“男朋友，你想多了，我是来和你做兄弟的。”
<br/>
<br/>        陈虞：“做兄弟可以，只要不和我们抢妈妈，你就是弟。”
<br/>
<br/>        潘禺：“妈妈我也是要抢的，兄弟也是要做的，你说怎么办呢？”
<br/>
<br/>        落妍：“来来，抢我，抢我，我比妈妈好看，还年轻！你这个大哥哥颜值不错，粉你了。”
<br/>
<br/>        陈子昂简直是哭笑不得，这三个活宝在一起不会翻天？
<br/>
<br/>        潘禺：“你们我都要抢，你们都是我的。”
<br/>
<br/>        陈虞和落妍看了一眼，蔫了，落妍：“哥哥，我们的苦日子来了。他铁定是妈妈的新男友。”
<br/>
<br/>        潘禺：“新男友？难道你妈妈还有旧男友？”
<br/>
<br/>        陈虞：“可不，刚刚出门，多财多艺，人又帅。”
<br/>
<br/>        落妍：“关键是对我们很好，天天让我们当马骑，话说，你能让我们当马骑吗？”
<br/>
<br/>        陈子昂，有这么坑娘的吗？
<br/>
<br/>        “陈子昂，你忒没良心了，亏我真心对你，你竟然还有男朋友。”潘禺伤心欲绝。
<br/>
<br/>        “你们三个戏精，演完了没有？演完了就滚到沙发上去，别站门口了。”陈子昂已经是无语了。
<br/>
<br/>        “财叔，春姨，这个是我三姐的孩子，潘禺，陈虞，落妍，叫哥哥。”陈子昂看着三个人斗嘴停不下来的样子，就打断了他们。
<br/>
<br/>        “哈哈哈，还真是哥哥啊，哥哥，我叫陈虞，这个颜控就是落妍，难怪长得不错，原来和我们有着同样的基因。”
<br/>
<br/>        “陈子昂，你这俩个孩子是你偷来的吧？怎么这么可爱？”潘禺开心的大笑，这么多年没有见，其实陈子昂一点也没有变，可能她生活的比较艰难，但是她的心态很好，要不然教育不出来这么可爱的孩子。
<br/>
<br/>        “潘禺，再叫我陈子昂，我可是真的生气了啊。”陈子昂看着潘禺那嘚瑟的样子就想踹他几脚，拧他的耳朵。
<br/>
<br/>        陈虞：“潘禺，你怎么能这样称呼你的长辈呢？”
<br/>
<br/>        “为什么不能，我们两个年龄差不多大，你看她哪里像我小姨的样子？你叫我什么？”
<br/>
<br/>        落妍：“潘禺啊！”
<br/>
<br/>        潘禺：“你们不是应该叫我哥哥嘛，好吧，小姨，我可以和弟弟妹妹一起玩了吗？”
<br/>
<br/>        陈子昂：“好吧，饭好了叫你们。”
<br/>
<br/>        潘禺被陈虞和落妍拉着手走到客厅坐下。
<br/>
<br/>        陈虞：“潘禺，特种部门的，今年的散打冠军，潘青松和陈子澜的儿子，比陈子昂小了三岁，小时候天天跟在陈子昂后面打酱油。”
<br/>
<br/>        落妍：“你一进门我们就知道了，不过是逗你乐一乐而已。”
<br/>
<br/>        潘禺：“你们怎么知道？小姨告诉你们了吗？”
<br/>
<br/>        陈虞：“妈妈事情太多，不会告诉我们很多事情，我们自己调查的，潘禺哥哥，我想我们遇到困难了。”
<br/>
<br/>        潘禺：“你们遇到什么困难了？”
<br/>
<br/>        落妍：“我们不想离开妈妈，所以你能保护我们吗？”
<br/>
<br/>        潘禺：“怎么回事？”
<br/>
<br/>        陈虞和落妍蹲着潘禺的身边，将张函新年到他们家的情况告诉了潘禺，也将陈子昂的决定告诉了潘禺：“其实，我们真的不想离开妈妈，和妈妈在一起就算是有危险，我们也愿意和她一起承担。”
<br/>
<br/>        “陈虞，落妍，妈妈送你们出去是对的，我会安排人保护你们的，你们在外面安安全全的，妈妈才能做更多的事情。大人的世界你们不懂，等你们长大了就知道了，你们现在的任务是平安而坚强的长大。”
<br/>
<br/>        潘禺有些心疼陈子昂的遭遇，他知道她有孩子之后会想到陈子昂很辛苦，没有想到陈子昂比他想象的更辛苦。
