        见童筱颖答不上话来，傅昼景苦笑：“筱颍，你真的变了。”
<br/>
<br/>        “我跟你很熟吗？我有没有变，关你什么事？”童筱颖一脸不悦地反驳。
<br/>
<br/>        傅昼景无所谓地笑了笑：“对，你跟我已经不熟了。你变心了，但我对你的心始终没变。”
<br/>
<br/>        “我变心？”童筱颖难以置信地看着傅昼景。
<br/>
<br/>        傅昼景似乎是打算点到为止，就此结语道：“我希望你能不忘初心，找回自己。我还要去谈合作，就跟你聊到这里。有机会再见。”
<br/>
<br/>        傅昼景说完，便起身离开了。
<br/>
<br/>        对于傅昼景的话，童筱颖突然间放心上了。放心上的原因，大概是因为大哥的缘故。
<br/>
<br/>        “变心”这两个字眼，让童筱颖感到有些扎心。
<br/>
<br/>        她独自一个人在这里坐了一会儿，点了一桌子的点心也没吃上几口，便结了账离开了“夜惑”。
<br/>
<br/>        不知不觉中，童筱颖游走在街边的时候，路过一家私人侦探事务所，或许是一时脑热，她便走了进去。
<br/>
<br/>        招待她的是一个年轻的男子，一看到她，他便关切地询问：“是不是想查你老公出轨的对象？”
<br/>
<br/>        被戳中心事，童筱颖毫不忌讳地点了点头。
<br/>
<br/>        大哥不肯说，那么她就请人查。
<br/>
<br/>        她坚信梦里的事情，一定是真的。
<br/>
<br/>        童筱颖付了定金，留下了自己的电子邮件。
<br/>
<br/>        私人侦探的老板允诺，一个星期后，就会把结果发到她的电子邮件里。
<br/>
<br/>        这天的事情过后，傅夜沉回来时也未再提及此事，童筱颖也没有再去追问，只是两人相处在一起时，话题要少了许多。
<br/>
<br/>        而这一个星期对童筱颖来说，是一种煎熬。
<br/>
<br/>        一个星期后，傅夜沉一如既往的外出对傅氏集团旗下各个子公司例行检查。
<br/>
<br/>        童筱颖一个人待在家里，拿着平板电脑点开了电子邮件，里面只有一张苏漫雪的相片，下面还有一段口述的文字说，苏漫雪曾经是傅夜沉的未婚妻，傅夜沉还捧苏漫雪入娱乐圈成为一线明星，并且苏漫雪曾在医院里做过人流手术。
<br/>
<br/>        后面，还附上了苏漫雪做手术的单子，家属签字人是“傅夜沉”，时间是半年前。
<br/>
<br/>        看到这里，童筱颖懵了。
<br/>
<br/>        她一个人坐在客厅的沙发上发了许久的呆，才想起要带着平板电脑去书房核对傅夜沉的字迹。
<br/>
<br/>        童筱颖将书房里翻了个遍，终于找到一本有傅夜沉签名的书，再跟平板电脑里苏漫雪的那张手术单子上的签名字迹核对。
<br/>
<br/>        下一个瞬间，书和平板电脑同时从手中掉落，童筱颖有些崩溃地跪坐在了书桌旁，捂着脸抽泣了起来。
<br/>
<br/>        这天晚上，傅夜沉比平日里要回得晚了一些。
<br/>
<br/>        此时，已经晚上九点了。
<br/>
<br/>        他一边松了松领带，一边往楼上主卧走去。
<br/>
<br/>        傅夜沉进屋后，见童筱颖躺床上睡着了，便去隔壁卧室里洗了个澡，他的动作很轻，生怕将她吵醒。
<br/>
<br/>        但是，洗完澡回到床上后，看到这个小女人时，心头又莫名其妙地感觉很热，脑子里有了想吻她的冲动。
<br/>
<br/>        童筱颖此刻侧卧在他身边，双手抱着被子握成了拳，模样娇美又有些可爱。
<br/>
<br/>        傅夜沉看得有些按捺不住，喉珠上下滚动着，望眼欲穿。
<br/>
<br/>        他情不自禁地俯下身，慢慢地靠近，覆上这小女人的粉唇后。
<br/>
<br/>        童筱颖从始至终并未睡着，枕头上的泪水也早已干透，但她仍旧闭着眼睛，默不吭声，同时不反抗也不配合。
<br/>
<br/>        慢慢的，房间里的气温好像在上升……
<br/>
<br/>        一阵饱满的填充感袭来，童筱颖倏然睁开了双眼，不料与傅夜沉四目相对。
<br/>
<br/>        只见他额头上渗出了细密的汗珠，黑眸里也染上了一丝情欲的氤氲。
<br/>
<br/>        “大哥，放过我。”
<br/>
<br/>        童筱颖突然没有任何前兆地说道，让傅夜沉身心一滞。
<br/>
<br/>        “我不要当苏漫雪的替身！”
<br/>
<br/>        “？！”傅夜沉瞬间一脸懵。
<br/>
<br/>        童筱颖鼻子一酸，忍不住抽泣了起来：“你对我好残忍！真的好残忍！”
<br/>
<br/>        她这莫名其妙地一哭，傅夜沉顿时有些束手无策。
<br/>
<br/>        “发生什么事了？”他皱起了剑眉，声音暗哑地低问。
<br/>
<br/>        童筱颖没有回答他的话，只是自顾自地捂着脸，开始失声痛哭。
<br/>
<br/>        傅夜沉不得不停下来，将童筱颖搂入怀中，抚摸着她的后脑勺安慰道：“乖，告诉我，到底发生了什么事？”
<br/>
<br/>        “你爱的女人是苏漫雪对不对？我只是她的替身，对不对？我以前不长现在这个样子，对不对？”童筱颖一边哭着，一边连连发问。
