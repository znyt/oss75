　　苏重眉头微皱，他确实感觉不到什么邪气。
<br/>　　“戾气吸收的还是太多了。”他只是一想就知道原因。
<br/>　　虽然有道种封印收摄，可在戾气池子里泡的久了，总会有影响。
<br/>　　“你赶紧想办法解决，就你这身邪气，你师傅看都不看就会拿剑劈了你！”六尾肃然道。
<br/>　　苏重也知道情况严重。
<br/>　　“还好我有解决方案，只不过要躲几天了。”
<br/>　　他没在竹屋多呆，反而悄悄进入黑竹林阵法。这阵法是他布置完善，随着苏重阵法造诣提高。黑竹林功效越发强大。
<br/>　　草木精气不断汇聚，甚至凝聚成丝丝雾气在黑竹林内飘荡。林中黑竹受到滋养，长得越发旺盛。
<br/>　　悄然潜入阵法，走进阵法最深处。
<br/>　　黑竹林经过修改，越往内部走，草木精气越浓厚，威力越强劲。
<br/>　　中心处已经被一片白雾笼罩。
<br/>　　“有黑竹林阵法相助，解决戾气问题轻而易举！”
<br/>　　五颗子体道种，是苏重给主道种准备的能量池。可实际上子道种并未彻底完成。
<br/>　　它们只是吸饱了戾气，苏重用的是剑气，还需要用草木精气融合。
<br/>　　从最开始修炼，苏重就在融合草木精气炼化戾气。现在操作起来得心应手，而且还有道种相助，这一步修炼反而只是耗费时间而已。
<br/>　　即便如此顺利，苏重也用了足足三天才把五颗道种内戾气全部炼化。足可见他这次到底吸收了多少戾气。
<br/>　　看了看周围稀薄近乎于无的草木精气，苏重无奈摇头。他也没想到会用这么多草木精气。
<br/>　　“希望不要打扰几位师兄修炼。”苏重再次悄然离开。
<br/>　　他并不担心黑竹林阵法，虽然消耗确实不小，但只要给它几天时间，就能缓缓凝聚恢复。
<br/>　　……
<br/>　　竹屋内，六尾围着苏重转来转去。
<br/>　　“你的到底是怎么练的，怎么一点儿怪异感觉都没有了？”六尾啧啧称奇。
<br/>　　三天前的苏重邪气冲天，如今看去却就是个普通青年。全身上下，甚至没有一点儿修炼过的气息。
<br/>　　“你这个隐藏气息的法门，越来越厉害了！真不能交给我？”六尾已经不是第一次见苏重隐藏自身气息，每次看都羡慕的不要不要的。
<br/>　　“不是不教给你，是教给你也学不会。你会阵法吗，不会阵法，根本练不成！”
<br/>　　“那还是算了吧！”六尾可惜道。黑竹林阵法他都看不明白，更别说更复杂的天机印了。
<br/>　　“对了，你还记得万人往的气味吗！”苏重突然道。
<br/>　　“你要干什么？”六尾狐疑的盯着苏重。
<br/>　　“鬼王宗要阴谋对付青云，他肯定就在青云周边藏着，你帮我把他找出来怎么样？”苏重眼睛眯起。万人往竟然敢派苍松来杀他，他当然要给他点儿颜色看看！
<br/>　　“你还真当我是狗鼻子啊！你怎么不去找大黄！”六尾愤愤不平。
<br/>　　大黄？大黄要真管用，我还用找你！苏重撇撇嘴。
<br/>　　田不易的大黄狗虽然被灵药堆成了精，但本质就是一头普通黄狗。鼻子确实灵敏，但和六尾比起来还差得远。
<br/>　　六尾是妖狐灵种，这种天生嗅觉，本身就带着些血脉神通的意思。灵敏度高，搜索范围大！
<br/>　　“你就不想看鬼王宗倒霉？”苏重暗戳戳诱惑。
<br/>　　他算是看出来了，六尾对鬼王宗不说恨之入骨，但绝对看不得他们好！只要能给鬼王宗找麻烦，六尾绝对会参与。
<br/>　　“我也想找他，没用的。”六尾无奈道：“第一，青云周边太大，我的搜索范围毕竟有限，咱们真要找，也只能是碰运气。”
<br/>　　“第二点，万人往太了解我们狐族了，他身上早就做了防备。一身气息或隐匿或用其他东西混淆，上次咱们在天上，我就仔细分辨过万人往的气味，找不到的！”六尾也无奈。
<br/>　　“找不到万人往，能找到青龙吗？”苏重突然道。
<br/>　　青龙作为万人往心腹，很多时候都跟随身边，找到他，说不定就能找到万人往！
<br/>　　“青龙？这个好像还可以？”六尾迟疑：“但绕着青云漫无目的寻找，就和个没头苍蝇一样，找到的几率也不大啊。”
<br/>　　“这个不用担心，我去找人打听一下就有方向了。”青云已经展开针对魔门的探查。
<br/>　　大师兄宋大仁就是探查人员之一，可以通过他获得相关情报。
<br/>　　果然，苏重找到宋大仁，说想了解鬼王宗动向，避免被鬼王宗偷袭。宋大仁几乎毫不犹豫就把所知情报告诉了苏重。
<br/>　　当天晚上，苏重就带着六尾偷偷下山，直奔河阳城而去。
<br/>　　找了个客栈住下。第二天苏重盖头换面，带上大斗笠遮住面孔，背上一个破旧竹筐，把六尾放在里面就出门绕着河阳城溜达。
<br/>　　六尾躲在竹筐内施展追踪法诀，很快就找到线索。苏重一路跟过去。走进一个小巷子里。
<br/>　　“是这家？”苏重装作寻常人走过小巷，朝中间一户人家瞥了一眼。外表看不出丝毫异样，就是一处普通人家。如果不是六尾分辨，他根本发现不了这里。
<br/>　　“就是这里，除了青龙，还有四五个其他人的气味，应该是他和一小股人躲在这里。”六尾笃定。
<br/>　　“很好！”苏重转身就走，毫不留恋。
<br/>　　又在河阳城内溜达几圈，借助六尾发现了一些可疑人员踪迹。苏重就回到客栈足不出屋。
<br/>　　等到晚上，苏重换上一身夜行衣，打开窗户身形一闪消失不见。一道白影紧随其后，瞬间没入茫茫夜色。
<br/>　　白天闲逛，河阳城地图已经了然于心。苏重在城内快速穿梭。他没用飞遁法诀。现在局势紧张，河阳城内不只有魔门隐藏，青云更是撒出了大把的探子。
<br/>　　遁光一起，必定引起各方异动。
<br/>　　悄无声息来到白天探查过的小巷子，苏重看向身后六尾。
<br/>　　六尾点头，表示气味还在。说明青龙依旧藏在其中。
<br/>　　天机印运转起来，苏重浑身上下没有一丝气息泄露。轻轻跳起，宛若一片落叶飘荡进入院内。
<br/>　　六尾紧跟其后，悄无声息落下。
<br/>　　“起阵！”
<br/>　　一声大喝陡然响起，黑漆漆小院顿时变得灯火通明。
<br/>　　地上蓦然亮起道道光芒，一股束缚之力骤然加身。
<br/>　　不等苏重反应，紧接着一张大网从天而下，把苏重和六尾直接罩了起来。
<br/>　　“怎么才只有一个人！”一个黑衣老者不满道：“万人往，你不是说这次能伏击青云长老吗？”
<br/>　　万人往不理对方，戏谑的看着苏重：“我以为你会很聪明，没行到你竟然会蠢到自己来！”
<br/>　　苏重哪里还不知道，自己这是一头扎进了人家陷阱里。
