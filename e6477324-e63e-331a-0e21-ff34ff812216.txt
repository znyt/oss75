        “咳——咳——”童筱颖被刘婶这句“生个孩子”的话给呛到了。
<br/>
<br/>        大哥可是亲口跟她说，这一辈子都不会碰她，当然，她对大哥也没有那方面的想法。
<br/>
<br/>        她现在是决定不跟他离婚了，但是她在坐等他跟她提离婚。
<br/>
<br/>        童筱颖不知道是不是因为自己的自卑心在作祟，明明已经得知他喜欢的人是她，但她心里总觉得，大哥他这样的优质男值得拥有一个比她更好的女人来当他的妻子。
<br/>
<br/>        所以她才能“纵容”落以柔在她面前肆意嚣张。
<br/>
<br/>        但其实，她心里又是矛盾的。
<br/>
<br/>        另一个声音，又在怂恿她，让她去尝试着爱上这个男人。
<br/>
<br/>        可是，就算她脑子里想爱，但是她心里却又不敢去爱。
<br/>
<br/>        归根究底，她还是怕自己爱了会是受到伤害。
<br/>
<br/>        翌日。
<br/>
<br/>        傅氏集团大厦。
<br/>
<br/>        智能煮茶机里正煮着奶茶，因为是高档红茶和鲜牛奶的混合，所以茶水间里顿时弥漫着红茶的芬芳和牛奶的醇香。
<br/>
<br/>        落以柔打算为傅夜沉泡一杯营养健康的自制奶茶，今天一大清早，她刚到医院他的VIP病房里，他便告知她，他可以出院了，让她给她师父郑忠威打电话命令企划部提前准备好企划案。
<br/>
<br/>        所以，她一路陪着他从医院回到了公司。
<br/>
<br/>        她在这等待的空暇里，坐在了一旁的玻璃圆桌前，拿着手机，看着自己手里偷拍的傅夜沉的相片。
<br/>
<br/>        他坐在办公桌前埋首看文件，他坐在会议室里开会……
<br/>
<br/>        手机里与他有关的相片也就那么几张，但每张都是他工作时的样子。
<br/>
<br/>        从今以后，她只能把对他的喜欢与爱，深深地埋藏在心里了。
<br/>
<br/>        玻璃帷幕窗外，湛蓝的天空，晨阳已经徐徐升起，一朵朵有如棉花糖般的白云，自由自在的飘过天际。
<br/>
<br/>        脑海里突然想到那个女人——童筱颖。
<br/>
<br/>        落以柔苦笑了笑，脸上有着说不出的羡慕与嫉妒。
<br/>
<br/>        “叮”地一声，奶茶煮好了。
<br/>
<br/>        须臾，她收起手机，拉回自己的小心思，从圆桌前起身，换上专业秘书该有的沉稳与冷静，走过去盛了一壶奶茶，然后倒入了一只精致的玻璃杯里，夹起几颗方糖丢进了杯中。
<br/>
<br/>        希望，大哥会喜欢。
<br/>
<br/>        落以柔抿嘴一笑，端起托盘，走出茶水间。
<br/>
<br/>        她端着奶茶进入总裁办公室，位子上的男人沉着脸看着一份企划案，一张英俊的脸庞略微僵硬，神情不悦，手下的钢笔，很不满意地企划案的资料里圈出一个又一个的问号。
<br/>
<br/>        傅夜沉看完后，将手上的企划案合上，递向落以柔：“小落，把这份企划案送回企划部，告诉部长，今天下班之前，我要看到一份完美的企划案。”
<br/>
<br/>        “是。”落以柔从傅夜沉的手中接过企划案抱在怀中，转身走出了总裁办公室。
<br/>
<br/>        这两天他没回公司，这一回来，就有一大堆的事情要处理。
<br/>
<br/>        傅夜沉忙了一个上午，不是批改文件就是看商业合同，不是在开会就是在去开会的路上，落以柔和郑忠威也没闲着，下面的员工更是一个个跟打了鸡血似的，一刻都不敢怠慢。
<br/>
<br/>        临近中午前，童筱颖再次为傅夜沉做了午餐，结果去医院扑了个空，被护士很鄙夷地告知他已出院。
<br/>
<br/>        瞧！
<br/>
<br/>        她这当妻子的是当得有多窘！
<br/>
<br/>        自己的丈夫已经出院了，她都毫不知情。
<br/>
<br/>        童筱颖不想给傅夜沉打电话，于是从医院里出来后，直接来到了傅氏集团大厦。
<br/>
<br/>        会议进行中时，落以柔接到了前台小姐打进来的电话，问：“落秘书，傅总的会议结束了吗？总裁夫人已经等了他一个多小时了。”
<br/>
<br/>        “让她继续等吧！我们这儿还没结束了。”落以柔起身离开了会议室，冷着脸说。
<br/>
<br/>        她说完就把电话给挂了，又回到了会议室里。
<br/>
<br/>        前台小姐皱了皱眉头，见电话被挂断后，抬眸看着童筱颖，一脸无奈地说道：“傅总还在开会，要不，您上楼去傅总的办公室里等？”
<br/>
<br/>        “好，麻烦帮我刷下卡。”童筱颖非常客气地说道。
<br/>
<br/>        有了第一次的教训，前台小姐这回对童筱颖可谓是毕恭毕敬。
<br/>
<br/>        童筱颖刚坐电梯上总裁办，碰巧遇上从楼梯口那边走过来的落以柔，赶紧迎了上去，问道：“落秘书，大哥他开完会了吗？”
<br/>
<br/>        落以柔手里正抱着会议纪要，没好气地说道：“夫人，傅总今天很忙。您要是闲着无聊，可以去逛街打发时间，没必要赖在公司里不走。”
<br/>
<br/>        听到落以柔用这样的口气跟自己说话，童筱颖真的是一点脾气都没有，反而十分淡定地将手中的保温饭盒递给了落以柔：“这是我给他做的午饭，等他开完会了，帮我拿给他。”
