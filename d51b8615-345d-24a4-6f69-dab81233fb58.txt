　　这一处宫殿，位于天商会中心，不过却极为寂静。
<br/>　　周围布满了无数禁制，封锁这灵识，让外面的人，无法洞察到其中人的交谈。
<br/>　　做生意，许多生意，自然都需要极为隐秘，这一点天商会做的，倒是天衣无缝。
<br/>　　这周围禁制，寻常生死境高阶的武者，都不一定可以动用灵识穿越其中，除非像是不死魔尊这样的存在，才能做到。
<br/>　　不过能够达到像是不死魔尊那等实力的强者，都是大人物，谁会去做这种窃听之事。
<br/>　　所以在这宫殿中交易，自然是最为安全无比的。
<br/>　　楚风眠来到宫殿中，便是毫不客气，坐在了其中的一座王座上，白萱也是做到了旁边。
<br/>　　李长老一进宫殿，便是对着楚风眠开口道。
<br/>　　“刚刚阁下手下留情，老夫在这里先谢过了。”
<br/>　　“这倒是没什么，在天商会中，自然要按照天商会的规矩来，不会在这里闹出人命的。”
<br/>　　楚风眠平静的开口道。
<br/>　　“我这里来天商会，有着一笔大生意，不知道你们天商会，要不要做？”
<br/>　　“大生意？”
<br/>　　李长老听到楚风眠的话，笑了笑道。
<br/>　　“不知阁下尊姓大名？应该是第一次来到我们天商会吧。”
<br/>　　天商会，可是混元大陆最为庞大的势力之一，还真没有多少他们天商会做不得的生意。
<br/>　　听到楚风眠的话，自然觉得楚风眠没有来过天商会，就是一个新人。
<br/>　　李长老笑着开口道。
<br/>　　“我们天商会，许多生死境高阶的强者，都会来做生意，不知阁下需要什么？我们天商会中应有尽有，不管是丹药，灵器，材料，天地奇物，甚至是一些情报，悬赏杀人，都可以做。“
<br/>　　“我叫楚风眠，不错，楚某是第一次过来。”
<br/>　　楚风眠平静的开口道。
<br/>　　“我准备来买一批天地奇物，以及一些锻体的丹药。”
<br/>　　楚风眠开口之际，便是拿了一张单子，直接递给了李长老。
<br/>　　楚风眠？
<br/>　　李长老听到楚风眠的名字，想了一下，在脑海中，北邙学院的强者中，并没有这样一个名字。
<br/>　　那也就是意味着，楚风眠应该是一个新崛起的天才，或者说是一个默默无闻的弟子，突然崛起。
<br/>　　这样的人，突然拥有如此实力，肯定是有着极大的机遇，这样的生意，正是他们天商会最喜欢做的。
<br/>　　听着楚风眠的话，李长老大笑一声道。
<br/>　　“天地奇物，锻体丹药，我们天商会应有尽有。”
<br/>　　说着，李长老便是接过了楚风眠递过来的单子，一双眼睛，扫视着上面一个个的名字。
<br/>　　“龙皇草？天晶？地母神元？这……”
<br/>　　每念到其中上面的一个名字，李长老的脸色都是越发的吃惊。
<br/>　　这上面的每一个名字，都是诸天奇物中品级极高的存在，尤其是像是天晶这等，简直就是传说中的名字。
<br/>　　李长老若非是博览群书，从古籍中见到过这个名字的话，只怕都不晓得这天晶到底是什么东西。
<br/>　　这里面的每一个名字，价值最低的，只怕都不在王品灵石之下。
<br/>　　李长老许久才开口道。
<br/>　　“这里面的许多东西，老夫都只有听闻过，从未见过，只有七八种，是我们天商会有的。”
<br/>　　“居然有七八种？”
<br/>　　这个回答，也是让楚风眠暗自有些心惊。
<br/>　　他这张单子上所写的诸天奇物，可都是对他有用的，以现在楚风眠的实力，真正对他有用的奇物，都是诸天奇物中品级极高的了。
<br/>　　本来楚风眠觉得天商会能够有三四种便是极限了，没想到居然会有这么多。
<br/>　　“这天商会，位于混元大陆，虚空海中盛产奇物，家底倒是丰厚的多，我现在正需要积累，将实力提升到神力境极限的地步，来准备度过生死大劫，能够得到这些奇物，对我有着大用。”
<br/>　　楚风眠心中暗自想到，脸上却是不以为然的说道。
<br/>　　“那就都拿过来吧，我全买了。”
<br/>　　这种不以为然的态度，活脱脱像是一个暴发户一般。
<br/>　　“这些东西的价值不菲，你确定要买？”
<br/>　　李长老犹豫了一下，出言提醒道。
<br/>　　“只怕是每一种，都比那龙血圣王丹价格要高。”
<br/>　　“当然。”
<br/>　　楚风眠点了点头，从他的手中，便是多出了一块王品灵石来，这王品灵石，正是楚风眠从颜古龙帝洞府中得到的。
<br/>　　每一块都已经被切割成了交易所用的大小，直接扔给了李长老。
<br/>　　“这块灵石，就算是楚某给李长老的见面礼了。”
<br/>　　“恩？”
<br/>　　李长老接过那块王品灵石，仔细一看，其中的灵力，简直是精粹到了极致。
<br/>　　在这一块灵石中的灵力，几乎已经是没有任何的杂质了，是纯净到不能在纯净的灵石，超越了现在武者，可以凝炼的灵石。
<br/>　　“王品灵石？这是王品灵石？”
<br/>　　李长老看着手中的王品灵石，吃惊的说道。
<br/>　　“这种王品灵石，如今已经是无法凝炼了，除非是一些远古洞府中，才有着王品灵石的存在，你居然拥有？”
<br/>　　楚风眠笑了笑，对李长老的惊讶，也并不奇怪，在这混元大陆中，交易所用的灵石，最多也就是极品灵石的范畴。
<br/>　　像是王品灵石，已经是极为少见了，只有一些远古洞府中，才能出现。
<br/>　　王品灵石，本身就足以算是天地奇物之列了，谁又会拿王品灵石去交易。
<br/>　　所以李长老才会如此吃惊。
<br/>　　“这块王品灵石，就算是我给李长老的见面礼，至于交易的灵石，还有着其他的。”
<br/>　　楚风眠开口之际，便是大手一挥，在他的面前，便是出现了数十块灵石，一闪而过。
<br/>　　这数十块灵石，每一块的品级，都要超越李长老手中的那块王品灵石。
<br/>　　其中的几枚，更是灵力精纯到极致，达到了皇品灵石的地步。
<br/>　　“皇品灵石！”
<br/>　　李长老的眼神中都是震惊无比，呆滞了一下，才急忙的开口道。
<br/>　　“楚公子果然富足，你所要的东西，都极为珍贵，在我们天商会的宝库之中，不如就由老夫带楚公子去宝库中看看，需要什么，自由挑选？”
<br/>(本章完)
