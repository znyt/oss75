　　不仅仅是围观的那些观众紧张，此刻场上的叶帝同样也是紧张万分，虽然他能够利用e技能来回复血量，不过毕竟回复的血量是非常的低的，这个时候如果他被对方的w技能命中，肯定是要身亡的。
<br/>　　所以这一波他不能出现任何的瑕疵，一定要做到非常的完美，看着追在自己身后的龙王以及挖掘机两个英雄，叶帝充满了专注。

<br/>　　台下的众多观众此刻都是屏着呼吸，不敢发出丝毫的声响。

<br/>　　抱着这种心情，此刻两人在身后紧紧的追着叶帝，虽然叶帝利用e技能回复了一些血量，但是在两人的眼中，这些血量还是不足为据的，只要给他们机会他们肯定是能够击杀叶帝的。
<br/>　　而且在他们心里也在计算着，这个时候距离叶帝疾步消失的时间已经没有多少了，这一波对方完全没有逃出去的道理。
<br/>　　龙王率先走进了叶帝这边的中路防御塔的攻击范围，而挖掘机因为刚刚被叶帝的大招命中，此刻虽然解除过来，但是距离叶帝已经有些远了，所以他只能跟在龙王的身后。
<br/>　　至于为什么跟着，主要还是他怕夜雪的蜘蛛支援过来，自己这样在龙王的身边也是能够起到一个保驾护航的作用。
<br/>　　只不过等到他们两人进入防御塔范围的时候，上路已经是传出了一道击杀声，原来就在挖掘机出现在中路的一瞬间，夜雪狂歌这边同样也是来到了上路，配合着皇子的伤害，两人在塔下直接强秒掉了对方的上单慎。
<br/>　　只不过这个时候镜头并没有给到上路，以为相比于上路而言，此刻中路这一波更显惊心动魄。
<br/>　　看到对方进入到自家的防御塔范围，叶帝并没有第一时间选择反打，而是继续拼命的向前跑，当然在跑的时候他还时刻注意着对方护体行星的位置，每次都能够险之又险的躲过去。
<br/>　　当看到龙王走到自家的中路防御塔前的时候，眼看着自己的疾步已经消失，这个时候叶帝终于是不再选择逃跑，因为眼下这种情况，在没有疾步的加成的情况下，他想要跑过已经出了法穿鞋的龙王，那明显是不可能的。
<br/>　　不过同样的在被叶帝输出了一圈，又被防御塔打了一下，虽然龙王是饭盒出装，但是这个时候也是仅仅只剩下了半血多一点。
<br/>　　下一刻在无数人的注释下，叶帝手中的w技能覆地终于是朝着龙王的身前释放了出去，他这个位置释放的非常刁钻，是卡在龙王的身前的位置释放出来的。
<br/>　　如果龙王敢继续深追，那么他就将一脚踩在蛇女的这个w技能上，不过这个时候如果不继续追击的话，那么两人之前一连串的操作可就白白释放了，加上此刻叶帝的血量还这么少，任谁都想上去摸一下的。
<br/>　　可以说叶帝就是卡着对方的这个心理来释放w技能的，当看到龙王一脚踩在自己的w技能上的时候，叶帝原本凝重的脸上顿时露出了一抹轻松的笑意。
<br/>　　下一刻他手中的e技能已经是朝着龙王的身上释放了过去，并且中间还释放了一次q技能，而此刻龙王除了自己的w技能之外，q技能还在冷却之中，e技能则是一个位移技能，根本无法用作输出，到了这个时候龙王的尴尬境地终于是显现了出来。
<br/>　　龙王一心想击杀叶帝，但是有着w技能的减速效果，虽然他已经做出了法穿鞋，但是愣是无法用自己的w技能外圈打中叶帝。
<br/>　　而到了这个时候eP战队的中单终于是发现了自己的尴尬境地了，就在这个时候他的血量已经是不足四分之一了，被防御塔和叶帝两人接连输出，在十分钟这个时间段就算是他做出了饭盒也是没有用的。
<br/>　　至于挖掘机此刻还在后方向着叶帝这边接近着，只不过此刻距离叶帝这边还有一段距离，他就算是有心想要打叶帝一下，也没有办法。

<br/>　　本身挖掘机过来的时候就只有三分之二左右的血量，加上对方还不肉，在叶帝和防御塔的输出下，仅仅2秒钟的时间，挖掘机便是随着龙王的身影而去。
