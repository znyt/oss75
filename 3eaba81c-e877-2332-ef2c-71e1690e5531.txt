　　第487章
<br/>　　知道程大雷回来的消息，和珅带着山寨的弟兄在城门迎接。
<br/>　　见到和珅，程大雷算是松了口气。他还真担心自己不在的这段时间，和珅把自己的山寨篡了。面对这第一贪官，程大雷难免有这种担心。不过，看到眼前的场面，程大雷知道自己的担心是多余了。
<br/>　　“大当家一路辛苦，这次出去收获怎样？”和珅陪着程大雷进城。
<br/>　　程大雷长叹口气：“嗳，别提了，这一路可是累坏了。”
<br/>　　大军进城，在和珅的安排下开始休整。伤员的治疗，牺牲者的抚恤，这一路行来，其实每个人都疲惫不堪。而大军携来的宝物，也交到和珅那里，由他归账后，存入仓库。
<br/>　　做这种事情，和珅要比刘悲擅长许多。兹事体大，程大雷拖着疲惫之躯，也亲自参与了这件事。
<br/>　　宝物一件件，摆在大厅前的空地上，不知引得山寨多少人前来围观。和珅竟然还有些见识，每件宝物都能叫上名字，记在账本上。当然，是宝物本身叫这个名字，还是他瞎编的，那就无人知晓了。
<br/>　　饶是如此，和珅是被震得一愣愣的，这些连城之宝，从来都是天上有，地上无，程大雷怎么出去一次，搞回来这么多。
<br/>　　“大当家，你该不会是把皇宫抢了吧？”
<br/>　　“也……差不了多少吧。”
<br/>　　程大雷本来心情还蛮开心，和珅这句话却又勾起他的烦心事，因为他想起这些宝物是从何处来。林少羽当初攻破长安，掠夺了许多宝物，怕帝国皇宫也被他洗劫一空。
<br/>　　如今这些宝物自然多有遗失，可经过一番流转，大部分还是落入了程大雷手中。
<br/>　　程大雷挥挥手，想将愁绪情况。死的人已经死了，活着的人还得活着，总不能为一件事一直牵心。
<br/>　　此刻城主府外围了许多人，都是来参观宝物的。有负责警戒的弟兄，免得把场面搞乱了。
<br/>　　“好啦，好啦。”程大雷冲外面的人道：“要看就站在外面看，不许往里闯，其实也没有什么好看的，一堆烂石头破铁而已。”
<br/>　　说着这话，程大雷见李婉儿也在人群中，倩影一闪而逝，等程大雷定神看时，她已转身离去。
<br/>　　程大雷心里忽然咯噔一下，李婉儿是帝国的明玉公主，这些宝物都是她家里之物，说不得她幼时贪玩，还曾把玩过几件。如今蛤蟆寨欢天喜地，她却睹物思人，这种欢喜对她反而是一种刺激。
<br/>　　程大雷将清点财物的事情交给和珅，自己离了城主府，去寻李婉儿。却是没想到，来到李婉儿的住处，她竟然不在。问过人才知道，她这个时间，大概在奥黛琳娜处。
<br/>　　奥黛琳娜，诺顿王朝万物教的圣女，生得国色天香，堪称蛤蟆城第一美女一点也不过分。不过，奥黛琳娜来到蛤蟆城后经历有些悲催，她抱着一腔传教的神圣信念来到东方，要让诸神的光芒普照这些没有信仰的恶人。
<br/>　　却是没想到，这些东方的恶人太恶了。对伟大的神明根本谈不上尊重，奥黛琳娜来了这么久，真正的信徒一个没能发展，当然，名义上的信徒还是蛮多的，可是论他们信仰的坚定程度……奥黛琳娜几乎不报希望。
<br/>　　不仅如此，因为奥黛琳娜经常和刘悲接触，都快要被他极端悲观的心念影响，成为某个空无致哀教派的创始人。
<br/>　　程大雷去的时候，奥黛琳娜坐在屋檐下和李婉儿相对饮茶，见到程大雷过来，奥黛琳娜微微点头致意，入乡随俗，到了东方后，她也学了许多这里的礼仪。
<br/>　　“哈，你果然在这里，我还担心找不到你呢。”程大雷见到李婉儿反而松口气。
<br/>　　这奥黛琳娜竟然颇识趣，起身把位置让给程大雷，之后便离开了院子。
<br/>　　一路从并州过来凉州，已经到了春季，万物复苏，暖风习习。程大雷自己给自己倒了杯茶，目光落在李婉儿身上，发现李婉儿仍旧是一副心事重重的样子。
<br/>　　“你勿要担心，你大哥很好，你六哥也很好，至于你其他的哥哥姐姐弟弟妹妹……说实话，我也不认得，但也应该很好吧。”
<br/>　　面对李婉儿，程大雷起身有些为难。林少羽是与程大雷是磕头换命的交情，然而李婉儿的父亲也的确死在林少羽手中，而宫中那些皇妃公主其实也没少遭正义教糟蹋。
<br/>　　程大雷绝对相信林少羽不会做出这种事，但他手下的正义教信徒，程大雷可就不敢对他们的道德水准打包票了。
<br/>　　简单的把并州发生的事情说了一遍，自然省略掉很多。但李婉儿一直没有太大反应，低头饮茶，两眼放空。
<br/>　　程大雷发觉自己有些看不透李婉儿，她若还有当初在长安时的泼皮不讲理，程大雷倒是能松口气。
<br/>　　“我却有件事，正要与你说。”李婉儿开口道。
<br/>　　“什么事？”程大雷忙道：“有事尽管包在我身上？”
<br/>　　“我打算皈依万物教。”
<br/>　　“呃……”程大雷眼睛瞪直了：“士别三日当刮目相待，我们才多久没见，你的幽默感就进步到这种程度了么？”
<br/>　　“我没有开玩笑。”李婉儿平静道：“你不在的这段日子，我经常与奥黛琳娜相处，她与我宣讲教义，我觉得她的话很有道理，万物有灵，万物有神，我若答应皈依万物教，奥黛琳娜便让我做正义教的圣女。”
<br/>　　“呃……”程大雷的眼睛已经没办法睁得更大：“你做圣女，那奥黛琳娜做什么，难道做教……皇？”
<br/>　　“不，她放弃圣女的身份，成为一名普通信徒，然后择一黄道吉日完婚。”
<br/>　　“完婚？”
<br/>　　一连串突变搞得程大雷头昏眼花，他摆着手：“你别着急，让我顺一顺，她怎么要成亲了，不是说圣女不能成亲么？”
<br/>　　“万物教的教义是很自由的，喝酒吃肉，生儿育女都和普通人无异，只要心底保留一份信仰就好了。”
<br/>　　“那么，究竟是谁让她能放弃圣女的身份，选择做一个普通女人呢？”程大雷认真问。
<br/>(本章完)
