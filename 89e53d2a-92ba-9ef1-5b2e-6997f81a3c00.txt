　　归墟天中，有着无数奇地，据说其中的一片区域，便是常年冰冷无比，从中足以诞生无数冰系圣药。
<br/>　　对寒月漓的玄寒之体来说，这里正是最好的修炼地方。
<br/>　　当然像是这样的地方，在整个归墟天中，并不是少数。
<br/>　　这归墟天仿佛便是将天地之间的无数奇地凝聚在了一起，形成了一方区域。
<br/>　　“在这其中遗址之中，有着曾经远古剑宗的遗址，若是可以碰到，领悟其中剑道，对我的九域剑术，也有着大用。”
<br/>　　楚风眠也是自言自语一句，直接开口道。
<br/>　　他的九域剑术，便是要贯穿古今，曾经远古时代的剑术，其中的种种也是玄妙无比，正好是给楚风眠带来领悟。
<br/>　　两人一边交谈这，一边已经是前来了北邙城。
<br/>　　在北邙城，有着通往其他八域之中的传送法阵。
<br/>　　楚风眠跟寒月漓二人，在其中连连穿行。
<br/>　　就算是利用这传送法阵，几乎一瞬之间便是就可以达到万里之外的地方。
<br/>　　他们二人在其中辗转反侧，也足足是花费了十多天的时间，才终于抵挡西域。
<br/>　　九域之大，无边无际，仅仅一个北域，若是想要穿行，让楚风眠动用遁光去飞。
<br/>　　就算是现在楚风眠拥有神脉，灵力无穷无尽，几乎不需要任何停留，也至少需要几个月的时间，利用这传送法阵，都需要至少十日。
<br/>　　在西域，他们二人又是连连通过传送法阵，终于是来到了归墟城。
<br/>　　这归墟城，便是距离归墟天，最近的一座城池。
<br/>　　正是因为归墟天的原因，所以是从周围建立起了这样一座城池，这座城池建立的极为繁华，却并没有多少人。
<br/>　　因为来到这里的，多半都是要进入到归墟天的，而其他人，没有进入归墟天的资格，来到这里也没有任何意义。
<br/>　　归墟天的名额，每一次开启只有一百个，所以人并不会多。
<br/>　　一眼望去，这归墟城中，有的只有大大小小，数十座洞府，用来给武者休息所用。
<br/>　　“归墟天开启，还有十日时间，先要找个地方，休息才是。”
<br/>　　楚风眠算了算时间，便是目光看向了那些洞府。
<br/>　　这些洞府建立出来，便是给这些进入归墟天的武者，一个休息的地方，这十日时间，正好是可以找个洞府，休息一下。
<br/>　　一边想这，楚风眠跟寒月漓便是一下子落入到归墟城中去。
<br/>　　“站住！来着何人？”
<br/>　　在归墟城中，几尊武者，猛然从中出现，直接将楚风眠跟寒月漓，给围困在了其中。
<br/>　　其中一尊武者，更是直接开口道。
<br/>　　“归墟城，不许闲杂人等进入，速速退去！”
<br/>　　“我们是北域武者，得到资格，前来归墟城，等待归墟天开启。”
<br/>　　楚风眠一边说着，一边大手一挥，两道令牌，便是直接飞了出去。
<br/>　　“原来两位是来参加归墟天的天才。”
<br/>　　那尊武者，接过身份牌，看了一眼，脸色便是变得缓和了许多，笑着开口道。
<br/>　　“刚刚多有冒犯，还请两位不要介意，两位请进。”
<br/>　　那尊武者一边说着，一边便是带着楚风眠和寒月漓二人，直接进入到了归墟城中。
<br/>　　对着武者态度变化的如此之快，楚风眠也并不意外，能够来参加这次归墟天的武者，都是九域的天才。
<br/>　　就算是从北域中出身的天才，也是要远远超越大部分常人。
<br/>　　这种天才，将来很有可能便是成为九域巨头的存在，得罪谁都不是明智的选择。
<br/>　　楚风眠心中自然也清楚无比，懒得去计较什么。
<br/>　　“这位兄弟，如何称呼？”
<br/>　　楚风眠看着那尊武者，突然开口道。
<br/>　　眼前这武者，看起来只是归墟天的一尊护卫，不过他的实力，也是达到了生死境高阶的地步。
<br/>　　放在这西域，绝对也算是一尊响当当的人物了，西域的整体，跟北域极为相近，不能说是一方老祖，也是至少一位大人物。
<br/>　　“王辉，终焉王朝的人。”
<br/>　　那名为王辉的武者，平静的开口道。
<br/>　　“王辉？”
<br/>　　听到这个名字，楚风眠脑海中仔细转动这，来到了西域之中，楚风眠也是从不少人的交谈中，了解了一些关于西域的情况。
<br/>　　“扬威公？”
<br/>　　楚风眠有些诧异的开口道。
<br/>　　这个名字，正好是楚风眠曾经听到过的，在终焉王朝之中的扬威公，正是这个名字。
<br/>　　“虚名，不足挂齿。”
<br/>　　王辉笑了笑道。
<br/>　　看来对着归墟城，终焉王朝也是极为看重，毕竟其中前来的，都是九域天才，甚至是七大宗门的天才，都会前来。
<br/>　　这随便一位护卫武者，都是在西域响当当的人物。
<br/>　　“王兄，我这里需要一处安静灵气充盈的洞府来休息，还请王兄好好安排一下。”
<br/>　　楚风眠突然开口道，不动声色之间，楚风眠手中便是多出了一块灵石，直接递给了这王辉。
<br/>　　“这是？”
<br/>　　接过灵石，王辉便是感觉到一种极为精粹的灵力，令他有些震惊。
<br/>　　“皇品灵石？”
<br/>　　皇品灵石，已经不是属于这个时代的东西了，寻常武者，根本便是都无法凝聚出来。
<br/>　　也只有一些隐藏的大人物，老古董可以凝聚。
<br/>　　但是这些大人物，老古董，又有几个愿意浪费时间，来凝聚这皇品灵石的。
<br/>　　就算是放在九域中，皇品灵石都足以算得上是一件诸天奇物，价值不菲。
<br/>　　“好大的手笔。”
<br/>　　王辉不动声色间，也是将那皇品灵石收下。
<br/>　　他们这些来归墟城作为护卫，应该是不允许索要任何好处的，被人抓到把柄，也是极为麻烦的一件事。
<br/>　　但是这皇品灵石的确是价值不菲，足以令他铤而走险一下。
<br/>　　“洞府？好说，这归墟天中上九天的洞府中，正好有着一座空着的，我就带二位前去。”
<br/>　　王辉笑着开口道。
<br/>　　像是楚风眠这样性格平静，出手大方的天才，也让他极为开心。
<br/>　　九域天才之中的许多人性格，都是极为极端，有的狂妄，有的冰冷，像是楚风眠这样性格正常的，却是不多，王辉也愿意照顾一二。
<br/>　　他们这些护卫的地位虽然不高，但是也有着一些权力，尤其是这分配洞府上，可以自由分配。
