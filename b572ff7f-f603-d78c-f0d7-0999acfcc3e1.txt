　　虽然前世，叶尘只在天海大学待了一年，便被叶无伤诬陷杀人，逃到了海外。
<br/>　　不过，在这短短的一年时间里，叶尘和这三位室友，倒也建立不错的关系，虽然比不上好兄弟吴磊，却也算是他前世为数不多的好朋友。
<br/>　　三人看到突然出现的叶尘，也纷纷望了过来，
<br/>　　“喂，哥们，你就是我们宿舍的最后一名成员，叶尘？”
<br/>　　为了防止找错房间，宿舍门口都贴着入住新生的名字。
<br/>　　他们三个，应该已经互通过姓名，剩下的自然也就是叶尘了。
<br/>　　叶尘点了点头，微微一笑，
<br/>　　“是我，你们好！”
<br/>　　三人一听，当即纷纷走了上来，
<br/>　　“还真是！没想到咱们宿舍，竟然还有一个大帅哥！以后哥几个的幸福，可就全指望你了！”
<br/>　　说话之人，是一个身材偏矮，有些微胖的少年，不过两眼亮晶晶的，给人一种很精明的感觉。
<br/>　　“对了！自我介绍一下，我叫李虎！”
<br/>　　矮个少年李虎介绍完自己，又指向另外两个人，一个身材高大魁梧，一个则偏文弱，
<br/>　　“这个大块头叫唐威！”
<br/>　　“还有他，他叫陈东！”
<br/>　　唐威和陈东也纷纷冲着叶尘微微一笑，算是打过招呼。
<br/>　　叶尘心想说，
<br/>　　我又怎会不知道你们？
<br/>　　不过，
<br/>　　叶尘的目光，从唐威身上扫过的时候，忽然感应到了一股不弱的气息波动！
<br/>　　顿时心头一动，
<br/>　　‘原来唐威竟然是内劲武者，难怪前世在学校里，这家伙打架从来没有输过……’
<br/>　　根据叶尘的判断，唐威此时的修为，应该才勉强迈入暗劲，不过对付寻常人，肯定是绰绰有余了。
<br/>　　而与此同时，唐威也在悄悄观察叶尘，
<br/>　　也不知道为何，唐威下意识的觉得，自己的这个室友，似乎很不简单，可是仔细观察了片刻，却又看不出任何出奇之处。
<br/>　　三人中，显然以李虎最为活泼，而且还有些话痨，没等其他人开口呢，他接着又道：
<br/>　　“哎，哥们，你的行李呢？你该不会什么都没带就来了吧？”
<br/>　　叶尘笑了笑，随口道：
<br/>　　“还没来得及买！”
<br/>　　不料，李虎十分的大气的一挥手，
<br/>　　“我这就给我爸的秘书打电话，让他帮你多准备一份！”
<br/>　　说完直接掏出一个苹果手机，拨通了一个电话，
<br/>　　“喂，徐叔叔，我要的那些生活用品，帮我再多买一份送过来！”
<br/>　　叶尘微微一愣，倒也没有拒绝，买那些生活用品，确实挺麻烦的，若是有人帮他准备，那自然再好不过。
<br/>　　“多谢了！待会你告诉我多少钱，我直接转给你！”
<br/>　　叶尘隐约记得，这李虎家里好像是做药品生意的，算是一个富二代。
<br/>　　李虎听到叶尘这话，摆了摆手道：
<br/>　　“什么钱不钱的？以后我们就是兄弟了，这就当是哥们送你的见面礼！只要你以后泡妞的时候，带上兄弟我就行了！嘿嘿！”
<br/>　　说到最后，李虎的脸上，露出一抹猥琐的笑容。
<br/>　　叶尘不由得一阵无语，
<br/>　　在他的记忆里，李虎这家伙有一个最大的缺点，那就是好色。
<br/>　　几人又随便聊了一会，渐渐的也就熟络起来，然后按照惯例，根据各自的年龄排了大小。
<br/>　　陈东年龄最大，其次便是叶尘，唐威排第三，李虎年纪最小。
<br/>　　几人正聊着呢，忽然外面响起了敲门声，坐在门口位置的陈东，当即起身打开房门。
<br/>　　众人顿时愣住了，
<br/>　　只见门外，出现了一道靓丽的身影，
<br/>　　居然正是之前，带叶尘报到的那个戴丽丽。
<br/>　　戴丽丽一双明亮的大眼睛，在屋内扫了一圈，很快便锁定在叶尘的身上，脸上露出一抹喜色。
<br/>　　李虎看到戴丽丽，顿时眼睛都要看直了，当即快步上前，笑眯眯道：
<br/>　　“戴学姐，你怎么来了？快快里面请！”
<br/>　　戴丽丽抿嘴一笑，
<br/>　　“不了！我来就是通知你们一下，今天晚上七点，学院打算在南门的君悦酒店，给你们举办一场欢迎会，你们可一定要记得参加哦！”
<br/>　　李虎听到这话，当即一拍胸脯，
<br/>　　“学姐放心，我们一定准时参加！对了，学姐也会参加吗？”
<br/>　　戴丽丽笑着点了点头，
<br/>　　“你们也不用太紧张，这只是我们学院弄的一个简单的联谊会，大二大三的很多同学，也都会过去，等过几天，学校还会有正式的迎新晚会呢！”
<br/>　　说完这话，戴丽丽并没有立即离开，目光又在叶尘身上看了一眼，犹豫了片刻之后，仿佛鼓足了勇气，道：
<br/>　　“叶尘学弟，我们晚上见！”
<br/>　　说完这话，戴丽丽便逃也似的离开了。
<br/>　　戴丽丽一走，李虎三人顿时纷纷望向叶尘，眼中全都透着一抹羡慕之色。
<br/>　　李虎更是冲到叶尘跟前，直接在他胸口上砸了一拳，
<br/>　　“叶哥，你牛逼啊！戴学姐可是咱们学院有名的大美女，这么快就被你搞定了？”
<br/>　　叶尘不由得摇了摇头，淡淡道：
<br/>　　“我对她没兴趣！”
<br/>　　听到这话，李虎三人顿时更加不淡定了，
<br/>　　“不是吧！戴学姐这样的美女，你都看不上，你丫眼光也太高了吧？”
<br/>　　“装！你就继续装！”
<br/>　　“我就不信，你就一点想法也没有？”
<br/>　　三人都是血气方刚的年纪，见叶尘一脸淡然的模样，自然觉得他在故意装清高。
<br/>　　叶尘淡淡一笑，没有多说什么。
<br/>　　至于晚上的迎新会，他压根就没有打算参加。
<br/>　　……
<br/>　　一晃眼，已经到了中午时分，
<br/>　　原本李虎提议大家一块吃顿饭，不过叶尘以有事为理由拒绝了。
<br/>　　出了宿舍之后，叶尘神念一动，便轻松锁定了萧若曦的位置，然后便慢悠悠的走了过去。
<br/>　　曦月每天只能占用萧若曦的身体两个小时，因而三人经过商议，每天中午相见，至于其他时候，叶尘不会干涉萧若曦的私人生活。
<br/>　　不一会，
<br/>　　叶尘来到经济管理学院的教学楼前，而萧若曦也刚好下课，从大楼中走了出来。
<br/>　　叶尘正要走过去，就在这时，一名英俊青年男子，手里捧着一束玫瑰花，拦在了萧若曦的面前，
<br/>　　“若曦，我喜欢你很久了，请做我的女朋友吧！”
<br/>　　萧若曦作为天海大学四大校花之一，平日里自然有不少追求者。
<br/>　　看到这一幕，叶尘顿时眉头一皱，
<br/>　　按理说，他和萧若曦有过协议，不会干涉萧若曦的个人生活，可是一想到萧若曦体内的曦月，叶尘顿时便忍不住了，直接大踏步走了过去。
