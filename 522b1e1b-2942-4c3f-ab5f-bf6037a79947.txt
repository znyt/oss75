
<br/>　　<br/>　　    当王惊蛰登上山头，往陈重收的那一块地皮望去的时候，从他这个方向看那块地正好比周边其他地方低洼了一些，四周略高中间偏低形成了一个类似于盆地的地形，就好像有一大片阴影笼罩在那块地皮上，这在风水上讲叫聚阴地，如果是建阴宅的话还算不错，但若建了阳宅那一块地里的人，长年累月下必定大病缠身小病不断，诸事不顺。<br/>　　<br/>　　    但除了聚阴，这块地还有个地方更怪，地的四面形状极其不规则，东面倒三角，南边锤子形，西面有棱有角，北面拉成一条线，西北对面的一座山上有个天然的洞穴正好冲着这边，东南一条大河蜿蜒而过。<br/>　　<br/>　　    风水地上有一种祥地叫四圣压阵，就是青龙白虎朱雀玄武各占四方稳稳的压着这块地，乃是中兴大福的地方，但也有一种大凶之地是被四大凶兽压着的，混沌、穷奇、梼杌、饕餮压地，任你哪怕是帝王命，在这块地上也得折戟沉沙了。<br/>　　<br/>　　    最关键的是，这块地皮正上方，两座光秃秃的石头山之间有一道长长的山谷正好对着这里，犹如一把利剑指了过来，正正的插在了这块地皮的中间点上。<br/>　　<br/>　　    这种大凶之地再往上一层，那就是风水地上几乎难得一见的绝版凶煞地，阎王坟场了。<br/>　　<br/>　　    所谓的阎王坟场，从字面上来了解的话，那自然就是阎王爷的坟场了，据说在阴间有那么一块地方，是专门用来埋葬十八层地狱里被镇压的那些曾经犯过穷凶极恶大罪的亡魂厉鬼的。<br/>　　<br/>　　    那里煞气冲天阴气滔滔，无时不刻都在鬼哭狼嚎着，刮起的阴风都跟刮骨钢刀一样，到处都是没有墓碑的坟头，这就是所谓的阎王坟场。<br/>　　<br/>　　    在阳间，自然是找不到这么一块埋葬无数亡魂厉鬼的地方，顶多也就是能碰到个乱坟岗就差不多了，不过这块地方就有点特殊了，早年间这里还是个镇子，距离宜宾不太远，两地中间有一所牢狱，但凡有行刑砍头的犯人都会拉倒这里来上铡，那些年间据说此地被斩首的犯人都不知道有多少了。<br/>　　<br/>　　    后来王朝更替，此地就成为了兵家必争之地的战场，两兵相争交战，无数人倒在了这片地上，据说死后的人流的血将这里的土地都给染红了。<br/>　　<br/>　　    如此一来，再加上这里得天独厚的四凶压地风水格局，阎王坟场就逐渐形成了。<br/>　　<br/>　　    王惊蛰在震惊之余，也明白为啥会有人对陈重下手了，这块地方对别人来讲可能是大凶之地，但对那么一小撮人来说，却是千载难逢的风水宝地了。<br/>　　<br/>　　    “如果······”王惊蛰回身看向陈臣，叹了口气说道：“如果这块地皮对你爸来讲，不是那么特别太重要的话，你最好劝他还是放手算了”<br/>　　<br/>　　    陈臣皱眉说道：“为什么？”<br/>　　<br/>　　    “这块地对你爸可能产生的就只是经济利益而已，但对有些人来讲却是块宝地了，你爸端着一块让别人垂涎三尺的聚宝盆，人家不强取豪夺都怪了，信我的吧，如果没什么大的损失就干脆放弃算了，如果你爸死守着不放的话，我猜人家是没那么容易就轻易放弃的，这次手段没奏效，下次他们可能还会接着下手，防贼没有防千日的，你不觉得累么？更何况人家在明，你们在暗啊，抽冷子阴你一下谁受的了？”<br/>　　<br/>　　    陈臣仔细衡量了一下，觉得也是，一块不太值钱的地皮而已，她家又不差这个钱，蜀地有太多的地可以拿了，真没必要守着一块烫手的山芋来闹心。<br/>　　<br/>　　    不知为何，陈臣对王惊蛰的话十分信服，可能是处于前两次她被救了的原因，当下她就拿出电话要给陈重拨过去。<br/>　　<br/>　　    但这时候，王惊蛰看着陈臣的面相，突然心头就狂跳了下。<br/>　　<br/>　　    “陈先生今天有什么安排，或者要去哪里你知道么？”王惊蛰忽然拦了下陈臣说道。<br/>　　<br/>　　    陈臣愣了愣，摇头说道：“不太清楚，一早我就从家里出来了，他这个时候要么去公司了要么就是在医院陪我妈呢”<br/>　　<br/>　　    “马上，快点，给你爸打电话，不管他此时在哪在干什么，让他都不要再动了·····”<br/>　　<br/>　　    陈臣张了张嘴，不解的问道：“为啥？”<br/>　　<br/>　　    “你哪里来的那么多废话？你爸可能要突遭横祸，你再磨叽一会就来不及了，快点给他电话，就现在”王惊蛰忽然之间发现，陈臣面相在刚才那一瞬间悄然发生了改变。<br/>　　<br/>　　    她天中塌陷高低眉，天中在脑门正中从发迹下两指左右，天中凹陷多为破祖的面相，损父母，山根左面塌陷为父亡，右脸放低则母死。<br/>　　<br/>　　    反之如果人的山根两侧要是饱满圆滑的话，则表明父母长寿，最关键的是陈臣刚刚左眉稍低，眉尾端宛若一把刀子直刺左面，这就表明她父亲有突遭横祸的迹象了。<br/>　　<br/>　　    见过陈重两次，王惊蛰对他的面相看得很清楚，他这时候正值壮年呢，再加上有红色血脉承受祖辈阴德，他没有横死早毙的面相，所以王惊蛰就看出来了，陈重今天有难估计十有八九是被人给算计，要突遭横祸了。<br/>　　<br/>　　    王惊蛰急促的催促了两句，陈臣慌忙拨出他的号码，响了几声之后电话接通了，她直接问道：“爸，你在哪里？”<br/>　　<br/>　　    “我和老马在去双流的路上，车开了能有一半的距离了，怎么了小臣？”<br/>　　<br/>　　    陈臣豁然一愣，她爸果然没有在公司和医院，就连忙说道：“你快停车，先不要走了·····”<br/>　　<br/>　　    “你搞什么鬼呢？”陈重明显有点发懵，看了眼车外面，此时车正行驶在路上，左边一侧是山体右边是坡道。<br/>　　<br/>　　    “爸，你有危险，快点停车啊，不要再开了”陈臣急的直跺脚。<br/>　　<br/>　　    “唰”王惊蛰一把将陈臣手里的电话拿了过来，语速极快的说道：“陈先生，我是王惊蛰，你在路上可能要出意外，你如果要是信我的话，马上停车不要再动了，就现在”<br/>　　<br/>　　    陈重顿时皱眉，狐疑不解，王惊蛰在电话中接着说道：“停下车而已，耽误不了你什么事吧？但你要是不停的话，你可能连后悔的机会都没有了”
