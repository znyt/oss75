　　被斩杀的郭剑青，体内溃散的玄力、气血和战魂等等一切，已经被吞噬战魂吸纳了过来。
<br/>　　而赵城整个人又被直接吞噬了，两名武帝境四重巅峰武者，为苏莫提供了足够的资源。
<br/>　　上一次，他冲击境界没有成功，一则是因为刚突破到武帝境一重不久，沉淀不够，根基不够扎实。
<br/>　　二则是资源太少，所以才会失败。
<br/>　　现在，两名武帝境四重巅峰武者成了苏莫的资源，他自然要再次尝试冲击武帝境二重境界。
<br/>　　轰隆隆！！
<br/>　　苏莫体内，玄力如咆哮的江河，战魂之内海量的玄力不断的涌入他的体内。
<br/>　　由于玄力太过浑厚，苏莫的肉身险些承受不住，全身经脉都有些许胀痛。
<br/>　　这是因为他的肉身强度太弱了！
<br/>　　肉身，他已经很久没有修炼了，主要是因为资源不够用，其次是肉身的力量，对于他实力的增幅，已经非常的小了。
<br/>　　他的修为实力太过强大，除非肉身实力远超修为境界，不然对于他的实力，根本没有大的作用。
<br/>　　所以，这么长时间以来，苏莫都没有再修炼肉身。
<br/>　　他暂时已经放弃了修炼肉身，太古龙象诀虽然强大，但修炼到极致，也就堪比武尊境界。
<br/>　　所以，他已经不准备修炼太古龙象诀了，尽量节省资源。
<br/>　　若是日后，能得到更为强大的肉身功法，再做考虑。
<br/>　　轰隆隆！
<br/>　　苏莫忍着全身经脉的胀痛，疯狂的炼化，好在这胀痛还在他的承受范围之内，不至于损失他的经脉和肉身。
<br/>　　五枚神丹剧烈的震动，仿似要突破到更高的层次，但有一层无形的壁障，始终束缚着神丹突破。
<br/>　　但苏莫并不着急，赵城和郭建青两人，全身的精气、玄力，非常的庞大，能支撑着他不断的冲击。
<br/>　　时间缓缓流失。
<br/>　　一个时辰又一个时辰，足足过去了十个时辰，到了第二日清晨，苏莫还没有突破。
<br/>　　他的额头上已经布满了汗水，十个小时不间断的冲击境界，他非常的疲累。
<br/>　　不过，苏莫的心中却是暗暗激动了起来，因为境界瓶颈已经松动了，突破已经近在眼前。
<br/>　　苏莫继续冲击境界，一刻不停。
<br/>　　终于，又过了一个时辰之后，他的身躯猛然一震，一股浩大磅礴的气势，突破从身上升腾而起，直冲云霄。
<br/>　　这一刻，他终于突破了。
<br/>　　武帝境二重境界！
<br/>　　苏莫没有停止，继续修炼，虽然修炼了十一个时辰，消耗了大量的玄力和精气，但还有不少。
<br/>　　就这样，又过了一个多时辰，苏莫才结束了修炼。
<br/>　　他的修为，已经达到了武帝境二重的中期阶段，距离二重巅峰也不远了。
<br/>　　呼！
<br/>　　苏莫睁开眼眸，张口突出一口浊气。
<br/>　　浊气如剑，洞穿了虚空，爆射百丈之远。
<br/>　　“武帝境二重！”苏莫面上露出了一丝笑容，心中却是叹了口气。
<br/>　　这境界的突破，完全是硬生生冲上去的，也不知道会不会影响根基！
<br/>　　看来以后，还是多多沉淀为好，虽然修炼速度可能会慢一点，但一切水到渠成，浑圆如意。
<br/>　　少倾，苏莫意念一动，吞噬战魂之中光芒闪烁，三样物品飞了出来。
<br/>　　分别是一柄金色长剑，一块黑色盾牌和一枚灰色的储物戒。
<br/>　　金色长剑自然是郭剑青的兵器，黑色盾牌和储物戒，则是赵城的宝物。
<br/>　　两人被他吞噬的渣都不剩，身上的兵器自然也被吞噬了。
<br/>　　只不过，这长剑和盾牌品质太高，储物戒也是高级武者特制的储物戒，就算被吞噬战魂吸收了，也无法被炼化。
<br/>　　至于郭剑青的储物戒，被一轮星宿剑暴扫荡而过，已经荡然无存了。
<br/>　　苏莫查探了一番赵城的储物戒，兵器、玉简、衣物等杂七杂八的物品不少，下品灵晶也有五百余块。
<br/>　　他也没有仔细查探，便将储物戒和金色长剑收了起来，唯独剩下了那块黑色的盾牌。
<br/>　　“好宝物！”打量了一番黑色盾牌，苏莫眼眸一亮，此盾虽然不是尊器，但也算是最顶级的帝级上品宝器了，防御力非常的强大。
<br/>　　无论是自己使用，还是拿出去售卖，都值不少钱财。
<br/>　　片刻之后，苏莫收起盾牌，离开了此地，向星空传送大阵所在的星云城赶去。
<br/>　　一路无话，两日之后，苏莫到底达了星云城。
<br/>　　星云城坐落在一片高原之上，海拔极高，距漫天星辰较近，所以因此而得名。
<br/>　　这是一座雄城，人口数亿，热闹非凡。
<br/>　　苏莫走进星云城，没过多久便找到了一座沿街的大殿，殿门上书写着‘云顶传送殿，几个恢弘的大字。
<br/>　　这是神武学府背后的十三个势力之一，云顶圣门掌控的传送之地。
<br/>　　嗖！嗖！
<br/>　　苏莫刚刚走到云顶传送殿的门前，大殿内便飞出了两道身影，一男一女，正是聂乾和韩盈。
<br/>　　“苏兄！”
<br/>　　“苏兄！你没事真是太好了！”
<br/>　　两人来到苏莫身前，打量了苏莫一眼，见苏莫没有什么伤势，顿时面露喜色。
<br/>　　他们等了苏莫这么久，本来已经生出不好的预感，现在苏莫完好无损的回来了，他们自然心中欣喜。
<br/>　　“呵呵！让你们久等了！”苏莫笑着说道。
<br/>　　“苏兄别客气，你一人为我们挡住危险，我们还没有谢你呢！”韩盈俏脸真诚，沉声说道。
<br/>　　“谢就不必了，我们是朋友，朋友之间何需言谢！”苏莫摇了摇头。
<br/>　　两人闻言点头，虽然三人相处的时间不长，但也算是共患难了多次。
<br/>　　“对了，苏兄，你是如何甩开那两人的？”聂乾好奇的问道，在他看来，苏莫毫发无伤，定然是甩开了那两名追击之人。
<br/>　　“我带他们到了地狱！”苏莫轻笑一声，随即便向云顶传送大殿中走去。
<br/>　　地狱？
<br/>　　聂乾两人闻言，顿时心中一震，满脸愕然之色。
<br/>　　苏莫，杀了那两名武帝境四重巅峰武者？
<br/>　　这不太可能吧？
<br/>　　两人的眼眸紧紧的凝视着苏莫的背影，随即眼眸一亮，纷纷倒吸了一口凉气。
<br/>　　因为他们现在才发现，苏莫居然突破到了武帝境二重境界！
<br/>　　聂乾和韩盈两人，心潮涌动，震撼不已。
<br/>　　这才多久时间，苏莫居然又突破了修为！
<br/>　　这也太快了吧！
