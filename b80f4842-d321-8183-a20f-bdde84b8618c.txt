　　自山洞中走出，眼前是一片茂密的荒林，这里的树木都异常高大，至少也在百米之上，甚至有些能高近千米！
<br/>　　周围灵气更是无比浓郁，比地球高出百倍都不止！
<br/>　　“果然是这里！”
<br/>　　叶尘不由得感慨了一声，前世的相关记忆，如同潮水般涌现了出来。
<br/>　　记得前世，他刚刚踏入这里的时候，可没少吃苦头。
<br/>　　这天狼星域，地处人族势力的边缘，不但荒凉无比，而且妖兽横行，前世的他，刚出山洞，就险些被妖兽吃掉。
<br/>　　就在叶尘思绪纷飞之时，忽然听到远处传来阵阵躁动之声，
<br/>　　“嗯？前面十里外，有人在打斗？”
<br/>　　神念扫过，叶尘脚下一步迈出，一眨眼的功夫，便已经来到跟前。
<br/>　　只见，
<br/>　　一处湖泊前，是两方人马正在对打。
<br/>　　这些人的修为并不高，最强者的也不过才金丹境初期而已，不过这些人的战斗力，却明显比地球上，同级别的修真者要强大许多，毕竟是修真界，各种道法、武技，乃至对法宝的运用，都不是地球上的修真者可以相比的。
<br/>　　叶尘大致看了一眼，其中一方，明显处于劣势，不但节节败退，而且已经死伤不少，只剩下七八个人在苦苦支撑，但也坚持不了太久。
<br/>　　只是随便看了一眼，叶尘并没有出手的打算，
<br/>　　他可不是那种路见不平，就会拔刀相助的人，更何况，他深知，弱肉强食乃是修真界的不二法则，多管闲事未必会有好报，还会惹不必要的麻烦上身。
<br/>　　就在叶尘准备悄无声息的离开之时，忽然听到那处于劣势的七、八人之中，一名模样秀丽的年轻女子，喝道：
<br/>　　“南黎川！你敢杀我们，我师父玄离真人和流光派，是不会放过你的！！”
<br/>　　“桀桀！区区一个流光派，就算知道是本公子所为，又能如何？难道你们流光派，还敢和我们烈日宗叫板不成？”
<br/>　　说话的是一名神情狂傲的青年男子，显然应该就是之前女子口中的南黎川了。
<br/>　　“云灵儿，本公子劝你还是乖乖从了我，否则待会等我将你擒住，定让你欲仙欲死，桀桀桀~~”
<br/>　　而叶尘听两人对话，脚下顿时一顿，眉头微微一挑，
<br/>　　玄离真人？
<br/>　　流光派？
<br/>　　记得前世，他就在这片丛林中，被一头妖兽攻击，后来多亏一名修真者路过，出手灭掉了那头妖兽，救了他一命，那人便是流光派的玄离真人。
<br/>　　后来，在这位玄离真人的推荐下，叶尘还曾拜入流光派门下，成为流光派的外门弟子，也正因为如此，叶尘方才在人生地不熟的修真界站稳脚跟。
<br/>　　不过可惜的是，他当初在流光派只待了一年不到，后来流光派被烈日宗所灭，叶尘侥幸逃脱，踏上了一条流亡之路，辗转各大星域，之后又得到了一些奇遇，这才一步一步，走上巅峰，成为一代狂帝。
<br/>　　当然了，在修为小有所成之时，叶尘便重返天狼星域，屠了烈日宗满门，为流光派报了灭门之仇。
<br/>　　“说起来，我前世也算是流光派弟子，既然眼前的这些流光派弟子有难，我自然不能坐视不理，而且这一世，也定不能再让悲剧重演！”
<br/>　　“按照时间推算，我这一世进入修真界的时间，应该比上一世早九年左右，一切应该还来得及！”
<br/>　　这些念头在叶尘脑海中闪过的同事，他身体微微一晃，已经冲入人群之中。
<br/>　　尽管叶尘如今的修为，才只恢复了五六成，可是对付一群只有筑基境的修真者，自然不在话下。
<br/>　　砰砰砰！
<br/>　　叶尘出手不留情，随便几掌拍出，围攻流光派众人的十几人，全部被叶尘当场击毙！
<br/>　　“你是谁？竟然敢插手我烈日宗之事？”
<br/>　　南黎川原本都已经胜券在握，甚至都已经开始幻想，将云灵儿这个大美女拿下之后，该如何折磨她了，却没想到半路杀出一个人来，竟然一口气杀了他十几个手下。
<br/>　　要知道，烈日宗在天狼星域，几乎相当于土皇帝一般的存在，还从来没有遇到过这种事情。
<br/>　　面对南黎川的质问，叶尘连理都没理，冲着那几名流光派弟子微微一笑，
<br/>　　“你们是流光派弟子？”
<br/>　　几人一时间也没有反应过来怎么回事，倒是那云灵儿反应最快，当即上前一步，躬身道：
<br/>　　“启禀前辈！我等正是流光派核心弟子，晚辈云灵儿，家师乃是流光派长老玄离真人！”
<br/>　　“哦？”
<br/>　　叶尘听到她自报家门，不由得眉头微微一挑，
<br/>　　在修真界中，各大门派和势力的门下弟子，一般都分为外门弟子、内门弟子和核心弟子。
<br/>　　核心弟子，可以说是门派弟子中，精英中的精英！
<br/>　　像叶尘前世，在流光派，就仅仅只是外门弟子而已。
<br/>　　若是前世遇到云灵儿这些人，非得立即行礼，口称师兄师姐不可。
<br/>　　‘是了，难怪我对这些人没有丝毫印象，看来前世这个时候，烈日宗就已经开始向流光派出手了，杀光一个门派的核心弟子，就等同于毁了他们的未来……’
<br/>　　心里正想着呢，后方忽然想起一道苍老的声音，
<br/>　　“阁下到底是何人？难道真的要为区区一个流光派，得罪我烈日宗吗？”
<br/>　　说话之人，是一名身穿灰袍的阴鸷老人，赫然正是这群人中，唯一的那位金丹境修士。
<br/>　　叶尘冷冷一笑，
<br/>　　“烈日宗？很了不起吗？”
<br/>　　轰！
<br/>　　叶尘这话一出口，不但那老者和南黎川脸色大变，就连云灵儿等人，也露出一抹惊愕之色。
<br/>　　烈日宗乃是天狼星域第一大势力，掌控着整片星域将近一半的地盘，宗内更有元婴老祖坐镇，可以说是生杀予夺。
<br/>　　正如之前南黎川之前所说，即便流云宗知道烈日宗迫害他们门下弟子，只怕也没有勇气和烈日宗宣战。
<br/>　　而如今，眼前这个神秘的年轻人，似乎丝毫没有把烈日宗放在眼里，莫非是来自高级星域的强者？
<br/>　　“阁下到底是什么人？还请见告！”
<br/>　　灰袍老者一时间也不敢轻举妄动，再次沉声开口道。
<br/>　　叶尘瞥了老者一眼，淡淡道：
<br/>　　“我是什么人，你还不够资格知道！十息之内，不滚，就死！”
<br/>(本章完)
