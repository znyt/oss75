　　第399章
<br/>　　李行哉认出了白元飞，猛地一惊，道：“你怎么来了，周围可有人监视。”
<br/>　　“放心，他们都被药倒了，现在睡得和死猪一样沉。”白元飞目光在周围转圈：“有吃的没有，可饿死我了。”
<br/>　　李行哉指了指茶几上一盘点心，道：“那他们明天早晨会不会发现问题？”
<br/>　　不仅周围没有什么动静，便是床上的歌姬此刻都睡得不省人事。
<br/>　　“发现问题又怎么样，反正我已经走了，至于你，看来他们不舍得杀你。”白元飞一边吃着点心一边道：“你可是不知道，这几天我可受了大罪，你如果有天做了皇帝，可别忘了我的功劳。”
<br/>　　李行哉无可奈何的苦笑一声：“你就莫打趣我了，你看我现在这个样子，连这院门都出不去。”
<br/>　　白元飞在屋子里扫了一圈，目光落在那赤身的歌姬身上，她睡熟的样子像个小猫一样，微微打着鼾。女人，真是一种奇怪的生物，哪怕明明知道她们体内藏着危险，可只要她们有一副好面孔，就令人觉得情有可原。
<br/>　　“你最近的日子可是真不错，山珍海味，美女美酒，真是给个神仙都不换呐。”白元飞道。
<br/>　　李行哉长长的叹了口气，他现在连叹气都不能自由，也就在白元飞面前能表露自己的真实情绪。
<br/>　　“你的药好用么，他们会不会中途醒来？”李行哉披上件薄毯。
<br/>　　“好用啊，是山寨的胡人拿出来的，说是叫什么炼金术师，这药叫神仙醉，哪怕闻到一丝丝，连神仙也能放倒，没有专门的解药必须昏睡一天一夜。”
<br/>　　“现在程大雷手下还真是人才济济呐。”李行哉悠悠的叹了口气。
<br/>　　“好了，我也不能在这里多待，迟则生变。凉州城一直没有出兵，大当家就怀疑有了问题，所以特意派我来看看，现在看来，你的问题果然不小。”
<br/>　　“是他派你来的？”李行哉顿了顿：“我便知道，他不可能没有任何反应。”
<br/>　　“你现在什么情况？回去我告诉大当家，大家想办法救你出来。”
<br/>　　李行哉沉默了一阵，白元飞再问一声，他才反应过来。从琴川关离开，李行哉只带了福德勒一人，他在琴川关三年，也有与他交好的朋友，如果李行哉说一声，他们未必不会跟着一起走。
<br/>　　但李行哉还是选择只带福德勒一个，因为他觉得，那些人是程大雷的手下，他不想借助程大雷的力量。
<br/>　　可到最后，自己身陷在此地，还是要依靠程大雷。他本想拒绝白元飞，但仔细想想，没有程大雷，自己怕还真无法离开此地。
<br/>　　于是苦笑一声，对白元飞说明了此间的情况。
<br/>　　白元飞听罢之后点点头：“我知道了，回去之后，大当家必然会想办法来救你，你安心等待便是。这里，他们好像也不会杀你。”
<br/>　　宋伯康当然不会杀李行哉，不仅不会杀，李行哉若有个头疼脑热，他怕是会比任何人都要担心。
<br/>　　白元飞不声不响的离开，大门明明没有上锁，可他还是从窗户跃出，静悄悄的没有发出任何动静。
<br/>　　李行哉苦笑一声，翻身躺下，搂住床上的歌姬，依旧没有半点睡意。
<br/>　　该让白元飞给自己留下些迷药的。
<br/>　　……
<br/>　　长夜漫漫，睡不着的不仅李行哉一个，福德勒同样没有入睡。
<br/>　　他和李行哉分开关押，待遇上自然会有不同，他没有李行哉的歌姬和山珍海味，即便仆人也没有一个。单独被关在院子内，兵器被收走，一日三餐专门有人送过来，此外，对他的监视和李行哉相比，自然松懈很多。
<br/>　　他同样无法走出院子，不同的是，福德勒是个行动派。
<br/>　　他没有李行哉那么多想法，也不会顾及太多，戎族人的思想总是直来直往。无非就是离开此地罢了，从一开始，他就在思考脱身的法子。
<br/>　　宋伯康收走了他的兵器，便以为福德勒失去了战斗力。这样想，委实太小看了福德勒。
<br/>　　在戎族，铁器是奢侈品，只有少数人才能够拥有。对于大多数人而言，石头、木棍，甚至是绳子，想要杀人，什么不能作为兵器。
<br/>　　杀人，从来不是一件困难的事情。
<br/>　　夜里，福德勒从床上睁开眼睛，眼睛很亮，精神状态很好。
<br/>　　他从床底下摸出一根木棒，这是用来顶门的。他拎在手中，踏出了房门。
<br/>　　屋外，繁星满天，但是没有月亮，所以周围还是很黑。
<br/>　　索性，戎族都有在深夜狩猎的习惯，在晚上，他们依旧能保持极好的视力。
<br/>　　福德勒推开了院门，一个站在门口的守卫下意识回过头，诧异的望着他。
<br/>　　不等对方开口发问，福德勒已经扑了过去，木棒重重敲在对方脑袋上。头破血流，对方的头骨直接被敲碎，双眼变得空洞。
<br/>　　福德勒向前跨出一步，托住了对方的身体，避免对方因倒地而发出任声音。
<br/>　　轻飘飘将对方搁在墙角，福德勒拔出了对方腰上的短刀，拎在手中。
<br/>　　他目光看了看周围，此时遇到了和白元飞同样的问题：他也不知道李行哉被关在哪里。
<br/>　　城主府是极大的，对于一个戎族人而言，简直可以用恐怖来形容。他被关在院子里，自然也无法得知城主府的建筑布局。
<br/>　　不过，这对他而言不算太大的问题，在帝国这边，重要的人物，总是会住在最豪华奢侈的建筑内，这一点是永远不会错的。
<br/>　　他站在门口，活动了一下筋骨，骨节发出啪啪的响声。然后他丢下手中的木棒，攥着刀前进，眼睛明亮，像一头在草原上猎食的狼。
<br/>　　当然，福德勒没有白元飞的轻身功夫，不可能静悄悄的行动而不被任何人发现。可论出手干净，杀人果断，白元飞就算是拍马也比不上他。
<br/>　　他隐在黑暗中前行，一旦被人撞见，便干脆利落的杀人，不分男女，不管是巡逻的卫兵，还是起夜的丫鬟，通通丧命在他的刀下。
<br/>　　所过之处，鲜血淋漓，福德勒一步步靠近城主府的中心。
<br/>　　那里住着宋伯康，那才是他的目的。
