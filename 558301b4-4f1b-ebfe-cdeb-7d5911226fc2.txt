　　　　楚风眠一行人，接连不断的走过了无数国度，利用传送法阵，前去北邙学院。
<br/>　　这北域之大，根本不是人可以计量的出的，楚风眠一路上，足足是利用了三四十次传送法阵，逐渐的才来到了临近北邙学院的区域。
<br/>　　要是让楚风眠飞行，只怕是给楚风眠十年时间，他都不一定可以飞的到北邙学院去。
<br/>　　一路之上，楚风眠一行人跨过了无数国度，来到了一处名为满元古城的地方。
<br/>　　这满元古城，不属于任何国度，是一个交通要道，其中传送法阵可以前往其他许多国度。
<br/>　　楚风眠一行人准备要去的下一站，也正是要从这满元古城中转移。
<br/>　　“恩？怎么会有着这么多人？”
<br/>　　楚风眠一行人，刚刚从传送法阵中出来，便是一眼看去，这诺大的满元古城，尽是武者。
<br/>　　看起来，密密麻麻，时有交谈，不知道是因为什么。
<br/>　　“拓跋鼎，你去看看，到底发生了什么。”
<br/>　　楚风眠转过头，看了一眼拓跋鼎，开口道。
<br/>　　“是。”
<br/>　　拓跋鼎过去，询问了一会，才回来，对着楚风眠开口道。
<br/>　　“这满元古城中的传送法阵，好像是坏掉了，如今无法动用，这才人如今在停滞在了这里。”
<br/>　　“传送法阵坏掉了？”
<br/>　　楚风眠听到这个消息，不由眉头一皱道。
<br/>　　这传送法阵，都是上古时代的强者建立的，如今无数岁月过去，倒是许多都年久失修，可能会坏掉。
<br/>　　虽然可以修复，但是的确要耽搁一些时间。
<br/>　　若是对不着急的人来说，倒是无妨，但是楚风眠如今可是要赶时间去前往北邙学院的。
<br/>　　距离北邙学院开宗收徒的日子，楚风眠算了算，只怕也不富裕，更何况楚风眠还必须要先弄到可以参加考核的资格去才行。
<br/>　　若是在这里耽误一阵的话，可能就会错过这次开宗收徒的机会。
<br/>　　北邙学院，每一次开宗收徒，都必须要三年的时间才行，楚风眠可不想白白浪费三年时间。
<br/>　　“这里，距离下一个中转站，有着多远距离？”
<br/>　　楚风眠看着拓跋鼎，出声询问道。
<br/>　　对着北域的地形，他还不是很了解。
<br/>　　“大约有着十万里。”
<br/>　　拓跋鼎沉吟了一下，开口道。
<br/>　　“从这满元古城，到下一个宣州古城，其实并不算远，横渡过去也不需要太久的时间，不过这其中，却是有着不少的流寇，比较棘手，不好对付。”
<br/>　　“流寇？”
<br/>　　楚风眠听到拓跋鼎的话，不由眉头一挑，这流寇，在他看来，也都是些没有什么资质的武者，有些实力，所以才会选择打家劫舍为生。
<br/>　　不过一般流寇的实力，神海境便是顶天了，御风境都极为罕见，对他们来说，可远远造不成什么威胁。
<br/>　　为何在这拓跋鼎的口中，却是极为棘手。
<br/>　　“少主有所不知，这批流寇，其实并非是叛乱盗匪，而是来自与北域的一大宗门，地魔门的弟子。”
<br/>　　拓跋鼎看着楚风眠，缓缓的解释道。
<br/>　　“这地魔门，在北域中势力极大，但是却是魔道宗门，行为做事为人所不耻，所以一般都称呼他们的弟子为流寇。”
<br/>　　“这些地魔门的弟子，实力却是不弱，经常占山为王，拦路劫财，许多阴狠之辈，不止劫财，更是会直接杀人，吞噬武者的精血。”
<br/>　　“这一片区域，正是地魔门的活动范围，所以不好对付，若是贸然横渡，很有可能会遇到地魔门的弟子。”
<br/>　　地魔门。
<br/>　　这个称呼，楚风眠倒是知晓。
<br/>　　这地魔门，其实便是从九魔城中分化出来的魔道宗门之一。
<br/>　　九魔城正名，成为了七大宗门之一，便是不太可以做着太过阴狠的事，所以九魔城中的无数强者，都分离了出去，建立起了无数魔门。
<br/>　　虽然在明面上，双方没有任何关系，但是楚风眠清楚，像是这地魔门，实际上还同属于九魔城之下。
<br/>　　不过这层纸却是没有被人捅破过罢了。
<br/>　　这地魔门，便是九魔城下无数魔门算是中上游的势力，其中门下，不乏有着生死境的强者，倒是的确不好对付。
<br/>　　不过一般来说，生死境的强者，在地魔门中，也都是属于老古董，平日里不会出手，只会去闭关修炼。
<br/>　　所以也是没有那么容易可以有机会碰的上。
<br/>　　“少主，我们是在这里等待几日，还是直接横渡虚空，前往宣州古城？”
<br/>　　拓跋鼎一旁出声询问道。
<br/>　　如今楚风眠是主人，他的一切都要听从楚风眠的话。
<br/>　　拓跋鼎的话，只能是作为参考，真正要决定的人，还是楚风眠。
<br/>　　“以我的速度，横渡虚空，只需要五日时间，便是可以达到宣州古城，要是在这里等待着，很有可能时间便是被拖延，导致时间不够。”
<br/>　　楚风眠听着众人的议论，得知这传送法阵至少还有一个月的时间才可以修好的时候，出声说道。
<br/>　　如今楚风眠在前往北邙学院的路程，还有着接近半个月的时间。
<br/>　　而北邙学院开宗收徒的日子，便是两个月后，现在的楚风眠，要是在这里白白等待一个月的时间，那对楚风眠来说可就时间不够了。
<br/>　　毕竟楚风眠还是要找一下参加考核的资格去才行，与其是在这里等待着，不如直接横渡虚空。
<br/>　　“就算是遇到地魔门的人，也无所谓，只要不是地魔门中的生死境老古董，我都可以轻易对付。”
<br/>　　“更何况才不过五日时间，能不能碰的上，都不一定。”
<br/>　　楚风眠开口道。
<br/>　　等待，可不是他的性格。
<br/>　　“我也赞同。”
<br/>　　寒月漓在一旁，也是开口道。
<br/>　　在这里等待一个月的时间，这传送法阵也不一定可以修缮的好。
<br/>　　还不如直接横渡虚空前去。
<br/>　　听到楚风眠的话，拓跋鼎便是点了点头，去购买了一份地图，来查前往宣州古城的路线了。
<br/>　　楚风眠用灵识来扫视周围，发现这其中的不少武者，也有着跟楚风眠一样的心思。
<br/>　　白白浪费一个月的时间，很有可能就赶不上去参加北邙学院的考核了。
<br/>　　倒是已经有着不少武者，选择出发，去横渡前往宣州古城了。
<br/>　　“走吧。”
<br/>　　不一会，拓跋鼎回来，已经查看好路线，楚风眠一行人便是起身，直接动身，横渡虚空前往宣州古城。
