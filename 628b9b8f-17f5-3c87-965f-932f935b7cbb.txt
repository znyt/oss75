　　第184章
<br/>　　拯救大兵林少羽的题目已经有了，可是提起笔，却无人能够作答。
<br/>　　这次要比救林冲更难，救林冲是敌人不知道他的重要性，而对于林少羽，敌人肯定严加看护。再想故技重施怕是不能，就算真能擒住对方一员大将，对方也未必同意以林少羽交换。
<br/>　　众人出了很多主意，却都没有太大成功的可能。算来算去，似乎只有白刃战生抢的地步。可真若白刃战，己方面对强横的戎族，究竟又有多少胜算。
<br/>　　林少羽被绑在城头下，相聚不过六百步，可这六百步就是难以跨越的距离。每日看着他在太阳下暴晒，看着他在孟子云刀下受虐，对山寨任何一人来说都是极大的折磨。
<br/>　　孟子云！
<br/>　　程大雷牙都快咬破了，若抓住此贼，必定将其剁碎了喂猪。
<br/>　　……
<br/>　　戎族大营。
<br/>　　赤眉的病已经好了，或许说是被金问道刺激出斗志，让他身上的病不药而愈。
<br/>　　今次走在营地内，和以前的感觉自然不同。曾经自己是这里的最高统帅，行走在这里就是巡视自己的领地，而今天却有一种寄人篱下的感觉。
<br/>　　他看到了福德勒。
<br/>　　福德勒比以前憔悴很多，眼像一只饿极的狗。
<br/>　　“福德勒把你的女人送给我吧，我给你一头牛。”
<br/>　　“福德勒让你的女人陪我一夜，拿刀给你换。”
<br/>　　每当有人经过福德勒的帐篷，说出这些话时，换来的只是福德勒凶恶的眼神。
<br/>　　大营里所有人都知道，福德勒拥有七个如花似玉的美人，可却像宝贝一样关在帐篷内，不让任何人碰，便是想偷看一眼都不能。
<br/>　　草原上真正的男人是可以拥有许多女人的，也不像帝国那样假惺惺的有妻妾之分。在他们眼里，女人和牛羊差不多，是可以互相赠送交换的。
<br/>　　『咱哥俩喝得不错，来来来，今夜我让我老婆陪你』
<br/>　　这样的对话在戎族不是没有发生过，甚至可以说是经常发生。
<br/>　　在这样一个环境下，福德勒的所作所为就显得……不够朋友。
<br/>　　木克走到帐篷前，咧咧嘴，露出一个男人都懂的笑容。
<br/>　　“听说你有七个女人，你小小的身板受得了么，让我一个？”
<br/>　　福德勒昂起头，握紧刀向木克小腹扎来。
<br/>　　“嚯，这么野！”
<br/>　　木克是金问道手下大将，他只是没想到福德勒一言不发就出刀，但这样的刀法自然伤不了他。
<br/>　　他立刻就握住福德勒的手腕，将他整个人提了起来。
<br/>　　“我是北蛮王的王子，你以下犯上，该当何罪！”福德勒抬出了自己的身份。
<br/>　　“草原上的男儿只尊重真正的勇士，可不尊重什么王子，一个连刀都握不紧的孩子，算什么男人。”
<br/>　　木克将福德勒放了下来，或者说砸下来，只摔得他鼻青脸肿。
<br/>　　刚爬起来福德勒就又向木克冲过去，却又轻而易举被击倒，然后再冲过去，再被击倒。
<br/>　　最后，木克忽然掐住福德勒的脖子，只卡得他满脸通红。
<br/>　　“不要再来犯我，不是看你的身份，我会直接杀了你。”
<br/>　　嗵！
<br/>　　福德勒被狠狠砸在地上，他看着木克大摇大摆的走出帐篷，而他却再也没有站起来的力气。他以哀求的眼神看向赤眉，至少赤眉还把他当弟弟，在其他王子眼里，根本不存在他这个人。
<br/>　　赤眉背过身去。
<br/>　　福德勒眼底的光芒暗淡下来，他挣扎着想要站起来，可胳膊却传来戳心的刺痛。刚才那重重一摔，令他胳膊脱臼。
<br/>　　他真的已无多少力气，只能听得帐篷内传来女人的尖叫和男人的笑声。
<br/>　　大家自然知道帐篷内发生着什么，有人嘀嘀咕咕脸上出现猥琐的笑容。
<br/>　　『我是福德勒，我是你们的主人』
<br/>　　『记住我的名字，从今日起我将保护你们』
<br/>　　『有福德勒在，没人可以把你们抢走，也没人可以伤害你们』
<br/>　　多么掷地有声的豪言壮语，又多么幼稚。却原来，自己什么都守护不住，只能像条死狗一样倒在地上，听着那讨厌的笑声。
<br/>　　多可怜吧。
<br/>　　良久，木克从帐篷内出来，脸上挂着心满意足的笑容。他看了地上的福德勒一眼，笑道：“手太重，杀了一个，不过我想你也不会在意。”
<br/>　　福德勒没有反应，这一刻他的灵魂像是被一枪捅透。
<br/>　　木克路过赤眉身边：“九王子，唤你去一趟。”
<br/>　　唤……还真是个意味深长的字眼呐。赤眉冷笑一声，踏进金问道的大帐。
<br/>　　“你的病好了？”金问道。
<br/>　　“本就无什么大碍，睡上一觉也就好了。”
<br/>　　“没有大碍……意思是，你可以上阵了？”金问道顿了顿：“你可愿为先锋？”
<br/>　　“要攻城？”
<br/>　　金问道点点头：“对方不投降，我也没有办法，只好强攻了。”
<br/>　　“有多大把握，对方可不简单。”
<br/>　　“不简单却也要看对谁而言，对弟弟你来说，自然是不简单的。对我来说么，说了三日破城，就是三日破城。”金问道看着赤眉的眼睛：“差一天，差一个时辰，差一刻钟都不算数。”
<br/>　　“何日攻城？”
<br/>　　“明日午时。”
<br/>　　大战之前，戎族磨刀霍霍，而蛤蟆寨内，程大雷自然也不会闲着。
<br/>　　但在这一天夜里，戎族大营内发生了一件小事。
<br/>　　福德勒提着刀，走进自己的帐篷，借着月光，他可以看到六个挤在一起的女人，还有一个……现在变成尸体倒在地上，散发着令人作呕的血腥味。
<br/>　　死的人已经死了，活着的人眼中没有任何神采，只有逆来顺受的麻木和深入骨髓的恐惧。
<br/>　　福德勒冲着他们深深鞠了一躬。
<br/>　　“对不起。”
<br/>　　女人们不知道他在做什么，但通过这些日子的相处，她们知道这个男人，或者说少年不像其他戎族一样吓人。
<br/>　　“你别难过……”
<br/>　　似乎某种女人骨髓里的母性发挥了作用，一个女人伸出手想抚摸他的脸。
<br/>　　福德勒抬起刀扎入她的胸膛，鲜红的血涌出来。
<br/>　　他每杀一人，便道一声对不起，鲜血和女人的惨叫都不能让他脸上的表情有任何动容。他只是干净利落的杀人，然后干净利落的道歉。
<br/>　　有时候，你若没有力气保护你想保护的，是否毁掉它，也是选择的一种。
<br/>
