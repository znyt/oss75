<br/>    杏花连忙摆手道：“不用了，不用了，子鱼先生次次都帮我挑水，村里会有闲言碎语的。”

<br/>        子鱼笑道：“我是个道士啊。”
<br/>    
<br/>        杏花看向子鱼，道：“寡居的俏寡妇，年轻的俏道士，怕是会传出一些闲言碎语。我倒是没什么，我的命里福薄，刚刚出嫁，新婚之夜便克死了丈夫。只是，若是对子鱼先生说三道四，总归是不好的。”

<br/>        子鱼没有继续说下去，只是默默地将水桶挑在了肩膀上，开始朝着水井走去。

<br/>        一个时辰之后，子鱼终于是挑满了整个水缸，水缸满满的，额角也是有着汗水滴落。

<br/>        杏花拿出自己的手帕，轻轻给子鱼擦汗。
<br/>    
<br/>        子鱼也是觉得没什么不妥，待得杏花擦完汗之后，就直接拱手道：“天色已晚，小道，就先告辞了。”
<br/>    
<br/>        杏花道：“你挑了一缸的水，喝杯水再走吧。”
<br/>    
<br/>        子鱼却是直接离开了院子，“该走了，回去晚了，师父会见怪的。”

<br/>        子鱼走到回道观的路上，他并没有第一时间禀报师父无字劫降临了的消息，原因很简单，这个消息不重要，比起教小孩子读书，给寡妇挑水，这件事并没有那么重要。
<br/>    
<br/>        待得回到了道观之中，他将给师父准备好的饭菜都做好，才去叫师父过来吃饭。、
<br/>    
<br/>        师父是一个颇为年老的老道士，道观之中自然只有道士。
<br/>    
<br/>        老道士和子鱼一起吃饭，这个时候，子鱼才道：“师父，今天有一个戴着面具的人来了，说是无字劫降临了。”
<br/>    
<br/>        老道士思考了片刻，然后道：“你今天去给杏花那个寡妇挑了水吧？”
<br/>    
<br/>        子鱼拱手道：“是的。”
<br/>    
<br/>        老道士道：“很好，这说明，你的心并没有乱。你应该知道我们与世隔绝的原因。”
<br/>    
<br/>        子鱼道：“与世隔绝，才能至诚于道。大师兄，便是因为无法忍受住这寂寞，所以去了星空中。”
<br/>    
<br/>        老道士道：“今天的菜，不太合胃口，明天，记得去要一尾鱼回来。”
<br/>    
<br/>        子鱼拱手道：“是，师父。”
<br/>    
<br/>        日子依旧是照常下去，第二天，子鱼带了一尾鱼回来，做了鱼羹
<br/>    
<br/>        ，师父吃的很开心。
<br/>    
<br/>        师徒再也没有去议论关于无字劫的只言片语，谁都没有提起，谁都没有再说。
<br/>    
<br/>        时间就这样缓缓地过去，直到三年之后，师徒俩再次坐在道观中，两人对坐，老道士道：“明天，你去看看那个渡劫的修行者。”
<br/>    
<br/>        子鱼道：“他有什么好看的？”
<br/>    
<br/>        老道士道：“总归是要去看看的，有人觊觎天道的权柄，而我们，侍奉天道。”
<br/>    
<br/>        子鱼道：“那我想好好看看。”
<br/>    
<br/>        老道士起身朝着苍天一拜，“天道不灭，大道永存。”
<br/>    
<br/>        子鱼也是学着老道士的样子，认真道：“天道不灭，大道永存。”
<br/>    
<br/>        年轻的道士一身轻松的下了山，老道士关了门。
<br/>    
<br/>        一个月后，燕山分部。
<br/>    
<br/>        卓青瑶正在八尺烬前悟剑，这个时候，山下忽然有些吵杂。卓青瑶道：“银翼，过来。”
<br/>    
<br/>        银翼很快来到了卓青瑶所在的地方，他问道：“大宗主，有什么事吗？”
<br/>    
<br/>        卓青瑶问道：“山下到底是什么事，这么吵？”
<br/>    
<br/>        银翼道：“在山下，有着一个道士，是个凡人，但是却是想要我们带他去神山之上。”
<br/>    
<br/>        卓青瑶微微皱眉，但是忽然似乎又是想到了什么，然后道：“你去把那个道士带上来。”
<br/>    
<br/>        不过一会儿，有陌生人的声音在卓青瑶的背后响起，“小道子鱼，见过燕山大宗主。”

<br/>        卓青瑶回头，果然是一个凡人道士。
<br/>    
<br/>        卓青瑶有些好奇地问：“你一个凡人，去神山做什么，那里可是有着很多的仙人。你去神山，很危险的。”

<br/>        子鱼笑道：“那上面，有着很少有人能够看到的风景，无字劫，应该是一种很难看到的风景，那里有着风景，所以，我想去看看。”

<br/>        卓青瑶道：“我有一个问题想要问你。”

<br/>        子鱼道：“大宗主不应该问我这一个凡人，修行的世界，我不懂。大宗主也不应该问我这个外人，这燕山之上，应该都是大宗主自己的人，亲近的人，他们会给大宗主更好的答案。”

<br/>        卓青瑶道：“我
<br/>    
<br/>        并不是很喜欢更好的答案。而且，正因为你是一个外人，当局者迷，旁观者清，所以，你看的更清楚，正因为你是一个凡人，你不懂，所以我才问你。”
