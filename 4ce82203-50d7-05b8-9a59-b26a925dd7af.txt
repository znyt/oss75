        他只是同情筱颖，并不是真正的爱筱颖。
<br/>
<br/>        季思妍心里一阵欣喜，眉眼红唇，染上一丝微醺的笑意，她伸手勾住了傅昼景的脖子，温柔地说道：“以后，我会和筱颖和睦相处。”
<br/>
<br/>        她的想法很简单，只要把他哄开心了，比她现在做任何事情都重要。
<br/>
<br/>        傅昼景现在是她所有的赌注，她的付出，只为了赌她自己的未来。
<br/>
<br/>        季思妍闭上了眼睛，想让自己的思绪彻底放飞。
<br/>
<br/>        为了他，她其实什么都愿意去做！
<br/>
<br/>        就连起初她在傅昼景和傅夜沉之间徘徊，最终，她还是选择了傅昼景。
<br/>
<br/>        傅昼景闭上眼睛的那一刻，脑海里想的全是童筱颖的身影。
<br/>
<br/>        筱颖……
<br/>
<br/>        他心心念念的筱颖……
<br/>
<br/>        筱颖的一颦一笑，筱颖那温香玉软的身体。
<br/>
<br/>        可是，渐渐地……
<br/>
<br/>        他与筱颖十指相扣，发誓要白首不相离。
<br/>
<br/>        可最后，筱颖却松开了他的手，对他说：“我们都回不去了……”
<br/>
<br/>        筱颖，我们真的已经回不去了吗？
<br/>
<br/>        傅昼景回过神来时，看到眼前的女人是季思妍，看着她妖娆妩媚的样子，眼眸里渐渐深邃幽黑了几分。
<br/>
<br/>        思妍变了……
<br/>
<br/>        她已经不再是以前那个单纯地只是喜欢着他的季思妍了。
<br/>
<br/>        更确切地来说，应该是她已经不再是他曾经喜欢的那个样子了。
<br/>
<br/>        如果不是这段时间的相处，或许，思妍永远是他心里那抹值得去回忆的白月光，因为，曾经，她把最美好的一面全都留给了他。
<br/>
<br/>        但现在，他心知肚明，她在打着爱他的幌子，利用他窥视傅氏集团。
<br/>
<br/>        毕竟，在滨江城里，只有季氏集团跟他们傅氏集团是死对头。
<br/>
<br/>        傅氏集团想要入手的项目，他们季氏集团总会插手并从中作梗。
<br/>
<br/>        季家有野心，但是，他傅昼景更有野心，季家想吞并傅氏集团，而他傅昼景，想鹬蚌相争渔翁得利。
<br/>
<br/>        傅昼景本就有野心的种子，只不过，季思妍是那催生剂，让他心底的那颗种子急速膨胀发芽。
<br/>
<br/>        两人的欢爱刚结束，傅昼景的手机不逢其时地响了起来。
<br/>
<br/>        季思妍下意识地偷偷瞄了一眼傅昼景那放在床头柜上的手机荧幕，“筱颖”这两个字眼，一如既往地深深地扎痛着她的心。
<br/>
<br/>        傅昼景淡定自若地拿起手机接听了电话。
<br/>
<br/>        “昼景，我今天在总部这边第一天上班。大家都待我不错！”童筱颖的声音，清甜里透着笑意，听起来能感觉到她的心情似乎不错。
<br/>
<br/>        “嗯，我有跟大哥说，让她多多关照你。你在总部，不会再遇到之前在地利置业的设计部所发生的那种事情了。”傅昼景薄唇抿了抿，情不自禁地勾起一抹浅浅地弧度。
<br/>
<br/>        他的一举一动，季思妍都看在眼里，痛在心里。
<br/>
<br/>        “嗯，大家都对我很好。不过，我今天又给大哥下厨了！大哥说他饿了，没吃午饭，又不喜欢吃外面的饭菜，非得让我给他做。”童筱颖一边抱怨着，一边跟傅昼景发起了牢骚。
<br/>
<br/>        傅昼景忍俊不禁地安慰道：“大哥还是那倔强的性子啊！算了！看在他在公司里有吩咐属下关照你的份上，我不跟他计较了！”
<br/>
<br/>        “昼景，你什么时候回来啊？我……我想你了！”童筱颖声音弱了些，却依旧甜得像抹了蜜糖一般，让人听着非常舒服。
<br/>
<br/>        傅昼景记得，筱颖唱歌的声音也很好听，只是没想到，在电话里，她娇羞的声音如此令人迷醉。
<br/>
<br/>        坐在一旁的季思妍，两眼看得发直。
<br/>
<br/>        从什么时候起，他已经毫不避讳她，甚至无所顾忌地当着她的面，和那个“筱颖”打情骂俏了？！
<br/>
<br/>        或许，是她自己纵容他的吧！
<br/>
<br/>        她说过的，愿意当他的地下情人，愿意接受筱颖，愿意包容他已婚。
<br/>
<br/>        季思妍以为，自己可以放宽心态接受这一切，但终究，她不是古时候的女人，无法容忍自己喜欢的男人还跟别的女人在一起。
<br/>
<br/>        “我忙完就会回来，先挂了。”傅昼景三言两语，结束了这通电话。
<br/>
<br/>        他的余光里，季思妍的脸色变了，所以，他已经意识到季思妍的心思或许起了一丝异样。
<br/>
<br/>        “思妍，如果……你无法接受我已婚的事实，我们现在就结束这种关系。”傅昼景云淡风轻地说。
<br/>
<br/>        季思妍回过神来，挪动着身子，依偎在了傅昼景的怀中，喃喃细语：“阿景，我不是圣人……爱你，所以才会有私心。希望你只属于我一个人。但是……我会接受筱颖的。阿景，你放心。我的存在，我不会让筱颖知道。既然阿景可怜她，那么我也会可怜她。把她当自己的亲妹妹看待。”
<br/>
<br/>        “阿嚏——”
<br/>
<br/>        童筱颖一个喷嚏，揉了揉鼻子，站在傅氏集团大厦的正门口，仰望着灰蒙蒙的天空。
<br/>
<br/>        这是要下雨了吗？
<br/>
<br/>        她刚下班，就给傅昼景打了通电话，哪怕她抱着最激动的心情想跟他分享自己的喜悦，而那边，昼景总是对她很冷淡的样子。
<br/>
<br/>        童筱颖有些沮丧地低着头，而这个时候，天空也开始下起绵绵细雨，好像上帝也在为她的忧伤而掉下了同情的眼泪。
<br/>
<br/>        “上车！”
<br/>
<br/>        就在童筱颖黯然神伤的时候，一辆黑色迈巴赫豪车的后座车门突然打开。
<br/>
<br/>        “大哥？”
<br/>
<br/>        童筱颖微微弯腰，看了一眼车内向她发出命令的人。
<br/>
<br/>        她记得，大哥的座驾是一辆霸气的越野车来着……
<br/>
<br/>        大哥这是换车了吗？！
<br/>
<br/>        “是我，上车。”傅夜沉淡淡地说。
<br/>
<br/>        童筱颖看清楚了人后，这才钻进车内，并关上了车门，坐在了傅夜沉的身旁。
<br/>
<br/>        司机踩了油门，流线型的车身平稳地往前滑动。
<br/>
<br/>        宽敞的车内，后座上的两个人没有任何的交谈，彼此之间的气氛显得有些沉闷。
<br/>
<br/>        童筱颖将手放在坐垫上时，傅夜沉也不约而同地将手放了下来，两人的指尖，在不经意间地碰到了一起。
<br/>
<br/>        恍若被触电了一般，童筱颖正襟危坐，与傅夜沉保持着一定的距离，连忙收回自己的手，双手交叠放在膝盖上，轻轻地抓着裙摆，身心都有点儿不自在。
<br/>
<br/>        她用余光偷偷地瞄了他一眼。
<br/>
<br/>        傅夜沉那俊逸脱俗的容颜，神情一片冷淡，精致的薄唇紧抿，喜怒不形于色，令人难以琢磨。
<br/>
<br/>        童筱颖隐隐地感觉到，一股清冽的气息幽寂地从他身上慢慢地向四周散发。
<br/>
<br/>        他就像一位高贵而矜持的王，静默地坐在那高高在上的皇位上。
<br/>
<br/>        雨断断续续，一直下到了天黑。
<br/>
<br/>        豪车驶进了傅夜沉的私宅。
<br/>
<br/>        傅夜沉先下的车，待到她推开车门从车内钻出来的时候，头顶多了一把深蓝色的大伞。
<br/>
<br/>        童筱颖抬头看了一眼这把大伞，视线又顺着这把大伞往下落去。
<br/>
<br/>        傅夜沉那俊美非凡的脸庞上，清冷的眉眼、淡色的唇瓣，一丁点的表情都没有。
<br/>
<br/>        如此一张帅得无可救药的脸，衬得他周身的环境，都跟着宛如幻境一般，惊艳的让人窒息。
<br/>
<br/>        他在她的面前，不管做任何事情，总是一副波澜不惊的样子，却显得她仓皇窘迫。
<br/>
<br/>        童筱颖不好意思让傅夜沉给自己撑伞，于是抬起手来，将手里的包包放在了头顶，冲上了台阶。
<br/>
<br/>        此时，大门刚好打开，苏漫雪手里拿着伞从门内走了出来，见童筱颖是淋雨上来的，连忙迎上前去给她撑伞。
<br/>
<br/>        “筱颖，小心淋了雨会要感冒的！”苏漫雪温柔体贴地说。
<br/>
<br/>        她这话，让童筱颖产生了一种恍若回到了以前两人友好地相处在一起时的错觉。
<br/>
<br/>        傅夜沉从容淡定地走在了后头，一步一步沉稳优雅地走上台阶。
