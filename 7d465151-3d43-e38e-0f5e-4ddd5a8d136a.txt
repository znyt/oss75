　　此时，傅同有些后悔了，后悔不应该挑战苏莫，而是应该等日后有人挑战之后，他摸清了苏莫的实力，再行挑战。
<br/>　　但是现在，后悔也已经晚了。
<br/>　　脸色变换了一阵，傅同咬了咬牙，随即手臂一挥，一小堆纯白色的晶石向苏莫飞了过去，正是中品灵晶。
<br/>　　所有观战之人，无不是感慨不已，傅同败的不冤，实力差距太大了。
<br/>　　众人望着苏莫，心中暗叹，不愧是七大势力会武第一，不愧是能横扫六大势力妖孽的人物，直接破例被提拔为真传弟子，的确是有这个资格。
<br/>　　苏莫望着飞来的灵晶，微微一笑，一挥袖袍，便将这些中品灵晶收了起来，同时他心中不禁感慨，这灵晶太好赚了！
<br/>　　若是能不停的有人挑战他，不断的给他送灵晶，他还去做什么任务啊！
<br/>　　这赚取灵晶的速度，可是比什么任务都要快啊！
<br/>　　随即，苏莫略作沉吟，便转头看向那些跟来的观战之人，朗声问道：“还有哪位想要挑战我的人吗？赶紧都来挑战吧！三百块中品灵晶的赌注，我来者不拒！”
<br/>　　苏莫又将赌注加大了，声音洪亮，传遍全场，让得所有人都是一怔。
<br/>　　随即，众人不禁无语，卧槽，这苏莫居然是想要靠挑战来赚取灵晶。
<br/>　　别的一些真传弟子，都担心被人挑战，从而失去真传弟子的身份，这苏莫倒好，居然还主动要求别人挑战自己。
<br/>　　“有人要挑战我吗？三百块中品灵晶，只要打败了我，不仅能得到三百块中品灵晶，还能得到真传弟子的身份！”苏莫见没有人出手，便再次喊了起来。
<br/>　　众人听了苏莫之言，面色各异，有的是眉头紧皱，不喜苏莫这种嚣张的姿态，有的是感到好笑，觉得苏莫真是奇葩；但更多的人是哭笑不得。
<br/>　　这特么也太让人无语了吧，苏莫怎么像外面的那些小商贩一样，把挑战变的像是售卖商品一般。
<br/>　　苏莫等了片刻，见没有人挑战，心中不禁暗叹。
<br/>　　略一沉吟，他便再次开口道：“我在星空中开个盘口，三日之内，谁都可以前来挑战，灵晶和真传弟子的身份，希望大家不要错过，若是三日内无人挑战，那我就要出去历练了，很久都不会回来，你们想要挑战都没有机会了！”
<br/>　　言罢，苏莫身形冲天而起，再次向星空中飞了上去。
<br/>　　他之所以要如此，也是想要再赚取一些灵晶，通过与傅同一战，他对自己的实力更加的自信了。
<br/>　　据说真传弟子，大多都是武尊境二重、三重的弟子，所以那些觊觎真传弟子之位的武尊境弟子，估计大多都是武帝境一重或者二重。
<br/>　　所以，这些人他还是有把握对付的。
<br/>　　再说了，若是真的有强人能打败他，夺取了他的真传弟子身份，那也无妨。
<br/>　　反正他灵晶赚到了，以他的成长速度，很快就能再夺回来。
<br/>　　眨眼之间，苏莫便冲上了九霄，再次的飞上了星空。
<br/>　　众人望着苏莫离去的背影，不少人都脸皮抽搐，这特么还开盘口了？
<br/>　　少倾之后，有很多人也再次飞上了星空，或是要观战，或是有挑战之心。
<br/>　　但也有一部分修为略低之人，纷纷的向问道山飞去，要将这个消息传回宗门，让那些强大的武尊境弟子，前来挑战苏莫。
<br/>　　“什么？苏莫在星空中开盘口了？”
<br/>　　“三百中品灵晶的赌注？”
<br/>　　“打败了苏莫就能得到三百中品灵晶，以及取代苏莫的真传弟子身份！”
<br/>　　“这苏莫真是狂啊！他哪来的底气？”
<br/>　　随着众多的弟子将消息传回道宗，整个道宗再次的沸腾了起来，大量的弟子成群结队，向星空之上飞去。
<br/>　　就连很多长老人物，都被惊动了，实在是这个动静闹得太大了。
<br/>　　“我天，这小子又要搞事情？”冯光听闻此事之后，二话没说，立刻放弃手头之事，向星空之上飞去。
<br/>　　整个问道山上，能看到大量的弟子络绎不绝的飞向天穹，如同天河倒卷。
<br/>　　山巅之上，一身道袍的真贤道宗，抬头仰望星空，深邃的目光仿佛洞穿了无尽的虚空。
<br/>　　良久之后，他无奈的摇了摇头，神情显得颇为无语。
<br/>　　……
<br/>　　太阴星外，星空之中，苏莫盘膝而坐，他的周围不远处，已经聚拢了大量的道宗弟子和长老。
<br/>　　他在等待，他相信肯定有人会挑战他。
<br/>　　因为他心中清楚，很多武尊境的弟子，都想要成为真传弟子。
<br/>　　所以，一定会有人挑战他。
<br/>　　时间流逝，周围汇聚的人越来越多，嘈杂的声音此起彼伏。
<br/>　　不过，依旧没有人挑战苏莫，不是别人不敢，而是不想做出头鸟。
<br/>　　因为就算战胜了苏莫之后，可能要应对其他人的挑战。
<br/>　　足足等待了一个多时辰，也没有人愿意挑战，这让苏莫有些无奈了起来。
<br/>　　“无人来战吗？”
<br/>　　苏莫站了起来，眸光扫视四方，朗声大喝：“真传弟子的身份，你们不想要吗？”
<br/>　　众人闻言，不少人眼神闪烁，不知心中所想。
<br/>　　苏莫继续道：“若你们连一战的勇气都没有，还做什么真传弟子，哪有资格成为真传弟子！”
<br/>　　郎朗的声音传遍四方，让得一些有挑战之心的人，脸色沉了下来，这苏莫区区武帝境九重的修为，居然如此有恃无恐，真当自己在武尊境之中也能无敌吗？
<br/>　　“我来败你！”
<br/>　　就在此刻，一道暴喝之声响起，只见下方太阴星上，一道人影急速而来。
<br/>　　此人是一位青袍青年，面庞枯瘦，长发飘舞，整个人给人一种生命力极为旺盛之感。

<br/>　　“是贾延！”
<br/>　　“贾延挑战苏莫了，看苏莫还怎么嚣张！”
<br/>　　“贾延可是武尊境二重的修为，实力非常的强！”
<br/>　　众人见到这青袍青年，顿时有些动容，终于有强者要挑战苏莫了，还是武尊境二重的修为。
<br/>　　武尊境二重对武帝境九重，这一次，苏莫不可能再胜了吧！
<br/>　　苏莫打量着眼前的青年，面上没有丝毫的惧色，微笑道：“三百中品灵晶的赌注，敢否？”

<br/>　　贾延闻言，嘴角勾起一丝笑意，摇了摇头，道：“你以为是孩童玩闹吗？三百中品灵晶未免太无趣了，五百倒是还说的过去！”
