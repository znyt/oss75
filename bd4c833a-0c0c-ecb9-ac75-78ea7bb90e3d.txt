　　任何事情的发展都是有逻辑性的，举个简单的例子：孩子调皮打破了花瓶，那么结果有两个，一是被大人发现，然后揍一顿；二是自己去坦白，然后被训斥。
<br/>　　这个例子很简单，看到了开头就可以根据事情的发展推断出后面将会发生什么。
<br/>　　但是当事情复杂到一定程度，可以影响结局的因素太多时，逻辑推导就变成了一件很麻烦的事情——比如崔钰的事情。
<br/>　　站在一个正常人的角度，李承乾无论如何都理解不了为什么会地府这样的东西存在，而且在大唐竟然从来都没有人听说过地府真实存在的例子，就连袁守诚、袁天罡、李淳风这样的道门老神棍也都没有听说过。
<br/>　　那些和尚也好，道士也罢虽然把道理说的条理清晰，但是如果真的说起地府、阴间之类的东西就会发现，他们说的更多只是一种理论，而不是实际存在的东西。
<br/>　　在后世的现代社会，人的灵魂是否存在一直都是人们争论不休的问题，在大唐，这个问题不用争论，所有人都相信人有灵魂存在，但李承乾却还是不断的在纠结着。
<br/>　　地府存不存在？阴间存不存在？崔钰到底能不能穿梭阴阳两界？说起来这些东西像是玄幻小说，可实际上他却亲身经历过。
<br/>　　说不信，他说服不了自己；说信，后世二十三年的无神论教育让他怎么也接受不了这个世界有鬼、有神仙这样的事实。
<br/>　　一只精致的小手在眼前掠过，沉思中的李承乾陡然一惊，抬头一看发现是长乐和林晓晓两个。
<br/>　　两个大小姑娘正饶有兴致的看着他，尤其是长乐，脸上更是带着狡黠的笑容，一看就是没打什么好主意。
<br/>　　“你们两个怎么来了？”收敛收神，李承乾若无其事的问道。
<br/>　　“哥，你在想什么？那么入神，我们叫你好几声都不答应，是不是在想那个美女啊？”长乐从李承乾身边的矮几上拿起一颗蜜饯塞进嘴里，脸上挂着顽皮的笑意。
<br/>　　“屁的美女。”李承乾翻了个白眼，看了长乐和林晓晓一眼：“满皇宫除了母后，谁有你们两个漂亮？”
<br/>　　“哥，你可是越来越会说话了。”听了李承乾虚伪的赞扬，长乐皱起可爱的鼻子说道。
<br/>　　“少说废话。”李承乾打开长乐过来羞他脸的手，脸色突然一正说道：“哎，你说如果我认识阴曹地府的判官……会怎么样？”
<br/>　　“判官？”长乐愣了一下，有些迷惑的看着李承乾，良久之后才说道：“哥，你这个牛吹的比以前可有新意多了。”
<br/>　　吹牛？长乐的评论让李承乾整个人都不好了，敢情以前说的那些长乐这小丫头片子都当自己是在吹牛？
<br/>　　不过很明显，长乐并没有发现自己话中的漏洞，而且她的注意力也被判官两个字吸引了过去，兴致勃勃的扯着李承乾说道：“哥，都说地府判官能掌人生死，你要是认识判官那可真就厉害了，以后看谁不老实，砍头都能砍两次呢。”
<br/>　　“砍头砍两次？”李承乾一时没有反应过来。
<br/>　　“对啊。阳间砍一次，到了阴间再砍一次。”长乐漂亮的大眼睛熠熠生辉。
<br/>　　“啊？！”李承乾彻底懵了，这还是以前的那个乖宝宝长乐么？怎么动不动就砍头啊？
<br/>　　很明显，李承乾的潜意识还是被后世的一些东西影响的，根本没有意识到他的这些个弟弟、妹妹性格就算再好，也改变不了身为皇子、皇女的事实，‘吃人’的本质已经深入他（她）们的骨髓。
<br/>　　“对了哥，你认识的判官叫啥？哪天带我也见见好不好？”找到了新的吹牛方向，长乐的状态明显有些嗨，根本没有注意到李承乾状态。
<br/>　　“啪”的一声，长乐的额头被李承乾轻轻弹了一下：“小丫头片子，判官是那么好见的？见了判官知道有啥后果不？”
<br/>　　“你不是认识判官么？见见有什么大不了。”长乐摸着自己的额头有些委屈。
<br/>　　“不行，判官可不是想见就见的，而且地府里面那么多妖魔鬼怪，你就不害怕？到了那里万一不小心被抓走了怎么办？”李承乾吓唬长乐。
<br/>　　没办法，这小丫头片子仗着被老头子宠爱，在宫里越来越无法无天，虽然不折腾那些宫女、内侍，但是李承乾兄弟几个却都没有逃过她的魔掌，时不时就被她坑上一回。
<br/>　　所以崔玉的事情，李承乾打死都不会告诉长乐，否则天知道她会干出什么事儿，说不定会让老崔带着她来个地府七天乐也没准儿。
<br/>　　“哥，你，你又吓唬我。”果然，提到妖魔鬼怪，长乐不由自主的缩了缩脖子，有些恼怒的瞪了李承乾一眼，然后转身拉过在一边给某太子喂蜜饯的林晓晓：“晓晓，我们走，去母后那里告状去。”
<br/>　　“去吧去吧，快点去吧!”李承乾挥手催促道。
<br/>　　长乐这小丫头片子，总是会拿长孙皇后来威胁李承乾，时间长了某太子已经品出了味道，也想出了应对的法子——欲擒故纵。
<br/>　　“算了，不去了！”已经拖着林晓晓走到门口的长乐看到李承乾没有反应不说，反而让她快走，也觉得有些颓丧。
<br/>　　拿兄妹两个互相‘吹牛’的事情去告状，长乐认为长孙皇后应该不会把李承乾怎么样，弄不好还会被老妈说成无理取闹。
<br/>　　“不去了就回来好好坐着，有点大姑娘的样子，都十三了还那么跳，被母后知道了一定又会罚你。”李承乾见招式奏效，脸上挂起胜利者的微笑。
<br/>　　“哼，那你去告状吧！”长乐有些不满李承乾‘小人得志’的表现，嘟起小嘴拉着林晓晓头也不回的出去了，看样子应该被李承乾气的不轻。
<br/>　　“这丫头……”看着长乐远去的背影，李承乾苦笑着摇了摇头，暗叹是不是老头子和自己把这她给宠坏了。
<br/>　　不过很快，他的念头就从长乐的身上转回到崔钰，‘狼狈为奸’四个字从脑海中闪过，作为大唐唯一知道崔钰秘密的人，李承乾认为长乐那小丫头说的不错，他和老崔之间如果合作一下……。
<br/>(本章完)
