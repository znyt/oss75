夏夕绾今天用蝴蝶头绳低低的扎了马尾，小圆圆抓到了她的头绳，她往后一退，乌黑的长发就如瀑布般散落开，在半空中划出潋滟的弧线。
<br/>
<br/>        这时身后突然传来了一道柔媚的嗓音，“绾绾。”
<br/>
<br/>        夏夕绾回头，叶翎来了。
<br/>
<br/>        今天叶翎穿了一件黑色宽大的T桖裙，裙摆落在她的膝盖上方，尤物般的曼妙曲线和慵懒软媚的气质将一件宽大黑t都衬出一股高级感，海藻般的秀发随意的披散着，唇红齿白的一张小脸端的明艳动人，如刚绽放的红玫瑰。
<br/>
<br/>        “绾绾，过来。”
<br/>
<br/>        夏夕绾迅速走了过去，“翎翎，你来了？”
<br/>
<br/>        叶翎将夏夕绾拉到了自己的身边，然后挑着一双柳叶眉看向前方的陆寒霆，“陆总，这位是我的闺蜜，她没有得罪你吧？”
<br/>
<br/>        陆寒霆看了一眼夏夕绾，“没有。”
<br/>
<br/>        “那就好，绾绾，这位可是帝都的商界巨子陆总，你以后看见他要绕道走，如果将陆总得罪了，我们在帝都就生存不下去了。”叶翎漫不经心的勾起红唇笑道。
<br/>
<br/>        夏夕绾点头，“知道了。”
<br/>
<br/>        “那陆总，我们先走了。”叶翎拉着夏夕绾的小手带她离开。
<br/>
<br/>        两个人没有再看他，而是聊天，“绾绾，你退了跟孙进的婚约了？”
<br/>
<br/>        “对啊。”
<br/>
<br/>        “早点退了好，一家女百家求，好男人多得是，随便你慢慢挑。”
<br/>
<br/>        陆寒霆听不到她们的谈话声了，她们走到了一个吧台上，点了两杯鸡尾酒，一个清纯纤尘的能掐出水来，一个明艳灼华如夏日玫瑰，两个人往那里一坐，很快就吸引了所有人的目光。
<br/>
<br/>        这时一个侍者将两杯鸡尾酒送了过来，“两位小姐，这鸡尾酒是那边的先生送给你们的。”
<br/>
<br/>        已经有人迫不及待的给她们送酒了。
<br/>
<br/>        陆寒霆没有离开，而是站在那里看着，这时耳畔就传来了顾夜瑾低醇的嗓音，“怎么，看上人家了？”
<br/>
<br/>        陆寒霆侧眸，顾夜瑾已经出来了，现在慵懒的抵靠在角落的墙壁上看着他。
<br/>
<br/>        陆寒霆蹙了一下英气的剑眉，“你跟叶翎怎么了，她话里有刺都是针对我的意思，她不会将你的错牵罪到我的头上了吧？”
<br/>
<br/>        “少来，这个锅我可不背，叶翎就是对你有意见，最近也不搭理我，开口就是我跟你一起玩的，你这样我也好不到哪里去。”顾夜瑾给了陆寒霆一眼，那意思是明明是你牵连了我。
<br/>
<br/>        陆寒霆可不知道他哪里得罪叶翎了，叶翎就是一朵带刺的玫瑰，他深邃的狭眸又落在了夏夕绾纤柔的身影上，“她，跟叶翎是闺蜜？”
<br/>
<br/>        顾夜瑾清冷的黑眸看了一眼夏夕绾，兜兜转转了一圈，哪怕彼此已是陌生人，但是陆寒霆的目光还是一眼就被她给吸引。
<br/>
<br/>        “恩，听说她是夏家的小女儿，那应该没错了，她从小走丢，一直在海城的一个乡下长大的。”顾夜瑾淡漠道。
<br/>
<br/>        陆寒霆没说话，他英俊的五官镀在昏暗的光线里，看不真切，“你慢慢玩，我先走了。”
<br/>
<br/>        陆寒霆抱着小圆圆离开。
