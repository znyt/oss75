　　这里是尸族势力范围的核心，山上有无数的洞府，越往高处，居住者的身份越高。在天弃山的山顶上，住着三大尸王：后卿，赢勾，和女魃。
<br/>　　一个身穿长袍，衣着素雅而严谨的高个男子，缓缓走在上山的路上。
<br/>　　虽然空界之战已经展开，但这里作为尸族的大本营，还是有很多尸兵镇守，以免被零界的那些不服管的邪恶生灵趁虚而入。
<br/>　　路上那些守兵见到男子，都会跪下，口中叫一声“主上”或是“真主”，他是尸族的主宰，是这个永夜帝国的主宰：后卿。
<br/>　　后卿一路上山，途中遇到一个尸族的将领，问了下天弃山的防御情况，之后一路上到山顶，来到自己的洞府前。洞府外面是一道水晶的拱门，两扇门的中间，有一个转盘，上面镶嵌着五颜六色的宝贝，这并不是装饰，而是五行石，五块石头，元素不同，好像八卦一样有相生相克，按照特殊的规律摆放在一起，会形成封印的力量，并且会一直保持。
<br/>　　后卿把手贴在圆盘上，轻轻用力，让自己体内的力量倾注进去，转盘上的封印，就像指纹锁一样，能够识别出力量的源头，确认主人的身份。
<br/>　　接着门好像卷闸门一样缓缓打开。
<br/>　　指纹锁配上卷闸门，简直堪比人间的高科技了。
<br/>　　门打开后，显示出来的洞府的内部，完全不是一个山洞的样子：洞府的四面都修成了横平竖直，细看是一块块石条镶嵌而成，每隔少许距离就掏出一个壁洞，里面摆放着龙眼大小的圆形宝石，发出蓝色的幽光，勉强能将山洞照亮。
<br/>　　山洞狭长而幽深。
<br/>　　后卿一路走进去，转过了两道石门，进去了一个好像大厅的所在，大厅很空旷，与外面的山洞不同，通体都是蓝色的玉石打造，将整个大殿映照的透亮。
<br/>　　在大厅的最中间，有一道环形的玉石平台，周围有一波清泉，绕着平台徐徐流动，水中不断冒着泡，形成的水汽在空中维持不散，远远看去如同一道半透明的纱帐。
<br/>　　玉石平台的中间，摆着一张雕花木床，床前还有一副沙发，和一个玻璃茶几。
<br/>　　在这种空灵的地方，这几件现代化的家具显得十分格格不入，有点不伦不类的感觉。
<br/>　　芮冷玉就坐在这张雕花大床上，听见脚步声，立刻站了起来，抬头望去，一眼看到后卿，愣了一会，本能地朝他冲了过去，结果被那道看似柔弱的水帘挡住。水帘好像一道坚不可摧的玻璃，芮冷玉两手拍在上面，纹丝不动。
<br/>　　“不要费力气了，这道封印，你是打不开的。”后卿在她对面不远处站住，定定的望着她说道。
<br/>　　“后卿，这是什么地方，我怎么会在这！你又怎么会在这！”总算见到个人，虽然是后卿，芮冷玉还是忍不住大叫起来。
<br/>　　后卿想了想，道：“你还记得什么？”
<br/>　　“我记得我跟少阳一起去星宿海……后来我们为了救少阳……好像苏沫摇铃，想要唤醒道风作为鬼童的意识……啊，少阳才是鬼童啊！少阳怎么样了！”
<br/>　　芮冷玉瞬间想起了这些事情，开始为叶少阳紧张起来。
<br/>　　后卿脸上露出无奈的神情，盯着她看了一会，说道：“之后的事情，你不记得了？”
<br/>　　“我……我不记得了，后面发生了什么？我为什么一点印象没有，还有，这里是什么地方，我怎么会在这，为什么会跟你在一起？”
<br/>　　后卿缓缓说道：“我还是一件一件告诉你吧，第一，鬼童觉醒了，但不是少阳。”
<br/>　　芮冷玉一下怔住了，捂住了嘴巴。“难道……是道风？”
<br/>　　后卿摇摇头：“也不是道风，而是……你。”
<br/>　　“你胡说什么，怎么可能是我！”芮冷玉愣了一下之后，冷冷说道。
<br/>　　“真的是你，不然你怎么会来这里，你刚醒，很多事情想不起来，我来慢慢告诉你……”
<br/>　　后卿不厌其烦地将她觉醒之后的事情讲了一遍，芮冷玉在震惊中听完，饶是一向处变不惊的她，在听完这些事之后，也近乎崩溃，双手用力扯着自己的头发，蹲在了地上。
<br/>　　后卿很想过去安慰她，但是想到这是她必须经历的过程，也只好默默在一边呆着了。
<br/>　　但芮冷玉的心理素质不是一般的强，这个痛苦的过程没有经历太久，就冷静了下来，开始根据后卿说的经过，在心里回忆起来，似乎感到有一点像是经历过，但是没有具体的记忆。
<br/>　　“我想不起来了，为什么我自己做的事，我会想不起来？”
<br/>　　“因为你那时候刚觉醒，戾气爆发，你自己做的事，完全不受心智控制，不然你怎么会伤了叶少阳。”
<br/>　　“我伤了少阳……”芮冷玉伸出手，低头看着自己的双手，眼泪止不住地掉下来，从指缝中滑落。
<br/>　　最后她捂着脸痛哭了起来。
<br/>　　后卿默默想着她，说道：“我说的都是真的，我没有必要骗你，我也永远不会骗你。”
<br/>　　芮冷玉哭了一会，擦了一把眼泪，问道：“听你说的，少阳被人救走了？”
<br/>　　“对，一个我不认识的老道士，很是厉害，用了不知道什么东西，把他跟东皇钟一起收了，但是我派人打听过，他不在人间，也不在贵族这样的地方，去哪了没人知道。也许他已经死了。”
<br/>　　“不，少阳不会死！”芮冷玉有点愤怒地看着他。
<br/>　　“你怎么知道？”
<br/>　　“我有直觉，他没死，他不会死！”
<br/>　　与其说是直觉，不如说是她心中的信念，她相信叶少阳一定不会死，一定还活着。
<br/>　　“后来，你就把我掠来了吗？这是什么地方？”
<br/>　　“这是我的洞府，你昏迷了很久。还好你恢复了。”后卿对她表现出了极大的耐心和柔和，跟平时那个在战场上那个冷酷绝情、威凌天下的他截然不同，“全天下都在追杀你，人间法术界，阴司，都要杀了你，没有人可以保护你，除了我。所以我把你带到了这里，这是我的洞府，只要我还活着，这里就是绝对安全的。”
