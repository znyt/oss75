        &lt;h3&gt;第三卷第一百七十五章  生命之花长存，友谊之树长青&lt;/h3&gt;
<br/>
<br/>        陈子昂公司的年会大家都参加了，因为各自身份的不同，他们比平时还忙，寒伯安、江俞轩、郝景文三个人开完公司的年会，各自带着自己的女朋友准备在一起吃顿晚宴，也算是送旧迎新。
<br/>
<br/>        寒伯安带着安男、江俞轩带着张倩楠、郝景文是毫不例外的带着她的小甜心陈思宇，陈子昂当初说的她要带着俩个孩子，却因为孩子有些感冒而只有她独身一人参加他们几个人的宴会，大家已经是很熟悉了，也就不那么拘束，安男因为知道了手镯的事情，就拉着陈子昂问东问西，对陈子昂的出生地充满了好奇，也对陈子昂的家庭充满了好奇。
<br/>
<br/>        安男：“子昂你兄妹几个人，你家里都有些什么人啊？”
<br/>
<br/>        “我爷爷奶奶带着我爸爸逃难到了我们现在住的地方，爷爷奶奶已经走了，我爸爸说当初只有他一个人，所以他就生了八个孩子，我有三个哥四个姐姐，不怕你笑话，我的年龄和我大姐的孩子一样大。”陈子昂笑着给安男讲她们家的情况。
<br/>
<br/>        “那你们的祖籍在哪里啊？”
<br/>
<br/>        “河北！具体的地方我不知道。”
<br/>
<br/>        “那你父母亲身体都好吗？”
<br/>
<br/>        “还不错呢。偶尔有个头疼脑热的。”
<br/>
<br/>        “那你们都在外面，谁照顾老人家呢？”
<br/>
<br/>        “我二哥呢。”俩个人一问一答。
<br/>
<br/>        “好羡慕你们家那么多兄弟姐妹，我家就我一个人。”安男真的是有点羡慕。
<br/>
<br/>        “你也可以将陈子昂当姐妹啊，她的兄弟姐妹不就是你的兄弟姐妹吗？”寒伯安走过来将安男起肩的短发拢了拢。
<br/>
<br/>        “是啊，我也是你的姐妹呢，我家的哥哥姐姐都是你的哥哥姐姐，不过六哥一直下落不明。”陈子昂不知道怎么了，今天心里有那么一丝丝的忧伤。
<br/>
<br/>        “六哥为什么下落不明？”安男很自然的跟着陈子昂一起叫起了六哥。
<br/>
<br/>        “我十二岁的时候六哥就出门了，一直到现在没有任何消息。”陈子昂的伤心已经是挂在脸上了。
<br/>
<br/>        “子昂，没有消息就是最好的消息，你不要担心，我们一起找六哥，伯安，你要帮我们找六哥。”安男对坐在身边的寒伯安说。
<br/>
<br/>        “好，子昂，你将六哥的信息和样貌给我说一下，我回去就发动人帮忙找。”老婆有令，寒伯安立马答应。
<br/>
<br/>        “其实吧，陈家姊妹长得差不多，我倒是前几年见过一个男的长得和子昂真是很像，如果子昂男装或者那个男的女装，俩个人怕是不分上下，一般的人很难分辨出来。”江俞轩接过话。
<br/>
<br/>        “是谁？”寒伯安他们齐声问。
<br/>
<br/>        江俞轩：“甸城的首富之子，谷强！”
<br/>
<br/>        “谷家的产业倒是设计的地域很广，也不端端是甸城，据说谷强很能干，外界有传说谷强是谷老大的义子，父子俩个人同样神秘，你能见到倒是奇迹。”寒伯安拿起雪茄，安男将雪茄拿到了手中。
<br/>
<br/>        张倩楠一言不发，她和张函去西城查过资料，知道陈子寒的身份，谷强的身份真的是不简单呢，让人从心底尊敬。
<br/>
<br/>        在这个和平的年代里，人们是不会发现像陈子寒这样的人的存在的，他们只是普普通通的工作人员，只有在需要的时候才会挺身而出，他们秘密的奉献着他们的青春，乃至一生，在黑暗的角落里蛰伏着，给予黑色以重击而换取人们的安宁。
<br/>
<br/>        “陈总，这就是您一直想要去甸城的原因吗？”陈思宇开口询问陈子昂。
<br/>
<br/>        大家都看了一眼陈思宇。
<br/>
<br/>        “嗯，可以这么说，那个谷强真的是太像我六哥了，我们见面的时候感觉很亲切，但是他好像又是第一次认识我。”在座的都是自己人，陈子昂也没有必要藏着掖着。
<br/>
<br/>        江俞轩当着大家的面接了一个电话：“函哥？对啊，我在北京，倩楠和我在一起啊，在什么地方？在mm酒店，我们几个聚餐呢，什么，你也在这里？咋到处都能碰到你？”
<br/>
<br/>        “倩楠，函哥在这个酒店呢，你要不要去看看？”江俞轩握着话筒问张倩楠。
<br/>
<br/>        张倩楠：“那我过去看看，你问一下他在哪里？”
<br/>
<br/>        “你过去干嘛？将你大哥喊过来啊，我还挺想见他的。”寒伯安不经意的说。
<br/>
<br/>        “是啊，张总人很好的，叫过来我们一起聊聊，还能听到很好的建议和见解。”郝景文也是很赞同。
<br/>
<br/>        “子昂？”江俞轩看着陈子昂。
<br/>
<br/>        “叫过来吧，大家一起聊聊多热闹的，你看你们成双成对的，就我一个人，拉过来凑个双数也好。”陈子昂也毫不在意的说。
<br/>
<br/>        几个人心里一惊，陈子昂如此不在意，是不是真的就放下了？
<br/>
<br/>        世界上哪有那么多巧合？聪慧的陈子昂已经从第一次大家的演戏中回过神了，今天大家也无非是如法炮制了一下而已。她心里很是感激大家为她做出的一切，那次见到张函和俩个孩子在一起的模样，她深深的被震撼了。
<br/>
<br/>        一切都成了过去，为了孩子她选择接受，但是只是接受张函是孩子父亲的事实，别的，就免谈吧。
<br/>
<br/>        张函如愿的和陈子昂坐在了一起，他感觉他的心离陈子昂又近了一点。
<br/>
<br/>        “张总，你这每次巧合就不是巧合了哦。”寒伯安待张函坐定后开了个玩笑，陈子昂能那么笨的认为每次都是巧合?
<br/>
<br/>        “三生石下，彼岸花开，见或者不见她就在那里，哪里有什么巧合不巧合的呢。”张函微笑着说。
<br/>
<br/>        “情深缘浅，见或不见，有什么意义呢？”郝景文竟然叹了一口气接话。
<br/>
<br/>        “沧海桑田，逐云追月，总有一天梦想会实现。”张函的语气里带着毋容置疑。
<br/>
<br/>        “愿花常开，愿情常在，愿天下所有的有情人能够携手到老，三生相依！”安男怕大家起了冲突，便举起了酒杯。
<br/>
<br/>        郝景文捏了一下陈思宇的手，俩个人同时拿起了酒杯。
<br/>
<br/>        江俞轩看着张倩楠：“倩楠，你意思一下。”
<br/>
<br/>        张倩楠点了点头，看着张函，心里替哥哥感到一阵悲凉，陆玉去了甸城，两年了都不曾回到张家，就这样死死的拖着张函，也不知道什么时候能到头。
<br/>
<br/>        “辞旧迎新，生命之花长存，友谊之树长青，张总、大哥、俞轩、文文、三位姐妹，来我们干杯！”陈子昂大大方方的举起酒杯一饮而尽。
<br/>
<br/>        人生路上，友情应该是最长久的，他可以陪伴你一生，在你需要帮助的时候伸出援手，给予你温暖，陈子昂怀着一颗感恩的心接纳着身边的一切，用一颗积极而温暖的心也感染着周围的人们。
