　　　　片刻过后，尖锐的声音绕耳不绝，伴随着六合镜投射出的光芒，黑云本体也暴露在众人的视线中。
<br/>　　放眼望去，六尺外全都被密密麻麻的黑色蝙蝠所包围，看它们的身形，比往日所见的蝙蝠大了一倍不止，每一只都张着血盆大口，在黑暗中，口里猩红一片，恐怖狰狞。
<br/>　　张小凡等人汗毛炸立，心中一阵后怕，若是刚才没有使用六合镜，他们恐怕已经被这群蝙蝠撕成了碎片，顿时看向慕清霄的目光满含深意。
<br/>　　此时，六合镜的护主功能起到了很大的作用，所有的蝙蝠都被隔离在光罩外面，任它们如何撞击挤压，光罩都纹丝不动。
<br/>　　而且，蝙蝠一旦接触光罩，便会有一股淡黄色光芒窜入它们体内，使得它们黑色身躯犹如雷击，哆嗦一阵就掉落在地，挣扎几下便失去了生息。
<br/>　　只是，凝成黑云的蝙蝠实在太多，放眼望去连头顶夜空星斗都被遮掩了，数以百万计都是少的，死在六尺外的只能算九牛一毛罢了。
<br/>　　无数蝙蝠前赴后继的冲上来，可都无一例外的被挡在光罩外面，那令人作呕的腥气也随之渗透进来，众人除了受到些惊吓，总的来说都平安无事。
<br/>　　六合镜毕竟是道家至宝，在这无数蝙蝠群的冲击下，竟丝毫没有动摇的迹象，那黄色光罩看似轻薄，可却屹立如山，仅仅片刻，六尺外的蝙蝠尸体就堆积成了小山。
<br/>　　蝙蝠群不愿离去，张小凡几人面色煞白，因为他不知道六合镜能护主多久，慕清霄则一脸平淡，蝙蝠虽然众多，可对他来说犹如土鸡瓦狗，不堪一击。
<br/>　　倒是站在他旁边的陆雪琪脸色苍白至极，纤纤玉手紧紧抓着他的衣袖。
<br/>　　慕清霄微微摇头，真是死要面子活受罪，一把将她拉入怀中，陆雪琪惊呼一声，想要反抗，可是感受到这安全温暖的胸膛，顿时犹豫起来。
<br/>　　慕清霄的举动，众人自然看在眼中，张小凡也见怪不怪，曾书书眼底闪过一丝失落，林惊羽面色严肃，众人各怀心思。
<br/>　　陆雪琪芳心‘砰砰砰’直跳，象征性的推搡几下，可是慕清霄将他抱的死死的，简直要将她揉进自己的身体，心里喜滋滋的，宛如吃到糖果的小姑娘。
<br/>　　瞥了眼外面如浪涛一般的蝙蝠群，陆雪琪恐惧也减少了几分，闭上美眸静静的趴在慕清霄怀中，不再管外界的情况，哪怕六合镜挡不住，能够与心爱的人死在一起，此生亦无悔。
<br/>　　随着蝙蝠尖锐的叫声，不断的冲击着光罩，张小凡等人身体已经被冷汗浸湿，唯有慕清霄与陆雪琪心情平稳，这种状态一直持续到黎明。
<br/>　　清晨，旭日东升，第一缕阳光照射在空桑山中，蝙蝠群发出一阵尖叫，仿佛是在恐惧，不敢在做停留，径直向昨夜飞出的返回，仅仅片刻，蝙蝠群就消失的无影无踪。
<br/>　　张小凡几人总算是松了口气，慕清霄轻轻的拍了拍陆雪琪的香肩，道：“雪琪。”
<br/>　　陆雪琪缓缓睁开美眸，她自然知道天亮了，蝙蝠群已经离去，只是有些留恋温暖的胸膛，不舍离去而已。
<br/>　　确认蝙蝠群不会在返回，张小凡撤去六合镜，光罩消散，周围堆积成山的蝙蝠尸体直接挤压过来，慕清霄随手一挥，恐怖的灵气席卷而出。
<br/>　　只听见‘轰’的一声，中间就冲出一条宽敞的道路。
<br/>　　曾书书几人都知道慕清霄修为极高，却发现依旧小觑了他，由蝙蝠堆积起来的小山坡，重量可想而知，随手一挥就轰出一条宽敞的道路，最起码他们做不到。
<br/>　　离开原地数百米后，众人停下休息了片刻，纷纷拿出干粮和水源，坐在树下休息。
<br/>　　由于死去的蝙蝠群距离此地并不远，迎面而来的空气中隐隐参杂着一丝腥气，陆雪琪依旧有些反胃，慕清霄只是递了壶清水过去。
<br/>　　两人早就是恋人关系，陆雪琪也没有道谢，接过水壶就喝上两口，总算是舒服的多。
<br/>　　就在众人休息的时候，上空传来一阵呼啸，众人抬头望去，只见天际闪现四道光芒，转眼在他们前方落下，两黄一白一青，光芒消散，出现四道身影。
<br/>　　左侧是两名和尚，右侧分别是一男一女，颇为年轻，男的俊俏，女的秀媚，站在一起极为般配，犹如金童玉女。
<br/>　　这四人看向慕清霄等人，看他们衣衫白净，都是皱了皱眉，眼中有些诧异，其中一名白净的和尚首先口宣佛号，道：“阿弥陀佛，请问各位施主可是青云门下弟子？”
<br/>　　不知不觉中，张小凡几人都以慕清霄为首，见他不说话，只顾着与陆雪琪交谈，场面一时间有些尴尬，曾书书连忙上前一步，回了一礼，道：“正是，在下曾书书，请问诸位是......”
<br/>　　白净的和尚微微一笑，道：“小僧是天音寺法相，这位是师弟法善，旁边这两位乃是焚香谷的杰出弟子李洵，燕虹。”
<br/>　　焚香谷李洵和燕虹都是神情倨傲，仿佛高人一等，只是微微点头，就算是见过礼了。
<br/>　　曾书书的性格也是见人说人话，见鬼说鬼话，既然如此，他也不想热脸贴了冷屁股，当下便不理焚香谷两人，向法相道：“久仰天音寺法相师兄大名，被正道修真誉为千年罕见的人才，今日得见，果然风采过人。”
<br/>　　听到天音寺，张小凡身躯一颤，目光有些复杂，想到自己现在的修为与天音寺有关，心中复杂的情绪逐渐化去。
<br/>　　法相微微一笑，心中颇为受用，道：“曾师兄实在过誉了，小曾资质愚钝，唯恩师普泓不弃，传授我真法，以期为天下苍生做些善事，却不敢与青云门诸位师兄相提并论。”
<br/>　　曾书书嬉笑一声，连连摆手，道：“法相师兄太过谦了，这是我诸位师弟师妹，我为诸位引荐。”
<br/>　　说着，曾书书便将张小凡和林惊羽几人相互介绍了一下，法相听到张小凡的名字，神情怔了怔，不禁多看了几眼。
