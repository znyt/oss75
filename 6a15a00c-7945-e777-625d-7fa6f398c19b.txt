        说完这句话，童筱颖便有些后悔了。
<br/>
<br/>        她都干了些什么，他一定是一清二楚，她还这么说的话，岂不是“此地无银三百两”。
<br/>
<br/>        “你喜欢他吗？”傅夜沉突然问道。
<br/>
<br/>        “啊？”童筱颖顿时一脸懵。
<br/>
<br/>        “算了，当我没问。”傅夜沉剑眉微蹙，偏头望向了车窗外。
<br/>
<br/>        回到民宿后，童筱颖跟老板娘介绍了傅夜沉，说他是自己的老公。
<br/>
<br/>        老板娘也夸她老公长得帅，还是看起来就很养眼的那种。
<br/>
<br/>        童筱颖暗自窃喜，傅夜沉似乎是对别人的这种夸赞已经习以为常。
<br/>
<br/>        进了房间后，里面的装饰风格不同于酒店的高端大气，而是一种女生的闺房的感觉，地方虽然小，但看起来很温馨。
<br/>
<br/>        因为没地方坐，童筱颖只能让傅夜沉坐床上。
<br/>
<br/>        他可是第一个进她房间的男人，就连秦烈，她都没让他进来过。
<br/>
<br/>        “我去问问老板娘，看还有没有房间，我再帮你开一间。”童筱颖实在是找不到话题了，杵在房间里，面对傅夜沉时，都不知道自己该做什么。
<br/>
<br/>        当她欲要转身离开的时候，手腕突然被一只强劲有力的大手捉住。
<br/>
<br/>        一个旋身，她落坐在一双温实的大腿上。
<br/>
<br/>        童筱颖本能地环住傅夜沉的脖颈，微微垂眸时，对上了他那双点漆般的黑眸。
<br/>
<br/>        “除了我，我不许你爱上其他的男人！”他突然不容置喙地说道。
<br/>
<br/>        童筱颖怔愣地眨巴着眼睛，完全没料到他会突然跟她说这个。
<br/>
<br/>        他放在她腰间的手忽然紧了紧，童筱颖能感觉到来自于他手心上的温度在渐渐升温。
<br/>
<br/>        “除了你，我好像真的没法接受其他男人啊！”童筱颖凝视着傅夜沉，微笑着打趣。
<br/>
<br/>        但是微笑过后，童筱颖凝望着傅夜沉的那双眼睛，慢慢地浮泛着一丝泪光。
<br/>
<br/>        “可是，大哥你……唔——”
<br/>
<br/>        她的话还未说完，他突然托住她的后脑勺，一吻覆了上来。
<br/>
<br/>        童筱颖惊怔地瞪大了眼睛，在他深吻着她的时候，脑海里闪过一幅又一幅，像是曾经真实发生过的画面。
<br/>
<br/>        她好像看到，另一个跟她长得一模一样的女孩，被他从水果店里拉了出来，然后又像现在这样，突然被他强吻。
<br/>
<br/>        这种感觉，和现在一模一样。
<br/>
<br/>        不知吻了多久，他慢慢地放开了她。
<br/>
<br/>        他的指腹，摩挲着她那被他吻得殷红而湿润的红唇，然后缓缓地落在了她的衣襟上。
<br/>
<br/>        傅夜沉看着童筱颖因为呼吸而起伏的胸脯，大手定格在她胸间衣襟的纽扣上踟蹰。
<br/>
<br/>        他终究还是没有进一步的动作。
<br/>
<br/>        童筱颖忽然间明白了什么，就在傅夜沉放下手的那一刻，她下意识地抓住了他的大手。
<br/>
<br/>        “你不碰我，是不是因为你认为我已经不是你爱的那个妻子？”
<br/>
<br/>        她的一句话直接戳中他的心结。
<br/>
<br/>        傅夜沉眸底暗了暗，并未吭声。
<br/>
<br/>        他的沉默，代表应证了她的这个说法。
<br/>
<br/>        童筱颖拉起傅夜沉的手，与他十指相扣，接着问道：“如果我哪天得了失心疯，你是不是就会很嫌弃我？”
<br/>
<br/>        “你这说的是什么话？”傅夜沉剑眉微蹙地冷斥。
<br/>
<br/>        童筱颖不禁振振有词起来：“我在实验室里的时候，那些医生说我生病了。所以不停地在抽我的血，不停地拿我去做研究。大哥现在很忌讳我，不就是因为我的病吗？大哥不愿意接受生病的我。又或者说，你把我当成了两个不同的人。我不明白，明明我就是我。为什么你要把我分割成两个不同的人？”
<br/>
<br/>        “那你告诉我，是谁给你下达，要你来杀我的命令。”傅夜沉十分淡定地问。
<br/>
<br/>        童筱颖却微微摇头，并举起另一只手来，一本正经地发誓道：“我向你保证，以后不会再出现上次那种情况！”
<br/>
<br/>        看到她如此认真的样子，傅夜沉原本一脸严肃的俊脸，在不经意间地染上了一丝欣慰的笑容。
<br/>
<br/>        他拉下她的手，轻轻地握在手心里，并凝望着她的眼睛，说了声：“对不起。”
<br/>
<br/>        童筱颖不明白他为什么要向自己道歉。
<br/>
<br/>        当他再次吻上她的唇，而后在她耳边低语呢喃时，她才明白他道歉的意义所在。
<br/>
<br/>        此刻，好像所有的心结都被解开了一样，童筱颖双手往傅夜沉的肩膀上一摁，直接将他推倒在床上后，自己整个身子直接趴到了他的身上。
<br/>
<br/>        “大哥，我明天想去西班牙！”
<br/>
<br/>        “好，我陪你。”
<br/>
<br/>        “然后，我们再去希腊！”
<br/>
<br/>        “嗯，我陪你。”
<br/>
<br/>        “那我要把我没去过的国家，都去一遍。”童筱颖满心欢喜地说。
<br/>
<br/>        傅夜沉搂着童筱颖的腰肢，微笑着看着她，连声应道：“好，我都陪你去。”
