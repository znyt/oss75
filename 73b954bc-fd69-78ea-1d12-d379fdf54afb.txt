
<br/>　　<br/>　　    小林昙一被压在了地上，陈三岁的姿势有点别扭，一脚锁着他的脑袋，一手按着他的胸膛，胳膊肘子压在他的两腿上，对方挣扎的力道太大了。<br/>　　<br/>　　    王惊蛰手插在口袋里，站在一旁低头看着下面，陈三岁咬牙喘气的说道：“看出什么来了嘛……”<br/>　　<br/>　　    如此费力的擒拿住对方，王惊蛰和陈三岁就是要品品此人到底为何失常了，鬼上身肯定不是，反倒还带着点神圣的味道，发疯发的太让人莫名其妙了。<br/>　　<br/>　　    “有点像是神打术？”王惊蛰不太确定的说道。<br/>　　<br/>　　    陈三岁愣了愣，觉得他说的不对，就摇头说道：“神打，就是请神上身么，你别告诉我这小鬼子往上祖宗几辈或者有什么师门是出自于藏区这边的，明显驴唇不对马嘴啊”<br/>　　<br/>　　    王惊蛰摊手：“我真的是想不通了，他的双眼失神，这是精神全失的迹象，魂魄肯定被附了，可什么孤魂野鬼敢来到这种地方？说不通，说不通，我刚开始还以为是神打，可仔细想想也确实不对，我搜肠刮肚的想了半天，也没找到对应的症状”<br/>　　<br/>　　    陈三岁说道：“那就是疯了？”<br/>　　<br/>　　    王惊蛰很笃定的摇头说道：“疯是疯了，可也不能全这么说，我觉得如果非得贴切的说，应该用崩溃来解释，最合适”<br/>　　<br/>　　    陈三岁张了张嘴，茫然摇头说道：“不懂，什么崩溃？”<br/>　　<br/>　　    “醍醐灌顶这个词你听过么？从这个角度理解的话就容易点了，我以前曾经听家里人说过，藏区有活佛，法王，尊者等可能会受到上一代的传承，其中有一项叫醍醐灌顶”王惊蛰想了想，努力用通俗易懂的解释道：“武侠小说里，有时就会写到这样一幕，当有武林高手临死之际，会把自己的一身功力传给有缘人，比如一甲子内功？书里前通常都会这么写，主角能受得了高手的醍醐灌顶那就一飞冲天了，平白多了一甲子的内力么，可要是受不了这个传功的话，就有可能爆体而亡了，俗称崩溃！”<br/>　　<br/>　　    陈三岁顿时无语的说道：“你武侠小说看多了吧，这是现实，哪里来的这种狗屁情节啊，怎么可能？”<br/>　　<br/>　　    王惊蛰很认真的说道：“在别的地方不可能，但是在藏区可以……老孔，你听说过么？”<br/>　　<br/>　　    孔良随即点头说道：“他说的没错，藏区活佛，尊者，法王有时先找到自己的转世身，是有可能将自己的一身佛力传承下去的”<br/>　　<br/>　　    陈三岁：“……”<br/>　　<br/>　　    王惊蛰忽然蹲下身子，指着小林昙一说道：“要想印证这一点很容易啊，杀了他”<br/>　　<br/>　　    “啊？”<br/>　　<br/>　　    王惊蛰又重复了一句：“杀人吧”<br/>　　<br/>　　    三井美奈捂着小嘴，轻声惊呼了一句，没想到自己看走眼真是走的太离谱了，她还以为对方是个有色心也有色胆的登徒子，一心想着要啪了她，可哪里想到这人狠辣的时候杀人都不眨眼，对她在现在更是从始至终都没有正眼看一下。<br/>　　<br/>　　    “男人果然都是善于伪装的！”三井美奈心里没来由的想起这么一句话。<br/>　　<br/>　　    “嘎巴”陈三岁干脆利索的扭断了小林昙一的脖子，然后嘟囔了一句：“心狠手辣的事，为什么要我来做，你不也一样么”<br/>　　<br/>　　    王惊蛰淡淡的说道：“你推过那么多的尸油再杀一个两个的，也不是什么大事，而我就不同了，手底下沾上一条人命就是一个因果，我不愿意受这种业障，所以还是你来吧”<br/>　　<br/>　　    陈三岁：“……”<br/>　　<br/>　　    当小林昙一闭眼咽气之时，王惊蛰突然快速的伸出手指点在了他的眉心上，随即他和陈三岁同时看见尸体上升起一道魂魄出来，两人对视一眼，满脸皆惊。<br/>　　<br/>　　    陈三岁随后是不解，迷惑，王惊蛰望了几眼后，点着头说道：“原来如此，原来如此，我想通了”<br/>　　<br/>　　    小林昙一的魂魄此时已经不是寻常的魂魄了，魂上萦绕着一圈金光，看着特别的宝相庄严，隐隐透出一股神圣的念力，就跟之前王惊蛰感觉到的一样。<br/>　　<br/>　　    “你都明白了什么？”陈三岁问道。<br/>　　<br/>　　    王惊蛰指了指神庙的大门，说道：“敢去么，我说的九死一生的局，试试？”<br/>　　<br/>　　    陈三岁默然，王惊蛰接着说道：“来了就是机缘，错过就是可惜，虽然对你来说心已半死，什么缘不缘的都已经不重要了，不过我却很想试试，不光是你我，老孔你呢？”<br/>　　<br/>　　    孔良完全都没听明白两人的交谈，
