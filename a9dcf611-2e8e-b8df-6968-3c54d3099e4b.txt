　　天际远方。
<br/>　　一道火焰流光，疾速飞行着。
<br/>　　但，火焰流光，却不时踉跄一下，似乎极为不稳。
<br/>　　火焰流光，自是远遁的萧逸。
<br/>　　“该死。”萧逸咬了咬牙。
<br/>　　面具之下，他的脸色已煞白无比。
<br/>　　嘴角处，蔓延滴落的鲜血，渐渐染湿大半衣衫。
<br/>　　之前远遁时，后背硬扛龙啸的两道攻击，他显然受了不轻的伤势。
<br/>　　他的修为，终究只是圣皇境七重修为罢了。
<br/>　　借着一身控火手段，以及兽手兽脚的增幅，还有龙炎暂作第七种火焰融合，方才有了与龙啸一战之力。
<br/>　　龙啸本就是绝世9999道，传奇之下第一人。
<br/>　　若再加上地域之力，六龙煞气，在地域之内，他等同无敌。
<br/>　　原本，那条雷龙煞气出现时，萧逸还有几分底气，也自信有一战之力。
<br/>　　事实证明，他手段尽出下，确实能与龙啸一战，甚至于，败龙啸也未可知。
<br/>　　可当第二条龙形煞气，也就是那条木煞出现之时，萧逸便知，在六龙地域之内，与龙啸一战，根本不可能会有半分胜算。
<br/>　　那时，他方算知晓，为何六龙地域称之为六龙地域。
<br/>　　他也才更加明白，为何龙家，以及龙家历代家主，传承无数年，却至今无人能撼动。
<br/>　　且龙啸本身，自傲狂傲到极点，似乎拥有着无比的底气。
<br/>　　那，才仅仅是第二条龙形煞气罢了。
<br/>　　若是六龙地域内的龙形煞气尽出，龙啸的实力，将强到何等可怕的地步？
<br/>　　这一战，本就是毫无悬念的一战，自然的，萧逸只能远遁逃离。
<br/>　　不过，事实上，萧逸一个成名不多久的年轻人，能硬撼龙啸这个成名多年甚至大陆公认的传奇之下最强者，已然相当了不起。
<br/>　　而且，战斗发生在六龙地域之内。
<br/>　　龙啸在自家地域内，被生生逼得使出了两条龙煞，且最后还只能眼睁睁地看着易霄逃离而无法追击。
<br/>　　单这几点，就足以让易霄之名再次飙升到一个极高的地步。
<br/>　　此事传出，怕是在外人眼中，更关注的，非是易霄败于龙啸之手；而是，一个恐怖的控火妖孽，正在以惊人的速度，快速崛起着。
<br/>　　言归正传。
<br/>　　萧逸逃遁，除了他不敌龙啸外，另外一点，便是他和龙啸的这一战，充满了莫名。
<br/>　　他当然知道，龙啸是因为龙天的重伤还有武道根基被废，而愤而出手，杀意凛然。
<br/>　　但龙天之事，萧逸可不觉得自己真的有出手如此重。
<br/>　　他是杀伐过人，可不代表他不分青红皂白。
<br/>　　而且，那不过是他随意一抛罢了。
<br/>　　若说他控制不住自己的力度，或者火焰威力，那根本不可能。
<br/>　　以他的控火本事，这点火焰威力都控制不住？那简直是笑话。
<br/>　　伤人还是杀人，就是一个普通武者都能轻松控制，更别说他萧逸。
<br/>　　唯一的可能，便是其中有端倪。
<br/>　　而最大的可能，便是出于一开始就瞬间接住龙天的三长老。
<br/>　　只是，他和龙家三长老无怨无仇，陷害他做什么？
<br/>　　非要说仇怨，也不过是在前些天在龙跃宗内，他将其打伤过。
<br/>　　可这丁点儿仇怨，至于三长老将他龙家少家主害得重伤，武道根基尽毁？
<br/>　　龙天的伤势，应该做不得假，毕竟龙啸也亲自感知过。
<br/>　　龙啸这等实力，不会感知错的。
<br/>　　那么，问题出在哪？
<br/>　　而且，龙家三长老，怎可能有那个胆子害龙天？
<br/>　　萧逸实在想不明白。
<br/>　　他唯一能想到的可能，只有一点，那便是三长老也是邪修。
<br/>　　只有如此，三长老才会这等事；一来，陷害他，让他无法继续剿杀六龙地域内的邪修分部；二来，直接让他和龙啸站在了对立面之上。
<br/>　　再者…萧逸蓦地想起来，当初在五虎门时，龙啸和二长老那堆莫名其妙的话。
<br/>　　当日屠杀龙跃宗时，龙跃宗内，没有留下邪修令牌？
<br/>　　三长老还被他打成重伤？
<br/>　　此事，当日萧逸一时在气头，毕竟龙啸放跑了虎护法，便没有深究。
<br/>　　现在想想，愈发不对劲。
<br/>　　若真如他猜测的那般，龙家三长老是邪修，那么这一切，便是顺理成章了。
<br/>　　可…这一点，萧逸却并不认可。
<br/>　　若龙家三长老是邪修，他早便发现了。
<br/>　　毕竟他和龙家三长老接触过多次。
<br/>　　他的太阴太阳之眼，窥探天地，可以准确知晓地域之内所有邪修的准确位置。
<br/>　　可这些邪修本身是什么身份，这一点是无法知晓的。
<br/>　　在他的天地感知中，邪修，不过是一道道黑气。
<br/>　　也是因此，他可以根据这一道道黑气，知晓邪修的位置；但却无法从一道道黑气中，发现那一道黑气属于毒袍。
<br/>　　自然的，也就不知晓毒袍在哪，也不知晓这些邪修的身份。
<br/>　　但，若是面对面接触，是否邪修，决计瞒不过他萧逸。
<br/>　　在萧逸与龙家三长老的接触中，并未发现三长老是邪修。
<br/>　　这一切，似乎成了一个死结。
<br/>　　若三长老不是邪修，他为何要做这些事？
<br/>　　之前的一切，根本解释不通。
<br/>　　若三长老是邪修，又根本瞒不过他萧逸。
<br/>　　事情，无论如何都说不通。
<br/>　　“噗。”萧逸一边疾速飞行，一边思索着，蓦地，一口腥血喷出，打断了他的所有思索。
<br/>　　这口腥血，才堪堪喷出，已在空气中瞬间蒸发殆尽。
<br/>　　看真切些，萧逸周遭，已是一片炙热。
<br/>　　周遭空气，不断在热浪中扭曲、蒸发…
<br/>　　而萧逸的身体，则热得发红。
<br/>　　皮肤，灼热无比，体内，亦是一阵五内俱焚，仿佛一股火焰要迸发而出的难受疼痛感。
<br/>　　事实上，让萧逸远遁逃离的最大原因，不是别的，而是他体内的龙炎反噬，已然快支撑不住了。
<br/>　　龙炎，绝非人力所能操控驾驭的。
<br/>　　非要强行驾驭使用，其结果，便是恐怖的反噬。
<br/>　　“我的手…”萧逸看了眼自己的手，陡然瞳孔一缩。
<br/>　　他的两只兽手，已然收回。
<br/>　　而他自己的手，此刻正滚烫发红。
<br/>　　这哪里像一只人类的手，反倒更像是一烧红了的铁疙瘩。
<br/>　　而最重要的是，这铁疙瘩，仿佛是从熔炉中取出。
<br/>　　此刻，仿佛要融化了一般。
<br/>　　“额…”萧逸要紧了牙。
<br/>　　手上的疼痛感，渐渐蔓延全身。
<br/>　　浑身炙热着，仿佛快要融化了一般。
