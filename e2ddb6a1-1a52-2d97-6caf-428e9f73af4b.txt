        保安将他和那个女生一起带去了教务处，女生一直在那哭哭啼啼，一口咬定是他要强暴她。
<br/>
<br/>        而且，又有两个保安作证说他连裤子都脱了，就差点要上了那个女生，还好他们及时赶到，才救下这个女生，真的是让他百口莫辩。
<br/>
<br/>        出了这事，自然是要叫家长。
<br/>
<br/>        那女生的家长，当晚就报了警，而且还将他给告上了法庭。
<br/>
<br/>        所以，他作为犯罪嫌疑人被带去了警局录口供以及让法医做全身检查和心理测试，还被迫在少年监里关了几天，等待警察的调查结果和法医的法检结果以及法院开庭。
<br/>
<br/>        最后，各种检查结果出来了。
<br/>
<br/>        他身心都没问题，也没有撒谎，身上最新的指纹，除了他自己的就是那个两个保安的，以及裤子两边有那女生的指纹，可以证明他的裤子是被那女生用力拽下的。
<br/>
<br/>        而那女生有撒谎的嫌疑，身体上和衣服上，有其他嫌疑人的指纹，唯独没有他傅夜沉的指纹，至少这点可以证明他傅夜沉没有碰过她，连她的衣服，他都没摸过。
<br/>
<br/>        所以证据不足，他傅夜沉被当庭无罪释放。
<br/>
<br/>        但那女生的家长却诬蔑，是他爷爷有钱有势，走了关系才让他无罪释放的。
<br/>
<br/>        对于那女生家长的诽谤，他们只给他家下了律师警告函。
<br/>
<br/>        后来，也就懒得理会他们了。
<br/>
<br/>        他爷爷虽然有钱有权有势，但不是那种徇私枉法的人。
<br/>
<br/>        他傅夜沉若是真的犯了事，爷爷绝对不会包庇他，若是真包庇他，他当时作为犯罪嫌疑人就不会被迫关在少年监里等各种检查结果了。
<br/>
<br/>        也正因为这件事情，傅炎廷才发现，他家这长孙还真是本事通天，普通高校的围墙压根就关不住他，所以才把他傅夜沉送去了军校，正好也随了他傅夜沉想继承父亲进入部队当兵的遗愿。
<br/>
<br/>        得知这件事情的来龙去脉后，童筱颖忍不住将傅夜沉扑倒在沙发上。
<br/>
<br/>        她匍匐在他胸膛上，忍俊不禁地说：“原来大哥当年也有犯傻的时候啊！”
<br/>
<br/>        “我那不叫犯傻，而叫完全不知所措。第一次遇上这种被女生冤枉的事情，我当时还担心我以后可能不能考军大了。”傅夜沉苦笑。
<br/>
<br/>        “这件事情，昼景他知道吗？”童筱颖接着问道。
<br/>
<br/>        傅夜沉微微耸肩：“我和爷爷都没告诉过他，不过，他后来也去了滨江一中念高中。可能听到过一些流言蜚语吧！”
<br/>
<br/>        “原来如此。”童筱颖若有所思起来。
<br/>
<br/>        “怎么了？”傅夜沉仍旧觉得童筱颖看起来像是有心事。
<br/>
<br/>        童筱颖回过神来，一手托腮，另一只手在傅夜沉的胸膛上画着圈圈，打趣道：“大哥，在我之前，你有喜欢的女孩吗？”
<br/>
<br/>        “没有。”傅夜沉不假思索地回答，“你是我第一个喜欢上的女孩子，也是我唯一的女人。”
<br/>
<br/>        其实，那件事情后，他对女生算是心有余悸了吧！
<br/>
<br/>        又加之，进了军校后，男生与女生之间的事情管得特别严。
<br/>
<br/>        他那个时候就读的军校并非大学，而是类似于全日制普高的军事化管理学习，除了日常普高课程以外，他们还得学习军事技巧，因为专业对口的大学只有军大，所以他们算是为以后进入军大做准备的。
<br/>
<br/>        能进军校的高中生，几乎全都是家里在军政界有背景的，所以，他们学校的学生并不多。
<br/>
<br/>        那个时候，他们同年级就两个班，每个班也就三十个人，来自五湖四海，班里的女生就十个。
<br/>
<br/>        而且，他们军校禁止学生谈恋爱，而女生们平常都是短发素颜，四季的校服均是长裤的军装连小腿都不许露，跟他们男生穿着一样。
<br/>
<br/>        他们还讲究部队精神，一个班就是一个小部队，所以，班上的女生们性格跟男孩子一样落落大方，大家都是铁哥们好兄弟的纯友谊，生活和学习上都会互帮互助。
<br/>
<br/>        所以，他没往谈恋爱那方面去想，也没想过要去谈恋爱。
<br/>
<br/>        后来在大学里，他选择的是特殊兵种专业，毕业后直接进入缉毒排暴除恶的特种部队，为了强化个人心理素质，抵制诱惑，也被上级安排接触过各色各样的女人。
<br/>
<br/>        在那些女人面前，他几乎是顺畅无阻的通关，要说唯一让他沦陷的女人，也就只有她童筱颖一人了。
<br/>
<br/>        那晚，他做任务受伤行迹败露，遭道上的人反追杀，从天台沿着外墙空调板一路跳下来，刚好跳到她那出租屋的半开放式阳台上。
<br/>
<br/>        本来他想一走了之，一边继续逃命，一边等待自己的同伴接应，哪知道那个时候，她童筱颖正好掀开窗帘，推开了玻璃门，与他撞上。
