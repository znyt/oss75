　　对此，战力已经无敌的楚涵也时不时的出现在天运国的王城里，将这些都打听的心知肚明。
<br/>　　国王的消失让两国关系紧张无比，同时每天都有不同的人前来此处林间，想要进入这个隐蔽的山谷。
<br/>　　当然，这些人一进入就消失的无影无踪，不仅没有任何信息传递出去，更让这一处地方让明成国的人忌惮异常。
<br/>　　人怎么会忽然消失，见鬼了不成？
<br/>　　到了第七天的时候，这里更是来了大批的人马，楚涵估摸着明成国派来了一整个军队！
<br/>　　人数多到几乎是地毯式搜索，更有里里外外的接应人员，明成国铁了心要将事情查的水落石出，根本不在乎这里是天运国的境内。
<br/>　　天运国的将军统领得到消息，也立即派出了军队往此处赶来，明成国的肆无忌惮已经让他们忍无可忍。
<br/>　　战争一时半会儿并没有打起来，双方都在僵持之中，一方不肯走，一方不肯让。
<br/>　　局势焦灼无比！
<br/>　　楚涵趁乱·摸进山谷里将国王带走，整个过程都悄然无息，没让任何一人发现。
<br/>　　终于能走出山谷的国王，此时简直像个乞丐，衣服脏乱不堪不说，浑身还散发着一股恶臭。
<br/>　　国王知道外界情况后，很想与天运国的大将军汇合，但楚涵的一句话就将她的行动堵了回去。
<br/>　　“你怎么知道大将军不是叛国者？”
<br/>　　国王不再吱声，默默的放任楚涵带她离开。
<br/>　　两人没再这处山林中继续训练，楚涵先带着国王回了一趟王国，依旧躲过了所有人的耳目，让其有机会好好收拾下自己。
<br/>　　收拾干净的国王换上了一身更方便行动的衣服，浑身不带一件装饰物，甚至只用了她平时梳洗打扮的十分之一时间。
<br/>　　“我可以了。”国王走到楚涵面前。
<br/>　　她不再一周前彷徨无措动不动还哭，此时的她眼神之中带着一股坚定，这也是楚涵最希望看到的眼神。
<br/>　　“眼神不错。”楚涵嘴角勾起，而后走到房间的窗边望着外面夜色。
<br/>　　偌大的王宫，没有任何异常，无人知晓在这短暂的几分钟里，国王已经回来过一趟。
<br/>　　“接下来我们做什么？”国王有些迫不及待，甚至带着凶狠：“杀过去吗？”
<br/>　　楚涵暗自感叹果然死人堆里爬出来的转变最大，而后道：“训练还没结束，杀什么杀。”
<br/>　　国王有些不解：“那上堂河旁边的树林，他们不会打起来吗？”
<br/>　　楚涵瞥了她一眼：“在那里打不是挺好的嘛，深山老林没有居住者，不伤及无辜不就是你想要的？”
<br/>　　国王皱眉：“可天运国的士兵……而且我认为大将军不太可能是叛国者，他平时就与国师不对付。”
<br/>　　楚涵看傻·子一样的看了她一眼：“哪怕大将军不是奸细，他也罪当处死。”
<br/>　　国王瞳孔剧烈收缩：“为什么？”
<br/>　　楚涵冷笑：“一国之将，敌军进入天运国境内甚至深入主城周边都不知道，大声嚷嚷着要将这群人赶出去，到了上堂河却全军止步不前，这么弱小的将领要之何用？”
<br/>　　这特么要是换成狼牙，楚涵都懒得想对方会被打成什么屁滚尿流的惨样。
<br/>　　一席话听得国王发愣，她没有任何理由反驳。
<br/>　　国王紧·咬着嘴唇，内心的一颗弦忽然就这么崩塌，她赖以信任的大将军，也有这么大的问题么？
<br/>　　“白允儿将我练兵的部分技术告诉给你了吧？”楚涵看了眼国王，问道。
<br/>　　国王点头：“可我只能理解一部分，在白允儿的帮助下，天运国的兵力有所增长，但真正打起来如何我却不知道。”
<br/>　　楚涵再次问道：“大将军参与其中？”
<br/>　　国王顿了顿，道：“是，他清楚我们所有会的战术。”
<br/>　　“哦。”楚涵不以为意，战术这种东西，他向来是临场发挥。
<br/>　　紧接着楚涵又心中一动，问道：“明成国为什么要偷走你的王冠？”
<br/>　　国王有些发懵：“我并没有说过是明成国偷走的，而且我也不知道王冠的下落，所以一直在寻找。”
<br/>　　国王忽然觉得，楚涵是不是记性不好。
<br/>　　楚涵看着她：“我的推断如果没错，王冠就在明成国，你要去夺回来吗？还有你的王冠上，是否有什么不一样的东西？”
<br/>　　国王一惊，愣愣的看着楚涵：“有颗蓝宝石……”
<br/>　　楚涵终于笑容灿烂了起来：“如果我帮你把明成国灭了，你能不能把那颗蓝宝石送给我？”
<br/>　　国王立即摇头：“那是天运国成立之初，就镶嵌在王冠上的，是天运国昌盛的象征！”
<br/>　　楚涵笑容带着些深意：“天运国的昌盛是靠蓝宝石？那你王冠消失的这二十年，对比你一直拥有王冠又如何呢？”
<br/>　　国王愣住，这二十年，是天运国发展最好的一段时间，壮大至让这片大陆的国家都忌惮。
<br/>　　楚涵继续道：“一个国家的繁荣昌盛与宝石有什么关系？重要的是坐在王位上的人，脑子里在想什么。”
<br/>　　国王不再说话，楚涵的到来，比白允儿给她造成的冲击大上了好几倍！
<br/>　　近乎是颠覆了她整个人生！
<br/>　　“带上你的王袍，我们大概很久都不会回来了。”楚涵淡淡笑着开口。
<br/>　　国王没有问为什么，将她的王袍打包背在了背上，就这么随着楚涵离开王宫。
<br/>　　她并不是没有武力，只是相比较于国师弱了太多而已，但比起一般的士兵却强上不少，在地球上的末世纪元里，相当于新人类五阶。
<br/>　　这样的武力当国王可以，但想要当首领却有些牵强。
<br/>　　不过楚涵却认为，足够了！
<br/>　　一对一的魔鬼训练当天夜里就开始，国王近乎不吃不睡，跟着楚涵徒步翻山越岭，从王宫到山林，楚涵不再给她任何便利，全部让其以最快的速度狂奔。
<br/>　　甚至他还给国王搞来了一柄锋利的剑，让国王一路保持不变的直行，前方有树就砍，有山就爬，有河就渡，有岩石就劈。
<br/>　　总之不管前方挡路的是什么，统统都将其毁灭的一往无前！
