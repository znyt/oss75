　　法相带路前往后山的同时，心中困惑不已，后山可是天音寺的禁地，哪怕是他，没有恩师的允许，也不得擅自进入，恩师怎会让慕清霄两人前往。
<br/>　　当然，困惑归困惑，他也没有特地去问，而是老老实实的带路。
<br/>　　慕清霄百般无聊下，倒是开口问道：“法相师兄，这无字玉璧究竟所在何处，从何而来？”
<br/>　　法相边走边笑道：“这说起来倒是话长了，无字玉璧何时出现，自然是无人知晓，只知道千年之前，天音寺创派祖师还是个行脚僧人的时候，四方云游，有一日不知怎么，误入须弥山崇山峻岭之间，竟是迷了路，再也无法走出去。”
<br/>　　“无奈之下，祖师便在这山林之间乱走，也是天生佛缘，竟然被他看到一片光滑如玉一般的石壁，那个时候，祖师困倦不堪，便在这无字玉璧下歇息。”
<br/>　　法相说道这里，顿了一下，继续说道：“传说那位祖师在那无字玉璧之下坐了三日三夜，不知怎么，竟然从最初的难以入定，心安而神定，进入了我佛门之中大圆满之境地，三日之后，他竟在这无字玉璧中顿悟了佛理，此外，更传说......”
<br/>　　法相转过头来向慕清霄神秘一笑，道：“更传说，那位祖师也就是在那无字玉璧下，领悟出了我天音寺世代相传下来的无上真法大焚般若，由此奠定了我天音寺一脉在天下修道中的地位。”
<br/>　　慕清霄微微点头，这些他自然都清楚，他想知道的是，这些石碑和玉璧，是否与天帝宝库的主人有关，而不是老僧的经历。
<br/>　　法相心细，将慕清霄的神情变化看在眼里，心中虽然惊诧，也不言语，至于跟在身边的陆雪琪，一直保持着一副冷若寒霜的模样，唯有看向慕清霄时，目光温柔。
<br/>　　三人顺着山路又走了小半个时辰，在崇山峻岭间曲折前行，不知不觉已将天音寺远远抛在了身后。
<br/>　　慕清霄也没想到天音寺后山山脉地势居然比想象中还要广大许多，只见周围峰峦叠翠，山风徐来，一路上或奇岩突兀，千奇百怪，或有断崖瀑布从天儿落，轰鸣声绕耳不绝。
<br/>　　这一路走来，只觉得一时心胸开阔，看望身边远近美景，倒也不觉得烦闷。
<br/>　　片刻后，只听见法相道：“前面就是无字玉璧所在了。”
<br/>　　慕清霄向前看去，却只见前方依旧是山路蜿蜒，路旁一边是茂密树林，另一边生着杂草荆棘，三尺之外便是一断崖处，哪里有法相口中所说得，高逾七丈的无字玉璧。
<br/>　　法相微笑，向前走了几步，来到那断崖之上，回首道：“就是此处了。”
<br/>　　慕清霄牵着陆雪琪来到他的身旁，举目望去，只见这断崖下雾气弥漫，如波涛翻滚，涌动不息，视线内是一个山谷模样，而远处隐隐能看见有模糊的山影，却都是在十分遥远的地方。
<br/>　　慕清霄凝神思索，看向法相，道：“难道就在下方的山谷中。”
<br/>　　“没错，就在你我的脚下了，我们下去吧。”说着，法相纵身跃下，慕清霄则搂着陆雪琪，向下飘然落去。
<br/>　　越是往下，雾气越是稀薄，片刻后，脚下景色顿时清晰起来，视线中乃是一面小小的石台，颇为光滑，周围有三丈方圆，树木稀疏。
<br/>　　慕清霄搂着陆雪琪飘然落地，道：“请问法相师兄，那无字玉璧在何处？”
<br/>　　法相微微一笑，道：“再过片刻，慕师兄就能看到了。”
<br/>　　慕清霄微微点头，就在这时，天上隐隐传来一声锐啸，是风声或兽嚎，穿云透雾而来，紧接着一束耀眼光辉，直接将浓雾撕开一道裂缝，倾洒在两人的身上。
<br/>　　慕清霄面色如常，抬头望去，只见山谷之间异声不断，似奔雷起伏，那片浓雾之海陡然起了波涛，随即出现越来越多的裂缝，浓雾也越来越薄，透出一束又一束光辉。
<br/>　　面对这天地异象，慕清霄注视良久，心中并无波澜，比这恐怖数百倍的异象他都见过，小小天地异象，很难让他心头荡起波澜。
<br/>　　片刻后，浓雾飘散，光辉洒下，瞬间天地一片耀目光辉，让陆雪琪与法相两人无法目视，慕清霄则嘴角勾起一丝笑意，因为无字玉璧不是从天而降，而是如波纹一样逐渐显现出来。
<br/>　　周围肯定有阵法，只是手段极其高明，没有留下一丝痕迹，否则他不会看不出来。
<br/>　　也就是说，刚刚他们所看到的，不是天地异象，可能是无字玉璧主人所设下的幻境，任何来此的人都能够看到。
<br/>　　就在他的身前，那石台后面，断崖之下，一片绝壁如镜，壁纸垂下，高逾七丈，宽逾四丈，山壁材质似玉非玉，光滑无比，倒映出天地美景，远近山脉，都倒映在这玉璧之中。
<br/>　　法相并不是第一次见无字玉璧，饶是如此，心中依旧惊叹，道：“慕师兄，恩师让你随意参悟，我在寺中还有要是处理，明日此时在过来。”
<br/>　　说罢，法相纵身一跃，向断崖上而去，慕清霄微微一笑，一天的时间，足以参悟。
<br/>　　等法相的身影消失，慕清霄就地盘坐而下，陆雪琪也盘坐而下，两人同时开始参悟。
<br/>　　片刻后，天际忽然响起一声惊雷，天空也黯淡下来。
<br/>　　四方风云滚滚而来，在无字玉璧光滑玉璧之上，从上到下，一点点如深深镂刻一般，现出一排大字，除此之外，更有无数金色古拙难懂的字体，如沸腾一般在玉璧上闪烁跃动，令人眼花缭乱。
<br/>　　而那一排大字特别的清楚，赫然正是——天地不仁，以万物为刍狗！
<br/>　　慕清霄心神停留在无字玉璧上，金色大字化作光芒莫入他的脑海中，系统提示声再次响起：“叮，成功收集五卷天书，任务完成4/5。”
<br/>　　慕清霄依旧盘坐在地，嘴角不动声色的牵起一丝笑意，如此一来，五卷天书就只剩下天帝宝库中的最后一卷了。
<br/>　　天书到手，慕清霄并没有起身，而是开始参悟无字玉璧上，古拙难懂的金色字体，也就是天地间的至理，这可是非常难得的机遇，他自然不会放过。
