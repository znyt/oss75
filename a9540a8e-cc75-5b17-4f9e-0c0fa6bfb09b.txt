　　叶少阳故作轻松地笑道：“老爸，你哪里是白发，你三十多岁就死了，头发比我的还黑。”
<br/>　　叶兵狠狠瞪了他一眼，又叹了口气，道：“你是没当过爹，不知道这种感受！”
<br/>　　叶少阳嘿嘿笑着，“老爸，我知道啦，你放心，你儿子我福大命大造化大，我死不了的，你就看着吧，说不定一两年后，我就给你抱个孙子回来！”
<br/>　　“那我就感谢八辈祖宗了，我今天找你，就是想跟你说这件事，你打算什么时候结婚？”
<br/>　　叶少阳张大嘴，半晌说道：“爸，人家都说当父母的喜欢催婚，我今天真的信了，你这都成了鬼了，居然还惦记这件事！”
<br/>　　叶兵劝说了一番，叶少阳敷衍地听着，最后叶兵自己觉得没意思，瞪了他一眼道：“行了，你这左耳朵进右耳朵出的，说你没用。”
<br/>　　叶少阳道：“老爸我认真跟你说个事，不如你不要在阴司呆着了，你跟我去青冥界，你看到那小子没有，对对就是那个长的像斯文败类的，他现在是空界联军元帅，那是我过命的弟兄，你不然到他那里，当个太上皇什么的，安心呆着，啥也不用管了多好，省的在这当个芝麻小官。”
<br/>　　叶兵严肃地说道：“这如何使得，我前世福缘浅薄，按说连这小官都当不起，我如今在这里做小吏，也是积累阴德，我没那么深厚的福缘，你说的那个，我受不起，反而要生出祸端。我在这里也挺好，而今也不在轮回司当差了，你若想我，随时能来看我。”
<br/>　　叶少阳只好答应，往回走的时候，心里其实美滋滋的，在人间的时候，经常听到别人被家长催婚，提起来都是烦不胜烦，如今他自己也第一次体验到了这种感觉（青云子从不催这个），内心还是很满足的，就算是烦恼，也是幸福的烦恼。
<br/>　　叶兵送他到高台下，又跟大伙都打了招呼，依依不舍地送他们走。
<br/>　　“老爷子，我是风之谷的，我一向在鬼域混的，以后有人欺负你，报我的名字！”小马拍着胸脯，被叶少阳拉扯着上了高台。
<br/>　　大家好奇地趴在井研上往下看，井水中是黑洞一般的漩涡，连接着虚空。四宝爬上井研就要跳下去，叶少阳死命抱着他，惊叫：“你干啥！”
<br/>　　“额，不是跳下去回去吗？”
<br/>　　“你疯啦，这海眼现在跟轮回井一样，跳下去就进入轮回了！”
<br/>　　四宝这才回过神来，吓出一身冷汗。
<br/>　　“这里才是还阳的路！”叶少阳白了他一眼，对着父亲照了照跳进了豫州鼎中，海眼激活之后，豫州鼎与海眼演化阴阳，鼎中形成了一个常开式的虚空裂缝，一阵穿梭之后，回到了人间。
<br/>　　鹰潭的快捷酒店，房间本来就不大，大伙都来之后，一下子就人满为患了，大伙只好都坐在床沿上。
<br/>　　从阴间回来，大家才真正觉得任务完成了，本来是很圆满的，但因为秋莹的死，给大家心头都蒙上了一层愁绪。
<br/>　　“别的事也没有了，你们都去空界吧，大家都当心点，有什么情况互相再通知。”都是自己人，叶少阳也不客气，抬头看到杨宫梓和小马，问他们有什么打算。
<br/>　　“我们当然还是要回鬼域的，不过我们要去找下道风，正好跟军师一起。”杨宫梓道。
<br/>　　小马道：“狗蛋，我不走了，反正风之谷也没啥事了，我就跟军师他们去空界呆着吧。”
<br/>　　杨宫梓马上说道：“不成，风之谷的力量本来就被削弱了，你要是再走，那就更不成了，你必须跟我回去。”
<br/>　　小马跟她软磨硬泡起来。
<br/>　　“少阳，你有什么打算？”林三生正色问道。
<br/>　　“暂时没什么，快过年了，好好过个年吧，我得先回去给秋莹做一块牌位，到时候你们记得都来祭拜，行了你们走吧，我想睡一会。”
<br/>　　林三生打开了虚空裂缝，大家互相道别，小九抓住叶少阳的手，用力捏了一下，说了几句关切的话，跟着大伙一起离开了。她是想留下多陪叶少阳一下的，但大伙都走，她也不好搞特殊。
<br/>　　四宝、吴嘉伟和瓜瓜三人留下来。
<br/>　　叶少阳让他们去开房间，总不能三个人挤在一起。
<br/>　　经历过这么一场大战，两人也都很累，虽然之前打坐吐纳过，也只是恢复伤势，人反而更疲惫，也没多说，各自开房睡觉去了。
<br/>　　第二天，一行人乘坐高铁回到石城。
<br/>　　老郭来车站接他们，之前在电话里，他已经知道了秋莹的死讯，见面也是十分伤感。
<br/>　　“我昨晚找你说的事，准备得怎么样了？”
<br/>　　“好了，到家里看吧。”
<br/>　　默默回到叶少阳家里，老郭从包里拿出两个牌位，一个是吴嘉道的，一个是秋莹的，说是昨晚自己连夜赶制的。
<br/>　　牌位八成是檀木的，拿在手中沉甸甸的，还带有一点木质的清香。
<br/>　　吴嘉伟看到弟弟的牌位，惊道：“这是做什么。”
<br/>　　“你不是看到了吗，供牌位。”叶少阳道，“昨晚我就想到了，他俩都是我们捉鬼联盟死难的兄弟，我们应该做点什么。”
<br/>　　老郭已经清理出一个床头柜，搬到客厅来，摆在客厅的东北角，紧紧靠墙，高矮大小都挺合适，于是又从包里拿出两只香炉，摆在两只牌位前面，自己上去点香，一边三只，都躬身祝祷了一番，才把香插进去。
<br/>　　叶少阳也十分严肃地过去上香。
<br/>　　他们是法师，知道魂飞魄散之后，生灵再没有什么了，供奉牌位的作用不大，但按照传统来说，只要祭拜的人诚心诚意，供奉牌位的确能起到一些祈福的作用，这是愿力的体现。
<br/>　　很多人不理解为什么祭拜要用上面捏的猪头牛头、香蕉苹果之类，死人当然是不会吃的，但要的就是这种仪式感，显得祭拜之人诚心并严谨，这才能产生愿力。
<br/>　　吴嘉伟最后一个上香，看到自己弟弟的牌位，思念之情油然而生，又看看秋莹的牌位，握紧双拳说道：“秋莹，虽然你加入捉鬼联盟时间不长，大家交往不深，但一样情同手足，我们一定会为你报仇！”
