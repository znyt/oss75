　　第750章
<br/>　　楼船水声夜色，虽然没有到八月十五正日子，但月色已经颇为可观。
<br/>　　船停泊在岸边，岸上船上灯火通明，月色歌声两相宜。天色暗下来的时候，收到请柬的客人便络绎不绝到了。
<br/>　　有人坐轿，有人骑马，有人乘船……但凡能得到请柬的人，多多少少有些身份。而外围还围着许多人，大半是没收到请柬，却想过来凑凑热闹的江湖豪客。
<br/>　　万一被那路诸侯瞧上，说不得便有封王拜相的路走。
<br/>　　据说，百里胜包下了满庭芳所有歌姬。那些娇媚娘子，凡夫俗子便是见一面都难得，可百里胜一声令下，她们便不得不听从吩咐。
<br/>　　时运造化，七分天定，三分人为，谁又能想到，当初沦落成山贼的百里胜，今日竟是一人之下万人之上的势力。
<br/>　　满庭芳的花魁唤作玉雀，据传说此女是江南来的，江南一地灵气都在她身上。生得玲珑体貌，更妙一双三寸金莲，尤擅盘中舞。舞时手中捧一盏酒，裙摆纷飞，上下颠倒，盏中酒一滴不落。传说，饮此酒三盏，比赛过云上仙人。
<br/>　　跳的了舞，谈得了诗，唱得了风华雪，说得了柴火、黑狗和姑娘。
<br/>　　当然，传说也仅仅是传说。关于这玉雀，听说得多，见过得少。到现在，也没听说那个浪荡公子得入罗帏。
<br/>　　今日百里胜将她请过来，也仅仅是一句话的事。楼船外围了不少人，多是想着能见上玉雀一面，瞧他生得如何模样。
<br/>　　在所有人中，程大雷有些奇怪，他既没有骑马，也没有坐轿，而是和徐神机与刘发财两人走路过来。
<br/>　　徐神机算是没见过多少世面，对周围所见所遇大呼小叫。刘发财相比之下稍好些，倒不是说他有什么世面，只不过他行骗出身，装模作样还是有经验的。
<br/>　　程大雷上了楼船，也没人要他请柬。他那颗憨大脑袋就是最好的请柬，如今帝国不认得程大雷的人不少，但没听过他名字的人屈指可数。
<br/>　　楼船一共三层，程大雷本来只打算凑凑热闹，但上船后便有了想做的事。绕着船舱甲板转圈，想看这大船这么造起来的。
<br/>　　三层楼船，大概是帝国造船的最高水平。程大雷也是职业病发作，想知道这大船如何建造，如何行动，日后万一疆场上遇到，也知道如何应对。
<br/>　　瞧过一圈后便大失所望，这船不是帆船，是靠人力驱动。底层船舱诸多壮汉一起踩动转轮，方可行水。
<br/>　　看上去很气派，不过用处并不大。如今帝国并没有拿得出手的远程武器，倾一国之力造一艘大船，上百人一起驱动，气派是有了，但与国何益，与民何利。
<br/>　　不知不觉，程大雷上了楼船顶层，隔着窗远远望去。
<br/>　　两岸灯火连绵，若两条细细长长的火蛇，偶尔有歌声乐色吵闹声。这次藩王入京，大家都带了不少银钱，长安城的百姓也能赚一笔。
<br/>　　可身处长安的人，为一顿酒一餐饭而喜而悲，却又谁知道帝国已摇摇欲坠，危在旦夕。
<br/>　　程大雷长出一口气，收回视线看着楼船上，想要瞧李行哉在何处。
<br/>　　李行哉也是好凑热闹的性格，碰到今天的事情不会不来的。遇见他，二人也好搭上线，自己就可以趁机见到李婉儿。
<br/>　　屈指算来，不见面也有些时日，不知她今朝是怎地模样。
<br/>　　程大雷看着楼下时，也有人在岸边静静看着他。
<br/>　　百里胜身旁立着一白衣书生，四十岁左右年纪，长须垂胸，面目有些儒雅。
<br/>　　“白先生，今日天下藩王大多聚在此地，以你所言，日后谁能成事？”百里胜淡淡问道。
<br/>　　其实这次赴京的藩王毕竟是少数，多是他们的子弟或者信得过的人。但多多少少，能看出他们主子的脾气。
<br/>　　白衣谋士名唤白山川，他目光看着最高处的程大雷。
<br/>　　“此子日后必然是帝国的心腹大患。”
<br/>　　“何解？”
<br/>　　“场上诸多人，有人爱酒，有人爱诗文，有人追逐敬酒的歌姬，唯独此一人，专注在这艘楼船上。如此心胸，一见便知高下。”
<br/>　　百里胜点点头，眼神变得有些沉重。
<br/>　　他忽然笑笑：“先生请了，今日戏台子已经搭好，我们该请正主出来唱戏了。”
<br/>　　“侯爷请。”
<br/>　　二人并肩踏上楼船，程大雷独坐三楼饮酒，只是瞧着船上形形色色的人。
<br/>　　“呔，此山是我开，此树是我栽，要想从此过，还不分我一碗酒。”
<br/>　　身后忽有一道声音响起，程大雷并没有回头。道：“酒都在这里，怕的是你喝不尽。”
<br/>　　哈哈一阵笑声，李行哉来到程大雷身边，自顾自坐下，又自顾自给自己斟满酒，放在面前一饮而尽。
<br/>　　二人相对而坐，目光望着窗外的月色水色，听着耳边歌声乐声，沉默半晌之后，李行哉方才开口。
<br/>　　“程大雷，你可曾想过，你我竟也能坐在长安高处，饮酒话江山。”
<br/>　　程大雷摇了摇头，二人关系不错，但出身却截然相反。一个高高在上帝国六皇子，出生便是奔着治天下去的。一个出身草莽，睁开眼第一件事想的便是如何填饱肚子。
<br/>　　“怎么你一个人来的？”程大雷问。
<br/>　　“怎么可能是我一人，路上不太平，随行还有几个护卫，我可没有你的本事。”
<br/>　　程大雷撇撇嘴，忍住抽他的冲动。李行哉明知自己问的不是这些，反而顾左右而言其他，可话程大雷也不好挑明。
<br/>　　月色比水色更浓，李行哉锦衣华服，相貌标致。只是一条腿毕竟有些旧疾，想来也是美中不足，令人唏嘘。
<br/>　　二人有一句没一句的闲聊着，李行哉的目光一直瞧着楼下，忽然道：“瞧，那就是玉雀。”
<br/>　　程大雷顺他手指的方向看过去，虽然隔得较远，看不真切，但也能瞧出几分妩媚。
<br/>　　“啧，果然是个尤物。”
<br/>　　“程大雷，你可不要乱动心思。”李行哉道：“这样，你帮我把她搞到手，我也同样送个人给你？”
<br/>　　程大雷顿了顿，竟有些不知该说什么。
<br/>(本章完)
