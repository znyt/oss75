　　“一言难尽。”
<br/>　　叶少阳倒了水，回到房间里，结果刚坐下，碧清就跟了进来，问他有关山海印的事。
<br/>　　“你居然为了这个放弃看电视，好神奇啊！”叶少阳不敢相信。
<br/>　　“放广告了，我没会员，跳不过去，你给我冲个会员？”
<br/>　　“想得美。”
<br/>　　叶少阳喝了水，往床上躺下去。
<br/>　　瓜瓜也来打听山海印。
<br/>　　叶少阳抬头看了一眼窗户上的符纸好好的，这才压低声音，把徐福对自己的暗示讲了一遍，然后得意洋洋地看着瓜瓜，道：“猜猜啥意思。”
<br/>　　瓜瓜知道他在显摆，歪着头冥思苦想起来，“物是人非……穿越……什么意思啊，这算哪门子暗示啊。”
<br/>　　“但是，我猜到了他要说什么。”
<br/>　　“好啦，老大你智商好不容易在线一回，就别秀了，快告诉我什么意思啊。”
<br/>　　叶少阳白了他一眼，道：“物是人非的意思，就是说东西还是那个东西，但人不是那个人……配合‘穿越’这两个字，这就让我有了一种猜测……瓜瓜，假如你是徐福，你要留一件东西给我，一边提防被别人找到，一边还要确保我能发现，还没法直接告诉我，换成你会怎么做？”
<br/>　　瓜瓜怔怔地想了半天，道：“藏在什么隐秘的地方？”
<br/>　　“他跟我不熟，他能想到的地方，别人也一定能找到。”
<br/>　　瓜瓜又想了想，摇摇头，“这么多限制，这也太费脑子了，想不到啊想不到。”
<br/>　　“一般人都想不到，但徐福想到了，他把山海印放在一个永远不会有人找到的地方……就算酆都大帝和地藏菩萨来都找不到。”
<br/>　　“快说嘛，别卖关子啦！”
<br/>　　叶少阳看着他，缓缓说道：“他把山海印藏在未来了。”
<br/>　　“未来？那是哪儿？”瓜瓜一时没反应过来。
<br/>　　“就是未来。”叶少阳见他还是不懂，拿出手机看了下日期，“举个例子啊，今天是元月十八号，星期五，但这是我的时间，徐福有山海印，我们假设他在被抓住之前，穿越到了元月十九号，把山海印放在我房间里……那么到了明天，我的房间里就会出现山海印。我这么说你懂吗？”
<br/>　　瓜瓜一双眼睛瞪的老大，缓缓点头，“我听是听懂了，但是……你之前穿越过一次，然后回到这里来，很多东西都没变，你也说了，有那什么平行宇宙之类的，不同的时空之间没有因果关系……如果徐福改变了你的未来，那也是另一个空间的你，你该是什么样，还是什么样啊，好像是这样？”
<br/>　　叶少阳倒是没想到他记得这么详细，沉吟道：“我经历的是这样，但我只体验了一次，肯定是片面的，这山海印在他手中这么多年，肯定早就摸透了，或许有我们不知道的秘密，万一，他有办法从若干平行宇宙中找到跟我们有因果关系的那一个呢？”
<br/>　　瓜瓜怔住，不做声了。
<br/>　　“老大，这些……是不是你自己想出来的？”
<br/>　　“的确是我灵光一现想到的，结合他之前说的那番话，我觉得就是这样。徐福是个极聪明的，从他的角度，这是最好的办法，一来能让所有人都找不到山海印，二来，也能确保山海印能到我手中，这才是最重要的，他要是藏到珠穆朗玛峰山顶，倒是谁也找不到，但又到不了我手里，又有什么意义？”
<br/>　　瓜瓜看着他，“所以……”
<br/>　　“所以我想不到他还有什么法子能把山海印送到我手中。”
<br/>　　瓜瓜似乎被他说动，喃喃道：“不过前提是，他有办法做到你说的这样。”
<br/>　　“是的。”
<br/>　　“那我还有个疑问，老大，他既然能穿越到未来，是不是已经看到一切的结局了，那……他为什么还要做这一切呢，反正，对于未来而言，这一切都已经发生过了。”
<br/>　　叶少阳缓缓摇头，“未来是可以改变的，我觉得，人的每一次选择，都会生出不一样的时空，结局也完全不同，而且，徐福跟我说过，他没有去过未来，因为他不想知道未来会怎样，未知，是奋斗的动力。”
<br/>　　瓜瓜似懂非懂地点了点头，突然撒娇地抱住叶少阳的胳膊，“那我也不管未来会是什么样，反正，跟着一直你混下去就好了。”
<br/>　　叶少阳心中感然，笑着摸了摸他的脑袋。
<br/>　　“对了老大，那如果徐福真的是用这种法子把山海印留给你，怎么办？”
<br/>　　“什么怎么办，等着呗。早晚一天会冒出来。”
<br/>　　瓜瓜发了一会呆，默默说道：“山海印真的这么好吗，连地藏菩萨都要来抢。”
<br/>　　以前，叶少阳也是这么想的，后来别人都来抢，他不由仔细琢磨了下，这才真正明白山海印的神奇之处：如果能无限穿越时空，那它就是普天下最神奇的东西，就像游戏里的Bug一样。
<br/>　　谢雨晴帮叶少阳和陈悦在一起订了机票，第二天，一起飞往贵州，三个人，再加上一个不用买票的瓜瓜。
<br/>　　叶少阳还是第一次到贵州，本来以为贵州是南方，会暖和一些，结果正赶上这边下雨，下飞机就冷的很，好在三人都带了足够多的衣服。
<br/>　　谢雨晴有个表弟在贵州上学，来之前联系过，特意赶来飞机场接他们。
<br/>　　小伙子叫乌宏志，浓眉大眼的，二十出头，长得还挺帅的，人很开朗，特意找同学借了一辆车开过来。谢雨晴给他们互相简单介绍了下，就说叶少阳和陈悦是她朋友，瓜瓜没有显形，也省得介绍了。
<br/>　　小伙子是学舞蹈的，怪不得叶少阳看着他举止挺妖娆的，身段也不错，是现在的小姑娘喜欢的那种小鲜肉。
<br/>　　谢雨晴跟表弟聊天，叶少阳就趴在车窗上看风景。
<br/>　　市区的感觉，全国各地都差不多，就是在高楼大厦的夹缝中，能看到一些青山的影子，到底是山区。
<br/>　　乌宏志作为地主，把住宿什么的都安排好了，先送他们去酒店，让他们休息了一个下午，晚上一起去吃饭，找了个口碑好的老店，吃了当地特色的酸汤鱼和肠旺面，席间叶少阳一直注意着陈悦，吃到肠旺面的时候，叶少阳见她发呆半天，问她怎么回事。
