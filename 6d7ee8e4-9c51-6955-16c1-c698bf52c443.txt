　　这一剑，快，快到没有任何人足以看清。
<br/>　　陆雀，也是如此，以他的实力，居然都无法看出，楚风眠的这一剑，到底是从何而来。
<br/>　　楚风眠的九域剑术，虽然只有一种，却是变化千万，可以用力破敌，也同样足以用速度称霸。
<br/>　　毕竟是将曾经剑道之主一生领悟的剑术，全部融入到了其中。
<br/>　　一剑而出，千变万化，这一剑，便是快到极致的一剑。
<br/>　　比起世间的任何一种剑术，都要快。
<br/>　　这一剑斩出，无穷剑气，席卷整个生死台，简直是任何人，都不要想要躲开。
<br/>　　所有剑气的目标，都是陆雀，万道剑气，几乎是同时，向着陆雀斩杀而去。
<br/>　　“狂妄小辈！你的剑术在高又如何？没有实力，也休要伤到老夫！”
<br/>　　陆雀当空一震，这金光大网便是猛然收回，化为了金罗殿的样子，同时陆雀的身躯，也是一下子钻入到了这金罗殿之中去。
<br/>　　来抵挡着楚风眠的剑术。
<br/>　　无数剑气，斩在金罗殿上，都是嘶嘶作响，却是无法打破金罗殿的防御。
<br/>　　“这金罗殿，实际上就是一个乌龟壳，只要是身处其中，便是立于不败之地。”
<br/>　　一些围观武者，看着金罗殿，开口道。
<br/>　　“楚风眠的剑，的确是快到极致，但是这剑，却是不可能破的开金罗殿的防御，如此一来，楚风眠迟早有着灵力消耗殆尽的时候，这一战，还是陆雀要胜。”
<br/>　　“躲在一个乌龟壳里，就以为可以挡住我的剑了？”
<br/>　　楚风眠看着陆雀的举动，突然冷喝一声道。
<br/>　　“在我的剑面前，你就算是躲在天涯海角，也要死！”
<br/>　　“剑！”
<br/>　　楚风眠一声长啸，他的身躯猛然爆发起来，身体上的青筋流露，他身上的剑意，一万两千九百六十滴远古战龙精血，也是全然凝聚出来。
<br/>　　令他身上的力量，开始节节攀升起来，无数力量，全部都蓄积在了楚风眠手中的祖龙至尊剑上。
<br/>　　“楚风眠是想要干嘛？”
<br/>　　“难道他想要破开这金罗殿的防御？”
<br/>　　“金罗殿最为强大的，便是防御，这防御，让生死境巅峰的强者来，都不一定能破开，这小子是疯了不成？”
<br/>　　“鲁莽！太鲁莽了！这样消耗灵力，只怕是一会灵力耗干，一会灵力耗感，他不可能跟陆雀抗衡！”
<br/>　　众人的声音未落。
<br/>　　楚风眠的剑便是再次动了。
<br/>　　这一次，他的剑却是变化了。
<br/>　　同样是一种剑术。
<br/>　　但是这一剑的力量，却是跟刚刚，截然不同。
<br/>　　刚刚楚风眠的那一剑，是绝对的速度之剑，以快到极致的速度，斩出的一剑。
<br/>　　而如今楚风眠的这一剑，却是截然不同，是纯粹的力量，凝聚出的一剑，无缝重剑！
<br/>　　在这一剑的力量下，几乎是找不到任何一点的破绽，纯粹的力量一剑。
<br/>　　这两种可以说是完全不同，甚至是截然相反的剑术，居然是从楚风眠的手中用了出来，还是动用的一种剑术，这令在场无数武者，都似乎有些无法理解其中玄妙。
<br/>　　“躲在一个乌龟壳里，我就杀不了你？”
<br/>　　楚风眠看着金罗殿，冷笑一声道。
<br/>　　“老匹夫，你就看看吧！我看你能躲到什么时候！”
<br/>　　说话之中，楚风眠手中的剑落下了。
<br/>　　这一剑落下，简直是天地都在颤动，这是一种霸道到极致的力量。
<br/>　　一股股的灵力，从剑锋上迸发了出来，单单是其中余波，都足以轰杀无数人。
<br/>　　甚至是周围的空间，都出现了丝丝裂缝。
<br/>　　这空间，也只有接近生死境巅峰的人物，才可以借助一些法决打破，像是楚风眠这样单纯是以力量将空间都给震破的，简直就算是生死境巅峰的大人物都做不到。
<br/>　　楚风眠的这一剑，竟然是惊骇之际。
<br/>　　之前那些还觉得楚风眠是自不量力的人，如今全部闭嘴了。
<br/>　　这种实力。
<br/>　　什么陆画。
<br/>　　什么夜天君。
<br/>　　在楚风眠的面前，简直都无法相提并论！
<br/>　　“这，这……”
<br/>　　白萱看着楚风眠，都是愣住了。
<br/>　　她跟楚风眠接触的时间很长，所以她以为，她已经是知晓了楚风眠的手段。
<br/>　　可如今这一剑，简直是超出了她的认知。
<br/>　　白萱看向楚风眠，简直是如同看向一个黑洞一般，仿佛是永远看不见底。
<br/>　　一尊神力境极限的武者，跟生死境高阶，逼近生死境巅峰的人物战斗。
<br/>　　这要是说出去，都没有人会相信，这简直就是一只蝼蚁再跟大象对抗一样。
<br/>　　而如今，现在反而是陆雀，被逼的躲在了金罗殿中，不敢出来，而楚风眠占据上风，居然还要打破金罗殿。
<br/>　　这一切都给颠倒过来。
<br/>　　“疯了！疯了！这种力量还是人吗？”
<br/>　　一尊种子弟子愣住了，几乎呆滞。
<br/>　　他无论如何也想不到，一尊神力境极限的武者，足以可以达到如此实力，跟他相同的境界，但是他在楚风眠的面前，却是都连蝼蚁都算不上。
<br/>　　“剑之力！力破群山！”
<br/>　　楚风眠长啸一声，他剑锋的力量，变得更加强大了许多。
<br/>　　在这力量蓄积完成的一刻，猛然之间，轰然落下。
<br/>　　这一道剑气，足有万米之高，仿佛是任何人，只要敢挡在这剑气之前，都会被毁灭。

<br/>　　这纯粹力量的一剑，直接是砸在了金罗殿上。
<br/>　　只见金罗殿上的无数禁制，阵法，都被疯狂的催动起来，以来抗衡楚风眠这一剑的力量。

<br/>　　但是在刹那只见，上面的无数禁制，阵法，却都是被全部毁灭，一个个全部破碎。
<br/>　　这金罗殿，都在剧烈的颤动起来，几乎快要散架。
<br/>　　“这是什么？这力量为何会如此强横？此人到底还是不是神力境的武者？这种力量简直足以媲美生死境巅峰的人物了。”

<br/>　　这种情况，陆雀想破脑袋都想不明白，楚风眠的力量怎么会那么强大。
