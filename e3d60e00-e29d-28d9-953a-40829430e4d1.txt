　　阁楼中。
<br/>　　君无惜和风少羽四人都在，四人躲在阁楼中根本不敢出去。
<br/>　　因为他们身处于神武学府中，外面全部是强大无比的气息，个个都让他们胆颤心惊。
<br/>　　见到苏莫出来之后，几人立刻围了上来。
<br/>　　“十一师弟，这里是什么地方？我感觉外面好多恐怖的强者！”
<br/>　　风少羽震惊的问道，他从来没有感受过如此强大的气息，几乎每一个都比武帝境的巫森，强大数倍不止，甚至强大百倍。
<br/>　　“这里是神武学府！”苏莫道。
<br/>　　“神武学府？”
<br/>　　四人闻言一怔，随即云非震惊的问道：“那你……已经离开了苍穹世界？”
<br/>　　其实，先前四人已经猜测到了，此地便是苍穹世界之外的世界了。
<br/>　　因为苍穹世界之内，根本不可能有这么多强者。
<br/>　　但是，现在听到苏莫亲口说出，四人还是不免心中震惊。
<br/>　　苍穹世界根本没有什么神武学府，那么此地定然是在苍穹世界之外了！
<br/>　　“嗯，我已经离开了苍穹世界，来到了古灵星河，古周星！”苏莫点了点头。
<br/>　　风少羽四人闻言，心中充满了震撼，古灵星河？古周星吗？
<br/>　　四人神情怔怔，苏莫居然出来了，怎么逃出来的？
<br/>　　巫森呢？
<br/>　　那些试炼者呢？
<br/>　　“十一师弟，巫森呢？那些试炼者呢？”花碎羽问出了几人都想知道的问题。
<br/>　　“巫森已经被我杀了，巫族试炼者，也被我杀的差不多了！”苏莫沉声道。
<br/>　　“什么？”四人闻言，再次心中一震，脑中掀起惊涛骇浪。
<br/>　　巫森死了，四人没有任何意外，但试炼者都被杀的差不多了，那就恐怖了。
<br/>　　苏莫如今的实力，真的强大到了如此地步了吗？能斩杀众多试炼者？
<br/>　　“你们以后有什么打算？”苏莫叹了口气，没有多谈试炼者的事情。
<br/>　　四人闻言沉默了，他们对于外界一无所知，又能去哪里呢？
<br/>　　“要不，你们都加入苍穹神宫吧！估计二师姐和十师兄也很想再见到你们。”苏莫沉声说道。
<br/>　　“这……！”四人闻言沉默了，加入苍穹神宫吗？
<br/>　　他们虽然很想与安暖和天辰团聚，也很想在这个世界安稳下来，但加入苏莫的苍穹神宫，心中还是有些芥蒂。
<br/>　　因为，他们知道，皇甫擎估计已经被苏莫杀了。
<br/>　　他们没有能力阻止苏莫，心中对于皇甫擎均是有些愧疚，所以对加入苍穹神宫，亦是有些芥蒂。
<br/>　　皇甫擎虽然胁迫过他们，但终究是他们的师尊，培养了他们多年。
<br/>　　“以你们的实力，在古周星很难有所作为，并且危险重重！”苏莫也看出了几人的想法，开始劝解几人。
<br/>　　他不想让这曾经的四位师兄师姐，在这浩大的古周流浪，想给他们一个安稳的成长环境。
<br/>　　“加入苍穹神宫，你们才能安稳的修炼，追求武道的高深境界！”
<br/>　　“古周星很大，你们又能去哪里呢？”
<br/>　　“你们没有背景，随时可能遇到生死危机！”
<br/>　　苏莫不停的出言劝解，晓以利害，片刻之后，四人终于同意了，愿意加入苍穹神宫。
<br/>　　他们也知道苏莫为了他们着想，苏莫如此好意，他们再不同意，就是摆明了记恨苏莫了。
<br/>　　最后，苏莫将四人再次收进了玉镯空间中，准备带下次会苍穹神宫之时，将四人带回去。
<br/>　　随后，苏莫就在阁楼中修炼了起来，巩固刚刚突破的修为境界，同时，思索如何尽快的将武魂提升为战魂。
<br/>　　……
<br/>　　时间流逝，翌日中午时分，一身白衣的木璃导师来到了苏莫的的阁楼。
<br/>　　“这是你的身份牌，从现在开始，你就是神武学府南苑的正式弟子！”
<br/>　　阁楼大门之前，白衣飘飘的木璃，将一枚巴掌大小的身份牌，交到了苏莫的手中。
<br/>　　这是一块银白色的石牌，其上闪烁着星辰之光，颇为的璀璨。
<br/>　　苏莫接过身份牌，随意的打量一番，便收了起来。
<br/>　　“多谢！”苏莫向对方抱拳道谢，同时他也心中恍然，看来这辽阔的外府区域，还不是一个整体，既然有南苑，那肯定就有北苑，可能还要东苑和西苑。
<br/>　　木璃眼眸瞥了苏莫一眼，眸中闪过一丝异色，因为她发现苏莫居然突破了修为。
<br/>　　短短一天的时间，居然就从武皇境九重突破到了准帝之境，这个速度未免也太快了吧！
<br/>　　“导师让我告诉你一声，你如果想通了，随时可以前往内府火元峰！”木璃冷冷的说道。
<br/>　　木璃的心中，微微有些嫉妒，对于苏莫能享受到的待遇而有一丝嫉妒。
<br/>　　她作为外府南苑的导师，武尊境二重的修为，尊级下品炼器师，能得到火元圣师指导的机会都不多。
<br/>　　而苏莫，不愿意跟随火元圣师学习炼器也就罢了，火元圣师居然还给苏莫重新抉择的机会。
<br/>　　这真的是太气人了！
<br/>　　“呵呵！我会考虑的！”苏莫尴尬一笑，立刻便点了点头。
<br/>　　他的心中也是有些无奈，他是炼器师，柴管事也是炼器师，柴管事既然推荐他来此，想必就是让他跟随火元圣师学习炼气之道。
<br/>　　而他，居然拒绝了火元圣师，这就是有点对不住柴管事了。
<br/>　　不过，他别无选择。
<br/>　　“嗯！”木璃闻言，轻点螓首，随即转身离开。
<br/>　　“木导师稍等！”苏莫见此，急忙出声喊住了对方。
<br/>　　木璃停下了脚步，转头看向苏莫，她还以为苏莫已经想通了。
<br/>　　“木导师，你既然是导师，我能不能请教你一个问题？”苏莫沉声问道。
<br/>　　木璃点了点头，示意苏莫可以询问。
<br/>　　苏莫沉吟了一番，沉声问道：“木导师，我想问你，武魂的等级，对于战魂的影响有多大？”
<br/>　　这是苏莫目前最想知道的问题，因为若是武魂的等级对于战魂的影响不大，他就可以尝试进化战魂了。
<br/>　　“什么意思？”木璃目露疑惑之色，声音清冷的问道，她不太明白苏莫所问何意。
<br/>　　“我的意思是说，如果两种一模一样的武魂，等级不同，晋升为战魂之后，差别有多大？”
<br/>　　苏莫解释了一番，他当然不能说出自己的武魂可以升级，只能举了个例子。
<br/>　　木璃闻言一怔，她不清楚苏莫为何会问出如此简单的问题。
