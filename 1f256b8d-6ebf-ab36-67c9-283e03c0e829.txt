        &lt;h3&gt;第三卷第三百三十八章  安排&lt;/h3&gt;
<br/>
<br/>        陈子昂又去医院看望她三姐陈子澜去了。
<br/>
<br/>        大家都在为冯睿的离去忙碌着，悲哀着，陈子昂却在寻找着冯睿的死因，她害怕她进入死胡同，也害怕是从潜意识里认为是和夏颜有关系。
<br/>
<br/>        毕竟她和夏颜有过不愉快，虽然说不在意，但是谁又能抹杀过去呢？
<br/>
<br/>        “三姐，我看了一下二嫂的电话，夏颜大概有一个礼拜没有和二嫂联系，我觉得很奇怪，既然她没有和二嫂联系，为什么要跑去冯家寻找二嫂，还撒谎说和二嫂联系不上？”
<br/>
<br/>        “学过心理学吗？”
<br/>
<br/>        “看过几本书。”陈子昂苦笑一下，那几本书还是在张函的书房里看到的。
<br/>
<br/>        “单凭猜测或者主观意识是不行的，你要了解她的动机和心理，还有一种叫变态心理，然后去分析她的行为导向结果。”
<br/>
<br/>        “我从高中就认识夏颜，可以说她是很善于伪装的一个人，她永远都是一副美丽可爱大方善良的人，其实她另一面是跋扈不择手段，我就是害怕我的潜意识会误导我，所以我才来找你。”
<br/>
<br/>        “我们所有的事情都讲究一个证据，但所有的证据都会建立在分析和推测之上。有所怀疑，便有所行动。如果你觉得怀疑了，那么你就去找证据。”陈子澜看着陈子昂。
<br/>
<br/>        “可是我的时间不多，我要去甸城。”陈子昂叹了口气，虽然冯睿之前对她不怎么好，但是她却不愿意让冯睿睿走得不明不白。
<br/>
<br/>        “那就交给有关部门来做吧，你做你该做的事情，我会盯着点儿。”
<br/>
<br/>        “也只能如此了，三姐，你一定要快点儿好起来，要保重身体，姐夫什么时间回来？”
<br/>
<br/>        “我给你姐夫打电话了，他那边的事情还挺棘手的，说一两个礼拜才能回来。”
<br/>
<br/>        “姐夫现在这么忙了吗？”
<br/>
<br/>        “现在的行业整顿、纠风正纪、严查，一切都在规范化，相对来说他们会忙一些。”
<br/>
<br/>        “你们夫妻两这个职业啊，真是让人又敬佩又担心。”
<br/>
<br/>        “我们倒不用担心，我最担心的还是番禺，他那个工作啊！才是让人担心。”
<br/>
<br/>        “那三姐，你有没有后悔让番禺去那里。”
<br/>
<br/>        “后悔什么啊？虽然危险，可是我很骄傲，一般人谁能进到哪里，就光是选拔都让人层层蜕皮。”
<br/>
<br/>        “你也是我们家骄傲呢！你看你家里奖状那么多。”
<br/>
<br/>        “哎！我现在老啦！想想年轻的时候，你姐夫他们几个小伙子都近不了我身。”
<br/>
<br/>        “在我心中三姐永远年轻，三姐，我爱你。”陈子昂煽情了一下。
<br/>
<br/>        “我也爱你。”陈子兰微笑着回应陈子昂。
<br/>
<br/>        “三姐，我明天就要回北京了，我要提前回去，回去之后我就要去美洲，然后从美洲，直接甸城。”
<br/>
<br/>        “不等着二嫂下葬了吗？”
<br/>
<br/>        “不了，我的行程比较吃紧，我还是想按照我的行程走。”陈子昂原本想等着冯睿下葬再走，可是她觉得她有些心塞，不想在西城呆下去了。
<br/>
<br/>        “那你好好的保重身体，记得你的身后有我有你的兄弟姐妹呢，不要什么事情都要自己扛着。”
<br/>
<br/>        “我知道啦，我哪有自己扛着啊，以后有什么事情都交给你们处理，我就当个乖妹妹就好了。”
<br/>
<br/>        陈子昂走过去拥抱了一下陈子澜，有些不舍的离开。
<br/>
<br/>        在众多的姐妹中，陈子澜是最理性也是思维最缜密的一个人，在某些方面，陈子昂是会经常请教陈子澜的。
<br/>
<br/>        从医院出来，陈子昂又去了一趟公司。
<br/>
<br/>        易倾秋和陈天天偷听到陈子建他们的谈话，心里震惊的无以复加，夏颜的人生原来是盗窃而来，怪不得如此的针对她们的公司。
<br/>
<br/>        试想一想，一个冒牌货怎么会容忍正品出现在自己的面前。
<br/>
<br/>        对于易倾秋来说他们的陈总已经够神秘的了，这又增加了她的好奇心和敬佩心，能那么淡然的面对夏颜和冯睿是该有多么强大的心理素质？
<br/>
<br/>        百炼成钢，她是经过了多少次千锤百炼才达到现在这种状态的？
<br/>
<br/>        “倾秋，这几天辛苦你们了。”陈子昂看着李秀封装好的标书笑着说。
<br/>
<br/>        “辛苦什么啊，都是我们分内的工作，我有些担心，怕又和上几次一样的。”易倾秋毕竟年轻。
<br/>
<br/>        “不用担心，生意上的事情得失是很正常的，没有谁能一直站在顶端，总会被人超越或者是追赶，只要你始终保守着一颗积极向上的心，一直向前就好。”陈子昂微微而笑。
<br/>
<br/>        “我觉得这一次把握很大，有一个小细节，按照推测，这一次应该轮到许弋澄中标，这个标目前是最大的一个标，但是许弋澄已经被延城的两个项目拖住了，而且，延城用的是我们的软件。”李秀分析着。
<br/>
<br/>        “嗯，我也打听了一下，这一次报名特别严格，乔小东控制得非常严，对于项目都是专人负责，所有的人都不会知道谁会参加这个项目，也不会知道有多少家，这样对我们来说是有利的。”谭九儿甜甜一笑，只要不做手脚，他们的机会还是很大的。
<br/>
<br/>        “合着你们几个是对这个项目势在必得？”陈子昂看着谭九儿那灿烂的笑容，心里也微微一松，他们公司的人有一个共同特性---愈挫愈勇！
<br/>
<br/>        “感觉被人欺负的久了，想吐一口气！”易倾秋伸了伸腰。
<br/>
<br/>        “这就被人欺负久了？打起精神，估计你们还得熬一段时间呢，人生路长着，什么事情都会碰到哦。”陈子昂看了三人一眼站起来继续说：“我明天早上就走了，不陪你们战斗了，我静候你们的佳音。倾秋，我有话要对你说，你送送我。”
<br/>
<br/>        易倾秋随着陈子昂一起走出去，陈子昂边走边说：“倾秋，你帮我盯着点夏颜，尤其是看看她经常会开哪一种车。”
<br/>
<br/>        易倾秋中午和陈天天去了一趟警局，了解了一下冯睿的情况，心里一惊，莫非陈子昂怀疑夏颜？
<br/>
<br/>        “好的，我会注意的，有什么情况我会及时给你说。”
<br/>
<br/>        “你不用给我说，你给我三姐说，我把我三姐的电话给你，有什么事情你和她联系。我过几天就要去甸城了，所以这边的事情会顾及不到，你就多担承一些。”
<br/>
<br/>        “陈总，你放心吧，我会将西城分公司带好的，还有天天.......”
<br/>
<br/>        “天天和江俞轩的关系有点暖昧，江俞轩是有妇之夫，你有时间劝劝她，让她想好！别到最后让人伤的体无完肤。”
<br/>
<br/>        在回去的路上，陈子昂接到了守中的电话，守中告诉她，有人跟踪他们，并且想闯进他们家里。
