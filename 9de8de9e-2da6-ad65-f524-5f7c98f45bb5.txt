        &lt;h3&gt;第三卷第二百七十八章  犹豫的陈天天&lt;/h3&gt;
<br/>
<br/>        陈天天看着电脑里的论文资料，她是怎么也写不下去，心烦意乱中她将电脑扔到床上，电脑在床上翻了一个滚，然后掉到了地上，笔记本屏被摔裂了。
<br/>
<br/>        陈天天看着裂了的笔记本屏幕，脚踏上去，一用力，整个电脑限于黑屏之中，彻底的死机了。
<br/>
<br/>        她在和易清秋去参加夏颜晚宴的第二天下午被她的生母冯睿拦住，将她带到了冯睿的家里，她几乎没有到冯睿的家里来过，她知道冯睿是她的母亲，可是因为从小就没有和冯睿在一起，她的心里对冯睿是排斥的。
<br/>
<br/>        “二妈。”陈天天叫冯睿二妈。
<br/>
<br/>        “你应该叫我妈妈，而不是二妈，我才是生你养你的人。”冯睿看着陈天天，陈天天今年已经20岁了，一张略有点婴儿肥的鹅蛋脸，呵，是冉灵卉的基因，陈子建的种。
<br/>
<br/>        陈天天的丹凤眼里有讥讽有不确定，还有一种渴望，陈家的人都生着一双勾人的丹凤眼，无论是男是女，那双眼睛里就像是盛满星辰，让人一看就不由自主的陷进去了，冯睿看着陈天天的眼睛不由自主的又想到了陈子建，陈子建的那一巴掌打落了她的两颗牙，现在说话都不敢张口。
<br/>
<br/>        “二妈和妈有什么区别吗？”陈天天冷冷的问冯睿，生她是可以，养她？
<br/>
<br/>        “天天，你是不是在心里怪着妈妈，没有带你在身边？”冯睿拉着陈天天的手，带着陈天天在沙发上坐下。
<br/>
<br/>        “我从来都没有怪过你。我只知道我的妈妈是严慧芝。”陈甜甜的心里还是有些不舒服。这么多年以来冯睿从来没有管过她。现在拉着她的手想干嘛？
<br/>
<br/>        “天天你怎么能这样想呢？我当初生你的时候是迫不得已，你看看，你看看！”冯睿撩起了自己的衣服露出了曾经做过手术的伤疤。
<br/>
<br/>        那长长的丑陋的伤疤蔓延在冯睿的身上。
<br/>
<br/>        陈天天沉默着。
<br/>
<br/>        “天天，虽然说我没有带着你。但是你小的时候吃的喝的穿的都是妈妈给你的呀！陈家人那么穷，农民出身，他们哪来那么多钱养孩子的？当初要不是妈妈拿着自己和你爸爸的公资接济他们家，他们现在会是这样的地步吗？陈子霖、陈子昂他们就是没有良心的人，当初我和你爸爸拿着工资供着他们上学，供着他们开公司。现在倒好啦！他们一个个有出息了。怂恿着你爸爸和我离婚，这就是他们陈家人！”
<br/>
<br/>        “可是我在奶奶家的时候，你也没有给我什么呀？我没有见到你给奶奶他们什么东西。”
<br/>
<br/>        “我怎么能让当着你一个小孩子的面给他们东西呢？你看看现在，就拿你七叔陈子霖来说。陈子昂对你七叔好吗？你七叔和他一起在西城共同奋斗，但是现在呢，你看公司做到这个规模，他却不放心你七叔，却让一个外人，她一个相好的来在公司当领导。”
<br/>
<br/>        “你怎么能这么说呢？我爸的文化层次是比江俞轩差一点儿。江俞轩和陈子昂不是一直在一起搭档着吗？将江俞轩放到西城是不是对西城给予厚望？”
<br/>
<br/>        “你真是个小孩子，你真的好傻。陈子昂宁愿让一个外人来在他们西城分公司。都不愿意让你爸也就是你七叔当领导，是不是害怕你七叔有外心呢？难道不是来监视你七叔的？”
<br/>
<br/>        “我不知道，我也不懂，这些对我来说重要吗？”
<br/>
<br/>        “我不知道陈子昂的生意做的怎么样？但是，陈家人有几个沾了他的光。她帮了陈家几个人？”
<br/>
<br/>        “我们家没有谁需要陈子昂帮助的呀！”陈天天觉得不可思议。
<br/>
<br/>        “你呀，你哥哥呀，你妹妹呀！难道陈子昂不给你们照顾吗？陈子昂将股份都分给别人了，她分给你们了吗？她跟你们兄弟姊妹聊过这些事情了吗？”
<br/>
<br/>        “这个好像是没有。”陈天天低声说。
<br/>
<br/>        “看吧！你看！如果她真是为你，为我们陈家着想的话，她为什么不给我们留着股份呢？她那么有钱，她只想着她自己，她从来都没有为陈家人考虑过。”
<br/>
<br/>        “就算是她为了自己的也是正确，人家的股份凭什么要分给我们呢？”
<br/>
<br/>        “凭什么？她是陈家的人呀，你们也是陈家的人啊！陈家现在有几个小辈？你哥哥陈冯伟，你，还有陈连翘，陈森森，才几个人啊，可是她替你们着想了吗？”
<br/>
<br/>        “好了，二妈，你就说吧，你找我有什么事儿？我们没有必要要为这件事纠结。”
<br/>
<br/>        “天天，妈妈现在已经老了，妈妈只有你。妈妈就你一个女儿，我希望你能多陪陪妈妈。天天。你看！妈妈这肚子上的刀口是有多长多深啊？”
<br/>
<br/>        “好吧。只要我有时间我就陪你。”陈天天叹了口气，怎么说冯睿也是生了她一场。
<br/>
<br/>        “天天，我就知道。你是最心疼我的乖女儿，妈妈以后下班回来就给你做饭吃，你以后就经常回来好不好？”
<br/>
<br/>        “好吧。我有时间了我就回来。”
<br/>
<br/>        冯睿给陈天天削了个苹果，一瓣一瓣的喂给陈天天吃。
<br/>
<br/>        两个人正在说话，敲门声响起来了。
<br/>
<br/>        “嗯，颜颜，你来啦！快过来，你妹妹天天回来了。”
<br/>
<br/>        “天天。啊，天天回来了哈。天天，我天天听冯妈念叨你。”
<br/>
<br/>        “夏总。”陈天天看着夏颜很习惯的脱掉鞋，从鞋柜里拿出拖鞋换上后自然地走到沙发上坐下。
<br/>
<br/>        “天天。叫什么夏总？她是你姐姐，夏颜，快叫姐姐。”
<br/>
<br/>        “姐姐？怎么就成了我姐姐啦？”
<br/>
<br/>        “天天。你知道吗？这么几年要不是你姐姐陪着我，你可能都见不到妈妈了。”
<br/>
<br/>        “你爸爸已经和我离婚了。你哥哥结婚之后也不着家呀！家里就我一个人。生个病呀什么的，都是你姐姐帮着我。”
<br/>
<br/>        “冯妈！快别那么说了。我也是你女儿啊！虽然不是亲生的，我也会和天天一样的孝顺你。你看天天对你多好的，我相信天天以后会经常来看你，会很孝顺你的。”
<br/>
<br/>        “你们两个是怎么回事儿？”陈天天不明所以。
<br/>
<br/>        “我和你姐姐认识很久了，我身边没人，就认你姐姐当干女儿了。”
<br/>
<br/>        “天天。咱们以后就是姊妹了。在西城。有什么事情？姐姐会帮着你的。最近呢，我将你哥哥的工作进行了调整。天天，你好好学习吧！等你毕业了，你想到我单位来也行你想去别的公司也行，反正你的工作就不用你操心了。”
<br/>
<br/>        “我的工作也不用你们操心，我自己会找的。”陈天天还没有到和夏颜那么亲密的程度。上一次和夏颜一起吃饭的时候，她能看得出来，夏颜和陈子昂的关系并不好。
<br/>
<br/>        “天天。你莫不是还想要去陈子昂的公司？我劝你还是不要想了。陈子昂的靠山一个一个的离他而去，之前是张家，后来是江家。天天，别指望着陈子昂会将你们陈家带上顶峰，他只要不拖累你们陈家就好了。”夏颜冷笑着说。
<br/>
<br/>        “张家江家？”陈天天并不了解陈子昂和张家及江家的关系。她也不知道她姑姑和张函还有江俞轩之间的关系，但是，她从大家的口中知道了江俞轩和陈子昂的关系不一般。
<br/>
<br/>        “是啊！天天。你还小。有些丑陋的事情其实没有让你知道。我和你小姑陈子昂是闺蜜。她做的一些事情有时间让我都觉得不耻。”夏颜又将她在北京说给蔡少芬的话讲给了陈天天听，然后分析了陈子昂后期的出路。
<br/>
<br/>        陈天天觉得她的三观碎了一地。
<br/>
<br/>        陈子昂比她大了将近九岁，从小也带过她。在她的心目中。陈子昂是善良的，上进的，没有心计的，但是像夏颜口中说的陈子昂，她是没有见过的。
<br/>
<br/>        陈天天的心乱了，如果像夏颜分析的那样，陈家迟早会出问题，她需要为陈家着想，爸爸陈子霖为了西城分公司也是付出努力了的，他们家有现在的状况也不完全是陈子昂的功劳。
