　　少倾，苏莫抛却了心中的杂念，静静的融合神魂与武魂。
<br/>　　聂乾和韩盈见此，心中无奈，不知道苏莫是何打算。
<br/>　　不过，他们已经提醒了苏莫，也算尽了作为朋友的职责。
<br/>　　时间缓缓流逝，海面上的两名弟子战斗不止，已经足足拼斗了近千招
<br/>　　这个时候，战况开始明朗了，其中一人逐渐占据了上风，没过多久，便将另一人击败。
<br/>　　不过，这两人并没有分个你死我活，胜负一分之后，便离开了海面之上，纷纷回到了周围的小岛上。
<br/>　　这时，一直在另一座小岛上等待的冷江，轻笑一声，一步踏出，瞬息千里，几步便来到了中央的海面上。
<br/>　　伫立在海面上，转头望向小岛上的苏莫，冷江眸光森寒，一声厉喝。
<br/>　　“苏莫，下来受死！”
<br/>　　厉喝声如同一声惊雷，在这辽阔的战界之中炸响，让得下方的海水形成冲天的浪涛。
<br/>　　所有在战界的神武学府弟子，都是来了兴趣，有好戏可以看了！
<br/>　　苏莫睁开了眼眸，面无表情的站了起来。
<br/>　　他的武魂和神魂，已经在彼此交融，要不了一时半刻，就能融合成功了。
<br/>　　然而，这个时候，战斗已经要来临了，没有了融合的时间。
<br/>　　“还不下来，难道是怕死不成？”冷江见苏莫迟迟不动声，满脸讥讽的说道。
<br/>　　其他人亦是望着苏莫，一名准帝之境的武者，不敢应战也在情理之中。
<br/>　　“苏兄……！”
<br/>　　聂乾和韩盈正想再劝一番苏莫，就在此时，苏莫身形一闪，如一道电光一般，撕裂长空，飞到了海面上。
<br/>　　“哈哈！”
<br/>　　冷江见此，顿时大笑一声，道：“苏莫，虽然你是个人渣，但这份勇气，还是非常让人敬佩的！”
<br/>　　在冷江看来，苏莫明知自己不敌，还敢迎战，这份勇气倒是非常不错！
<br/>　　只不过，敢肆意抢夺他兄弟的储物戒，这种人渣死不足惜！
<br/>　　“人渣？”苏莫闻言，眸光一寒，心中亦是杀机大起。
<br/>　　唰！
<br/>　　长剑出现在了手中，苏莫面色凝重，体内五枚神丹尽数运转了起来，他不敢有丝毫的大意。
<br/>　　“先接我一刀！”
<br/>　　冷江大喝一声，突然身形冲天而起，长刀在手，无尽的锋芒浩荡九霄。
<br/>　　“斩！”
<br/>　　一刀怒斩而下，恐怖的惨白色刀气倾洒长空，如同九天上劈下的闪电，直击苏莫。
<br/>　　这一刀强大异常，蕴含强烈无比的金之奥义，刀威惊世，锋芒耀世。
<br/>　　咻！
<br/>　　刀气所过，虚空被平滑的切开，整个天地好像都被这一刀分开。
<br/>　　刀气即便还没击中苏莫，已经让他全身冰寒，如坠冰窖。
<br/>　　“果然厉害！”苏莫见此心中一沉，此人的修为暂且不提，光是这股凌厉的金之奥义，怕是已经达到了两阶。
<br/>　　两阶的金之奥义，对于实力的加成，可是非常的大。
<br/>　　唰！
<br/>　　苏莫没有硬挡这一刀，身形急速闪避，瞬间横移千丈。
<br/>　　轰！
<br/>　　恐怖的刀气斩入了海水之中，立刻将海面一分为二，仿佛整个大海都被分开，掀起万丈狂涛。
<br/>　　“躲得了吗？”
<br/>　　冷江见此冷喝冷哼一声，再次出手，手中长刀不停的斩击，一道道刀气怒斩而下。
<br/>　　万千刀气纷纷洒洒，覆盖苍穹，粉碎天地，裹挟无限的杀机，向苏莫绞杀而去。
<br/>　　冷江可是不会有丝毫的留手，他的目的就是将苏莫当场斩杀。
<br/>　　不少人见此，无语的摇了摇头，这个苏莫，在这一招之下，必败无疑了吧！
<br/>　　根本没有任何悬念啊！
<br/>　　聂乾和韩盈两人，面色一紧，虽然他们知道苏莫的实力，但也不认为苏莫能抵挡这铺天盖地的刀气。
<br/>　　搞不好，会当场陨落！
<br/>　　小岛上，古树下，木璃导师面无表情。
<br/>　　她眼眸静静的望着海面上的战斗，面色没有丝毫的波动。
<br/>　　就算苏莫死了，她也不会出手帮助。
<br/>　　一切，都是苏莫咎由自取。
<br/>　　她之所以来此，的确是观看苏莫和冷江的比斗，到时候比斗的结果，她还要回去禀告火元圣师。
<br/>　　若是苏莫死了，火元圣师以后自然不会在关注他了。
<br/>　　若是苏莫没死，火元圣师虽然也不会再邀苏莫去内府火元峰，但以后若是有机会，还是想要招揽苏莫。
<br/>　　说实话，木璃对于苏莫，心中有些嫉妒。
<br/>　　换做是任何人，都不免有些嫉妒之心。
<br/>　　她是炼器师，而且是尊级中品炼器，无论是武道天赋还是炼器天赋，都是百年难得一见。
<br/>　　但即便如此，她也只是在偶尔的时间，才能得到火元圣师的指点。
<br/>　　虽然她称火元圣师为导师，但在她心中，火元圣师就是她的师尊。
<br/>　　她也知道，火元圣师一直想找一个真正的传人，传授其一生高深莫测的炼器之术。
<br/>　　她非常的清楚，火元圣师看重了苏莫，想要培养苏莫，然后收为亲传弟子。
<br/>　　但这种天大的好事，苏莫居然拒绝了，而且是毫不犹豫的拒绝了！
<br/>　　木璃不清楚火元圣师对此是和感想，但她的心中非常不是滋味。
<br/>　　试问，自己梦寐以求，却得不到的东西，别人却根本不在乎，随意丢弃，这种感受落差，任何人都难以接受。
<br/>　　所以，木璃就是想要看看，不知好歹的苏莫，若是等会落败生死，是不是还能如此的高傲！
<br/>　　此刻最高兴的人，莫过于华服青年何传岩了，在魔云海之时，苏莫抢夺了他，他早已对苏莫恨之入骨。
<br/>　　今日就能亲眼见到苏莫被杀，大仇得报，这是何等快事。
<br/>　　他的三位朋友外出历练了，等三人回来，这个好消息应该能让三人惊喜吧！
<br/>　　此时，还有一人对于苏莫非常感兴趣。
<br/>　　这是一名身穿华贵紫衣的翩翩公子，相貌英俊，气度不凡。
<br/>　　此人正是太史轩。
<br/>　　此刻的太史轩，伫立在一座小岛的矮山上，手上拿着一柄折扇，给人一种悠然自得之感。
<br/>　　他的目光一直在凝视着苏莫，眸中露出沉吟之色。
<br/>　　他自然认识苏莫，上次在南苑广场，他还和苏莫有过一面之缘。
<br/>　　不过，他在意的不是苏莫，而是苏莫手中的那柄银色长剑。
<br/>　　那柄剑，好像是他太史世家外族弟子廖云剑的兵器，怎么会在苏莫的手中呢？
<br/>　　虽然银色的长剑多不胜数，模样也都相差不多，但他却认识那柄剑，因为那柄剑正是他当初赠送给廖云剑的兵器。
<br/>　　“难道廖云剑死了？”太史轩低声自喃，廖云剑若是还活着的话，怎么可能将自己的兵器送人？
<br/>　　这显然不太可能！
