　　第110章
<br/>　　码头上乱糟糟一片，人群如蚁般围上来，有人踮起脚尖往这边望。
<br/>　　“谁呀，谁死了。”
<br/>　　“是漕帮的路子野，他被人杀了。”
<br/>　　“胡说八道，在漕帮的地盘谁敢杀他。”
<br/>　　“死得好，这混蛋早该死了。”
<br/>　　呼啦啦声响，一群拎着大棍的壮汉围上来，黑压压一片，看人头数足有百八十众。
<br/>　　这些人都是漕帮的弟兄，一时间将货船围得水泄不通。围观人看到这一幕，都纷纷摇头：他们闯下滔天大祸，这下是难以脱身了。
<br/>　　一人分开人群，走到船前，昂起头打量程大雷等人一眼。
<br/>　　“什么来头，还没下船就敢杀人，够狂的啊！”
<br/>　　程大雷以眼神示意徐神机，却见他一张脸吓得蜡黄，此刻已经没了主意。再看苏樱，却也没有好上多少。
<br/>　　哎，手下没有得力人，这种场合还得自己亲自说话。
<br/>　　“你是什么东西？”
<br/>　　“某叫李佳明，是路爷的副手，听漕帮孙老爷子的命令，在这里管事。”
<br/>　　“你这样的狗东西，有资格和我说话么？”程大雷把玩着手上的扳指，仰起头看着周围一脸厌烦：“扬州是什么破地方，连个懂规矩的人都没有。”
<br/>　　“你！”
<br/>　　李佳明眉毛跳动两下，大手一挥：“弟兄们动手，替路爷报仇。”
<br/>　　漕帮手下群情激奋，挥舞着大棍就要冲上来。此刻只听得呼啦一声响，从船舱内涌出一帮人。手中握着短刀，身上背着弓箭。此刻刀已磨利，铁弓已开圆，直指船下诸人。
<br/>　　双方剑拔弩张，隔着船头对峙。
<br/>　　都是山贼出身，真如果动起手来，却也没有半点怕的道理。
<br/>　　李佳明却有些迟疑，这帮人没下船就敢杀人，一言不合就要动手，他们来得当真好凶。
<br/>　　“最近公子信佛，不想多造杀业。”程大雷摆摆手，冲李佳明道：“千舟城最好的客栈是哪里？”
<br/>　　“渔阳楼，你问这个做什么？”李佳明拧着眉头。
<br/>　　“你这种狗东西不配和爷说话，爷今天在渔阳楼歇脚，让你们漕帮管事的人今晚来渔阳楼来见我。”
<br/>　　周围人齐刷刷的看着程大雷，此人当真是好大口气。程大雷如此狂妄，一时间李佳明却是摸不清程大雷的底细，心中有些打鼓。
<br/>　　“下船。”
<br/>　　程大雷一行人沿着木梯向船下走来，对围着他们的漕帮壮汉视若无睹。他们都将目光看向李佳明，等着他拿主意。
<br/>　　李佳明心中翻来覆去，猜不透程大雷的来路，眼见程大雷已要走出他们的包围圈。
<br/>　　“等等。”李佳明忽然道：“杀了漕帮的人，漕帮自然有人去找你说话。阁下，留个名字吧。”
<br/>　　程大雷突然回头：“姓欧，欧巴欧伐若就是我。”
<br/>　　“欧巴！”
<br/>　　李佳明张开嘴巴，等再回过神来时，程大雷已带着人扬长而去。
<br/>　　“李爷，咱们现在怎么办，就让他们这样走了。”
<br/>　　李佳明咬咬牙，忽然恶狠狠道了一声：“去找孙老爷子，绝不能饶了他们。”
<br/>　　……
<br/>　　穿街过巷，扬州繁华尽收眼底。千舟城还不算扬州最繁华的城池，也不知扬州府的主城会是怎样情景。徐神机一行人的眼睛都不够用了，单单一个千舟城，已经不知比落叶城好上多少。据苏樱和柳芷所说，千舟城的胭脂都比落叶城强上许多倍。
<br/>　　街道两边商铺林立，行人穿行不息，程大雷查看街上行人的信息，发现优秀人才一抓一大把，顶级人才也不算罕见。
<br/>　　这才是一个古代大城该有的景象，相比扬州，幽州根本就是穷乡僻壤，早知道自己就该穿越到扬州啊。景色好，女人也好，江南女子肯定不和幽州府的大脚老娘们一样。
<br/>　　当然，真若在扬州府当山贼，说不得早就被那个少侠当怪杀了。
<br/>　　不过扬州虽然人杰地灵，但路上程大雷也是一个绝世人才都没看见。看来普通人中，能达到顶级就是极限，绝世人才万里无一。
<br/>　　但程大雷手上已经有四个传说潜力的人，前景还是蛮好的。
<br/>　　前方乱哄哄的，行人围成一团，扯着嗓子喊好，却也不知发生了什么事情。
<br/>　　程大雷等人信步走了过去。
<br/>　　一座硕大的擂台，大旗上斗大一个武字。
<br/>　　最近是金龙镖局东家武天吉的五十大寿，金龙镖局在此摆下十日擂，一为扬金龙镖局威名，二为宴请江湖兄弟。
<br/>　　守擂之人是金龙镖局总镖头罗银山，江湖号称铁爪鹰，使柄铁鞭，横行江湖诸多年，从未失过镖。
<br/>　　擂台立下规矩，无论胜负，上台就有一两银子的赏钱。能在罗银山手中走上十个回合，便会被金龙镖局引为上宾，能打赢罗银山就可得二百两雪花银。
<br/>　　这差不多等于海选了，千舟城习武之人上台挑战的不少，今天已经是立擂的第十天，莫说获胜者，便是能在罗银山手中走上十个回合者也寥寥无几。
<br/>　　程大雷近前看的时候，擂台上正在动手，一个年轻后生与罗银山站在一起。三个回合，年轻后生已被罗银山打得只有招架之力，毫无还手之功。
<br/>　　罗银山近前，抓住后生的胳膊，硬生生将他小臂折断。一阵哭爹喊娘般的惨叫，白色骨头碴子都露了出来，本来还有几个想上台的，此刻也吓得不敢动弹。
<br/>　　摆了十日擂，上台就有一两银子拿，金龙镖局撒出银子无数，有些根本没有本事的人也上台，就为了一两银子赏钱。
<br/>　　罗银山被搞得烦不胜烦，今日出此重招，就是为了杀鸡儆猴。
<br/>　　丢下一两银子抛在那后生身上，罗银山道：“抬下去，用这钱给他看伤。诸位，擂台上拳脚无眼，若有真才实学，欢迎上台挑战，大家以武会友。若半点本事没有的人，就不要上来丢人现眼，免得被我的铁爪打死。”
<br/>　　程大雷斜眼往台上凑了一眼，道：“子龙。”
<br/>　　“属下在。”
<br/>　　“上去，废他一条胳膊。”
<br/>　　赵子龙还未上台，突然听得哇呀呀一声爆叫，一人窜上擂台。
<br/>　　“那家的娃娃不知天高地厚，今天让你三爷爷来教训教训你。”
<br/>　　程大雷一捂脑袋，怎么让这憨货上去了。
<br/>(本章完)
