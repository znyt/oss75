　　平复情绪，慕清霄盘坐在床榻上，闭上双眼，双手结印，心神守一，运转心法，压制起体内的邪火。
<br/>　　“呼...!”
<br/>　　嘴中吐出一口热气，再次睁开双眼，眼眸中已经恢复清明，舔了舔嘴唇，其上依旧残留着一缕幽香。
<br/>　　片刻后，微微摇头，嘴角牵起一丝苦，走下床榻，径直向屋外走去。
<br/>　　回到卧室中，床榻上的佳人，柔美的脸颊上，依旧挂着恬静的笑容，睡的很香，估计正做着美梦。
<br/>　　眼眸中闪过一抹温柔，慕清霄轻轻走到窗前，望着内院的夜景，轻轻抚摸了下吞天蟒的小脑袋，拉下袖口，沉思起来。
<br/>　　“这里是什么地方，怎会有如此多强大的气息？”
<br/>　　就在这时，脑海中响起一道悦耳的声音，声音清冷，但清冷中却隐隐含有一丝温柔。
<br/>　　微微一笑，慕清霄道：“这里是迦南学院，内院，你可不要随意暴露气息，里面有几个老家伙，厉害着呢。”
<br/>　　“内院，你来迦南学院做什么？”脑海中，清冷的声音再次响起。
<br/>　　慕清霄也不隐瞒，直言不讳，道：“迦南学院中有一枚异火，异火榜排名第十四的‘陨落心炎’。”
<br/>　　“真是没想到，迦南学院隐藏着一枚异火的消息，居然属实，当初在找到青莲地心火前，我也本想来迦南学院，只是......”
<br/>　　虽然她不再说下去，但慕清霄却清楚，在这个实力为尊的世界中，弱者没有任何权利。
<br/>　　“留在我身边吧。”
<br/>　　忽然，慕清霄冷不丁的开口说道，语气温柔，同时也有一丝期待。
<br/>　　“...蜕变让我的灵魂变得很虚弱，想要完全恢复，恐怕需要不少的时间，而且，另一具灵魂占据着身体...”
<br/>　　闻言，慕清霄嘴角牵起一丝笑意，看来短时间的相处，效果还是不错的，因为她没有拒绝。
<br/>　　...
<br/>　　...
<br/>　　竖日清晨，旭日东升。
<br/>　　清晨的第一缕阳光，毫无阻隔的倾洒而下，带来一丝温暖的同时，也使得弥漫在内院四周的白雾，消散了一些。
<br/>　　此时，两人的住处，却迎来了数十名不速之客。
<br/>　　小医仙打开大门的一瞬间，微微一愣，因为站在门外的，正是一起进入内院的新生，足足二十多名。
<br/>　　其中，一名领头的新生上前一步，他的脸上青肿未消，青一块紫一块，看上去犹如小丑，自以为很帅气的笑容，显得有些瘆人。
<br/>　　看着他的脸，小医仙实在是认不出来，玉手抓着门闩，忍不住问道：“你是？”
<br/>　　“是我，我是新生前五的罗成啊。”
<br/>　　女神居然没认出自己，使得罗成有些不悦，他好歹也是新生中的翘楚啊。
<br/>　　罗成？
<br/>　　听到后者自报家门，小医仙柳眉一皱，脑海中思索起来，随即想到，罗成不就是，给老生暴露两人行踪的新生么？
<br/>　　想到此处，小医仙美眸中闪过一丝厌恶，不易察觉，语气冰冷，道：“你们，有什么事情吗？”
<br/>　　“小医仙姑娘，请问慕清霄在不在？”
<br/>　　话语刚落，屋内就走出一道身影，视线在屋外的新生们身上扫过，道：“有什么事情吗？”
<br/>　　“慕同学，同为新生，您可一定要帮帮我们啊，内院老生实在是欺人太甚。”
<br/>　　罗成脸色涨红，眼中满是恨意，没等慕清霄说话，就接着说道：“就在刚才，便有一些老生团体，进入我们新生区域，嚷嚷着要收新生纳贡费，每人两天的火能......”
<br/>　　“是啊，是啊，慕学长你也知道，我们新生火能本来就不多，慕学长你实力强大，还请你一定要搓搓老生的锐气。”
<br/>　　慕清霄瞥了一眼此人，虽然脸蛋被揍的挺惨，但声音可不就是罗成的弟弟么：“没空，你们找别人吧，两天的火能也不是很多。”
<br/>　　听他拒绝，罗成眼底闪过一丝怨恨，但脸上依旧堆着笑容。
<br/>　　“慕同学，好歹我们也是同期新生，不就应该互相帮助么，我们新生被欺负，你面上也过不去啊。”
<br/>　　闻言，慕清霄依旧面色如常，语气比刚才却冷冽了一分，道：“说完了吗？”
<br/>　　“说...说完了，慕同学你的意思？”
<br/>　　“说完了就给我滚，若是在来这里，我就打断你们的腿。”说罢，慕清霄就直接将门关上，将新生拒之门外。
<br/>　　“你...”
<br/>　　话语刚落，木门就已经关上，罗成气的浑身发抖，不小心牵动伤势，疼的他倒吸凉气。
<br/>　　他本想着，让慕清霄去对付老生，顺便吸引老生的视线，若是能来个你死我活最好，却没想到，后者居然不鸟他。
<br/>　　“好你个慕清霄，我们来找你是给你面子，真以为自己是个什么东西。”
<br/>　　“没错，没错，真是丢我们新生的脸。”
<br/>　　...
<br/>　　...
<br/>　　屋内，听着外面的嚷嚷声，小医仙严重满是冰冷，刚想要出去教训他们，却被制止。
<br/>　　慕清霄拉着她的玉手，微微一笑。
<br/>　　“仙儿，你要知道，世界上总有一群自以为是的蠢货，不用去理会他们，新生前五？莫名其妙，让老生去收拾他们就行了。”
<br/>　　“可是...”
<br/>　　小医仙刚想说什么，慕清霄捏着她的玉手，轻轻抚摸，道：“去收拾他们，岂不是脏了我家仙儿的手，去吃早餐吧，吃完就出去转转。”
<br/>　　闻言，小医仙脸颊一红，微微颔首。
<br/>　　果不其然，没过多久，外面的嚷嚷声就消失不见。
<br/>　　这群新生也只敢嚷嚷几句，毕竟这里可不是外院，更不是新生区域，乱来的话，下场可是会很惨的。
<br/>　　对于这群不要脸的新生，慕清霄丝毫不想理会。
<br/>　　在内院中，最重要的就是实力，而且，罗成是什么心理他能不知道，彻彻底底的伪君子。
<br/>　　这种会耍点小心机的小人，慕清霄实在见的太多了。
<br/>　　在他面前玩心机，简直太嫩了点。
<br/>　　至于老生为何不来找他收纳贡费，那是因为他住的地方，已经不属于新生区域范围。
<br/>　　而且，夏管事管理这一代，听说他可是出了名的暴脾气，被他收拾过的老生，绝对不下于二十名，而且都有头有脸。
<br/>　　哪个老生，敢在他的管辖区域收保护费，简直就是不将他放在眼里，除非老生是不想在内院继续混下去了。
