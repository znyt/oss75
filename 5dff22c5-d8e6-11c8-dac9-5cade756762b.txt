　　数百年时间眨眼即逝，人族自从有了修行法门后，便意味着了自保之力，他们的人口数量在剧增，如今已经足有数亿人口，比之之前十数万人口，简直不可同日而语。
<br/>　　人口的剧增，都快要将不周山的山脚占领了，但随之而来的就是各种问题，还好随着薪火点燃，人族灵智大开，人祖教化人族也容易了许多。他们先后定人伦，建部落，传承文明，功德也不断降落，导致人族的气运上升不少。
<br/>　　同时，随之而来的还有食物问题，人口数量增多，食物已经不足够了，还好人族已经开始修炼了，一些人已经可以走出不周山周围，去其他地方狩猎，所以，暂时食物的问题算是解决了。
<br/>　　同时，人族的发展潜力已经惊动了一些其他种族，但是他们暂时碍于女娲圣人的圣威，不敢妄动，但谁也不能保证，他们以后不会动人族？
<br/>　　因为没人愿意放任这样一个恐怖的人族崛起，威胁道自己的利益。
<br/>　　人族的潜力是不错，但谁让人族出现的时间太短了呢？在没有将潜力转化为实力之前，潜力终将是潜力，这是一个吃人的世界，在这个世界，弱小即是原罪，弱小是不可能存活下去的。
<br/>　　所以，人族迫切希望变强。
<br/>　　……
<br/>　　玄天闭关的道场中，其内什么都没有，有的只是一片黑暗与虚无，这是玄天故意为之。武道修炼，修的就是寂寞，是孤独，是慢慢求索，是高处不胜寒。只有耐得住寂寞，会品尝孤独之人，才能登临绝顶。
<br/>　　突然，在那漆黑无光的虚空之中，传出一丝光亮，这光亮璀璨无比，就仿佛开天之初的第一缕光亮。紧接着，那光亮渐渐在扩大，瞬间便冲破黑暗的束缚，世界划破黑夜，迎来光明。
<br/>　　一个闪烁着无量光的“武”字出现在虚空之中，它就好像是光之源、道之源，各种道韵弥漫其中，虚空中传来一阵淡淡的威压，看到此字，就仿佛是看到了天地，看到了大道。
<br/>　　这是一枚先天道文，也是一道之源。
<br/>　　这是武道的源头，一切武道，好似尽在其中。掌握一字，便可掌握一道，以一字演万法。
<br/>　　这是玄天此次闭关所得，他将自己一身武道，尽皆简化浓缩，最后形成一个“武”字，掌握“武”字，他才是真正的武祖。因为这种先天道文只可能在大道创始者手中出现。它就是一道之祖。
<br/>　　那“武”字散发出无量金光，驱散黑暗，点燃光明，闪烁片刻之后，它渐渐缩小，化作一道流光，迅速没入玄天眉心之处，出现在他识海空间之中，之后化为一座大山，镇压住他的识海。而在他的眉心处，却留下一道细小的金色神纹印记，为玄天更增添了几分威压霸气。
<br/>　　“匡！”
<br/>　　这天，关闭数百年的玄天殿大门被打开，玄天武祖从中走了出来。他出来的第一件事，便是将燧人氏等三人召唤过来，准备商量大事。
<br/>　　人族三祖身穿兽皮做的衣服，大步走了过来。玄天略微感应，便发现他们三人的气血旺盛，周身武道意志更加凝练，一股强者才具有的威势、自信在他们身上出现。
<br/>　　他们的修为都已经洞天境圆满了，已经快要凝聚出天地法相了。不过，他们修行时间还短，对道的领悟不够，一时间还无法领悟出法相之道。
<br/>　　“大哥，你何时出关的？怎么不来通知一声？”
<br/>　　燧人氏龙行虎步，颇具威势的来到近前，说道。
<br/>　　“是啊，大哥，你此次闭关想必收获不少吧？叫我们来又有何事？”
<br/>　　“人族有何事，大哥自己定夺即可。”
<br/>　　有巢氏二人也来到近前，大声说道。
<br/>　　燧人氏三人心情都不错，人族自从有了自己的修行法门后，人口就越来越多，越来越广，强者也多了起来，不再像之前那般没有一丝自保能力。
<br/>　　“我刚出关，了解了一下人族最近发展形势，找你们是为了商量一下我人族接下来的发展问题。”玄天点了点头，面带凝重之色开口说道。
<br/>　　“近百年来，人族虽日益壮大，有繁荣的趋势，但问题仍然不小。”
<br/>　　“其中，地盘问题最为严重，随着人口的繁衍，人族仅仅只靠不周山这点地盘明显是不够的，离开这片地盘，去洪荒之中其他地方寻找生存之地，势在必行。”
<br/>　　“但，这其中最大的问题就是，我人族的实力太弱了，虽然我人族已经开始修炼了，但奈何修炼时间尚短，族中没有太多高手，没有能力与其他众族争夺生存空间。”
<br/>　　“所以，我唤你们前来，是为了商量如何增强人族实力之事。”
<br/>　　玄天将人族面临的问题与难处说了出来，想让他们意识到问题的严重性。
<br/>　　“这……”
<br/>　　听玄天这么一说，他们顿时也觉得是这个理，面色不由的变得沉重起来。洪荒之中，弱肉强食，强者为尊，一切都是要靠实力去争、去抢，没有实力，什么也做不了。
<br/>　　人族虽然现在没有遇到太大的危险，但他们作为人族首领，不得不为人族的未来做全面的考虑，未雨绸缪是必不可少的。更何况，人族面临的危险本就不小。
<br/>　　“大哥，修炼不是一朝一夕之事，你说该怎么办吧？”三人将目光投向玄天，想让他拿主意。
<br/>　　玄天点了点头，面带威严道：
<br/>　　“说来说去，无外乎就是实力问题，若是我们人族有实力，一样可以去争、去抢。所以，为了尽快增强人族的实力，我打算让人族走出去。”
<br/>　　“什么，走出去？大哥此话怎讲？”燧人氏三人不解，面带疑惑道。
<br/>　　“走出去就是要到洪荒各地走走，不要憋在一个地方。”玄天淡淡道。
<br/>　　“大哥，不行啊，人族如今还很弱小，洪荒又十分危险，在不周山附近还好，因为此处有盘古大神遗留下的威压，但若是出了不周山，我人族儿郎岂不是要死去很多？”
<br/>　　“是啊，大哥，这不可以啊。”
<br/>　　“大哥，冷静一点。”就连缁衣氏也劝解道。
<br/>　　“我早有说过，武道之路，生死之途。这世上哪有不拼不杀，不搏不斗就能成为强者的人。人族想要尽快提升实力，不去拼，不去搏，难道要等着别人欺上我们人族的时候，才去拼搏吗？”玄天大声喝道。
<br/>　　“这……大哥，我们不是这个意思，我们只是不想让人族受伤而已。”三人低声道。
<br/>　　“哼！受伤！此时受点伤，总比以后被人屠杀要好得多。”玄天冷声道。
<br/>　　“大哥，没那么恐怖吧！”
<br/>　　“以人族的发展潜力，你以为其他种族能够发现不了？现如今他们只是还没有意识到人族的威胁，在加上女娲娘娘的圣威的威慑下，他们一时还不敢动手。但，若是他们觉得以后人族会带给他们威胁，你们认为其他种族会放过人族吗？”
<br/>　　“尽快人族提升实力这件事，已经迫在眉睫了。”
<br/>　　……
